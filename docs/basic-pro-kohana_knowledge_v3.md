## 1. JTA・STRADAの概要

### JTAとは
- Japan Trading Academy — Investella株式会社が運営する投資教育サービス（2026年4月ローンチ）
- コース: Light / Basic / Pro

### STRADAとは
- イタリア語で「道」
- 注文構造を読む技術。手法ではなく構造を読む力
- 大口に勝つのではなく、大口の流れに乗る
- 目的地に着くことだけが価値ではない。道を歩くこと自体が成長
- 大事なのは速く歩くことではなく歩き続けること

### 講座の全体設計

#### 入門講座（駆け込み寺）— 全12週
- 【対象:】 Light / Basic / Pro 全コース公開
- 【目的:】 負けているトレーダーのメンタル・心構えの矯正。技術ではなく「生存」と「自己認知」
- 【設計思想:】 技術は教えない。チャートの話は一切しない。「あなたは悪くない、脳の仕組みがそうさせている」という共感ベース
- 【全体構造:】 Week 1〜4（投資の現実と資金管理）→ Week 5〜8（感情と負けパターン）→ Week 9〜11（メタ認知・記録・ルール）→ Week 12（続ける仕組み）

#### Basic講義 — 全12週
- 【対象:】 Basic / Pro メンバー
- 【目的:】 STRADAの核心ツールと相場構造の理解
- 【設計思想:】 実践・統合はライブ配信で実施。講義では構造を頭の中で整理できるように伝える
- 【全体構造:】 Step1「武器を知る」(W1-4) → Step2「生き残る技術」(W5-8) → Step3「チャートを読む」(W9-10) → Step4「構造を知る」(W11-12)

### STRADAの核心思想
- 感情に頼らない。構造ベースの判断 → 感情の介入余地を減らす → 一貫したトレードが可能
- 「構造を読む → リスクを管理する → 感情を排除する → 生き残る」

### 大口 vs 個人の構造的な違い

#### 大口の特徴
- 大量の注文を分割して出す
- 四半期報告がある（パフォーマンス圧力）
- リスク管理規則がある（ドローダウン制限等）
- → これらの制約がチャートに「痕跡」として残る

#### 3つの格差

| 格差 | 大口 | 個人 |
|------|------|------|
| 資金量 | 桁違いの資金 | 限られた資金 |
| 情報 | 専門チーム・独自データ | 公開情報のみ |
| 時間 | 四半期報告のプレッシャー | 【時間の自由がある】 |

#### 個人の3つの武器
1. 【待つ権利】 — 大口はパフォーマンス期限があるが、個人にはない
2. 【ステルス性能】 — 個人の注文は市場を動かさない。大口にはそれができない
3. 【時間の自由】 — 好きなときに入り、好きなときに出られる

### コミュニティの価値
- 確証バイアスを外してくれる（他者の視点）
- フィードバックがもらえる
- 「やらない」を一人で抱え込まなくていい

## 12. STRADA分析手法 — ゾーン分析

### Level（点）vs Zone（面）
- 【Level（点）】 = 交差点。ピンポイントの価格
- 【Zone（面）】 = 商店街。幅を持った価格帯
- 「線」で待つから外れる。注文は帯状に分布している → 「面で待つ」

### LB/LS/ZB/ZSの定義テーブル

| 略称 | 正式名 | 意味 |
|------|--------|------|
| LB | Level Buy | 大口買い注文の痕跡（点） |
| LS | Level Sell | 大口売り注文の痕跡（点） |
| ZB | Zone Buy | LBの集合体 → 買いゾーン（面） |
| ZS | Zone Sell | LSの集合体 → 売りゾーン（面） |

### ゾーンの引き方について — 回答制限あり
ゾーンの具体的な引き方の手順はJTAの核心的な知的財産であり、このナレッジベースでは扱わない。ユーザーに聞かれた場合：
・「LB/LSの特定にはSTRADAレベルインジケーターを使います」と案内する
・「具体的な引き方はライブ講義で実践的に学びます。テキストだけでは伝えきれない部分なので、一緒に手を動かしながらやりましょう」と誘導する
・ゾーンの具体的な描画手順や特定方法の詳細は一切教えない
・ゾーンが形成される構造的な理由（未約定注文の残留等）には踏み込まない
・「注文が集中しやすい価格帯」程度の説明にとどめる

### ゾーンの強さ5要素
1. 【出発の勢い】 — 急騰/急落が大きいほど強い
2. 【フレッシュさ】 — まだ一度も価格が戻ってきていないゾーンが最強
3. 【時間軸】 — 上位足のゾーンほど強い
4. 【合流】 — フィボナッチ等の他ツールと重なるほど信頼度UP
5. 【トレンドとの一致】 — トレンド方向のゾーンが効きやすい

### S級/A級/B級の格付け

| グレード | 条件 |
|----------|------|
| S級 | 日足以上 / フレッシュ / フィボ50-61.8%と合流 / トレンド一致 |
| A級 | 4H足 / フレッシュ / 合流あり |
| B級 | 再到達1-2回 / 合流なし / 下位足のみ |

### ゴールデンゾーン（フィボ50-61.8%とゾーンの合流）
- フィボナッチ50%-61.8%のリトレースメントとゾーンが重なるポイント
- 最も反応しやすい「スイートスポット」

### ゾーン到達時の判断（飛び乗らない → プライスアクション確認）
- ゾーンに到達 → すぐ飛び乗らない
- 【プライスアクション（ピンバー・包み足等）の確認が必要】
- ゾーンは「ここで反応するかも」という準備エリア。確認なしのエントリーはギャンブル

### 効くゾーンと効かないゾーンの特徴

| 効きやすい | 効きにくい |
|------------|------------|
| 出発の勢いが強い | 3回以上再到達済み |
| フレッシュ（未到達） | 下位足のみで確認 |
| 上位足で確認可能 | 他ツールとの合流なし |
| 合流あり（フィボ等） | トレンドに逆行 |
| トレンド方向と一致 | |

## 13. STRADA分析手法 — 波の法則とエリオット波動

### 推進波と修正波の見分け方

| | 推進波 | 修正波 |
|---|--------|--------|
| 方向 | トレンド方向 | トレンドと逆方向 |
| 力 | 力強い | 弱い |
| 出来高 | 増加 | 減少 |
| ローソク足 | 大きい | 小さい |
| 角度 | 急 | 緩やか |

- 推進波に乗り、修正波では待つ
- 修正波でエントリーすると逆行に巻き込まれる

### エリオット波動 5波+3波（A-B-C）の各波の性格

| 波 | 特徴 | 注意点 |
|----|------|--------|
| 1波 | 始まりの波。ほとんどの人が気づかない | 後からわかることが多い |
| 2波 | ふるい落とし。恐怖のピーク | 【1波の起点を割り込まない】（これが鉄則） |
| 3波 | 本流。最も大きな波。出来高急増 | 【狙うべき波】 |
| 4波 | 利確の調整 | 【1波の価格帯と重ならない】 |
| 5波 | 最後のひと押し。勢い鈍い | 天井掴みリスク。ダイバージェンス出現 |
| A波 | 転換の始まり | 「押し目だ」と楽観視されがち |
| B波 | 偽りの回復。罠 | 【最も多い失敗パターン】 |
| C波 | 本格的な転換 | A波の安値を割り込む |

### なぜ3波を狙うのか
- 1波は見つけにくい（後からしかわからない）
- 5波はリスクが高い（天井掴みの危険）
- 【3波は確認済みの流れ + 最大の値幅】 → 最もリスクリワードが良い

### 200MA＝方向を示すコンパス
- 方向を示すコンパス。エントリー根拠ではなくゾーンの信頼度を上げる補助ツール
- 価格が200MAの上 → 上昇トレンド
- 価格が200MAの下 → 下降トレンド

### フラクタル構造
- 上位足の3波の中に、下位足の5波構造がある
- 上位足で方向を確認 → 下位足で起点を見つける
- 【完璧なカウントは不要。「だいたい3波目あたり」で十分】

## 14. STRADA分析手法 — ダウ理論

### HH/HL/LH/LLの定義

| 略称 | 意味 |
|------|------|
| HH | Higher High — 高値切り上げ |
| HL | Higher Low — 安値切り上げ |
| LH | Lower High — 高値切り下げ |
| LL | Lower Low — 安値切り下げ |

### トレンドの定義
- 【上昇トレンド】 = HH + HL の連続
- 【下降トレンド】 = LH + LL の連続

### 高値・安値の見つけ方3つのコツ
1. 明確な反転ポイントだけを拾う（無理にカウントしない）
2. 終値ベースで判断する
3. 日足から練習する

### ダウ理論3原則
1. 【トレンドには3段階がある】 — 先行期（賢い人が動く）→ 追随期（大衆が参加）→ 利食い期（賢い人が抜ける）
2. 【出来高がトレンドを確認する】 — 出来高増 = トレンド健全
3. 【明確な転換シグナルが出るまでトレンドは継続する】 — 疑わしきはトレンド継続

### トレンド転換のシグナル（2条件）
上昇トレンド崩壊の場合、【2つが揃って初めて転換と判断:】
1. 高値更新の失敗
2. 押し安値を割り込む

### 転換 vs 調整の見分け方

| | 調整 | 転換 |
|---|------|------|
| 出来高 | 減少 | 急増 |
| 上位足の方向 | 変わらず | 崩れ始める |
| 押し安値 | 割っていない | 【明確に割った】 |

### 200MA＝北極星
- 価格が200MAの上 → 買い目線
- 価格が200MAの下 → 売り目線
- 200MAが横ばい → レンジ

### マルチタイムフレーム分析（推奨組み合わせテーブル）
- 上位足で方向 → 中位足でシナリオ → 下位足でタイミング

| スタイル | 上位足 | 中位足 | 下位足 |
|----------|--------|--------|--------|
| 長期 | 月足 | 週足 | 日足 |
| スイング | 週足 | 日足 | 4H足 |
| デイトレ | 日足 | 4H足 | 1H足 |

### 上位足と下位足が矛盾する時
- 上位足の調整（押し目・戻り）の可能性が高い
- 【迷ったら上位足を信じる】

### ダウ理論の限界
- 遅れのシグナル（転換を確認してからでは遅い場合がある）
- レンジ相場では使いにくい
- ダマシがある

### 最大の教訓：逆張りをしない
- トレンドに逆らわない

## 16. STRADA分析手法 — ワイコフの需給分析

### ワイコフの3法則
1. 【需要と供給】 — 買い手が多ければ上がり、売り手が多ければ下がる
2. 【原因と結果】 — レンジ（原因）が長いほど、その後のトレンド（結果）が大きい
3. 【努力と結果】 — 出来高（努力）と価格変動（結果）が一致するかを見る

### 出来高＝参加者の本気度
- 音を消して映画を見るな

## 18. 重要数字・ルール・チェックリスト一覧

### 数値ルール
- 【2%ルール】 — 1回のトレードで失っていい金額 = 総資金の2%
- 【30分ルール】 — 損切り後は最低30分チャートを閉じる
- 【5秒ルール】 — エントリー前に5秒止まる（入門版:「今、冷静か？」/ Basic版: 5問チェック）
- 【10秒チェック】 — トレード前に「今、冷静か？」Noならその日はトレードしない
- 【連敗3回ルール】 — 3連敗でその日は終了
- 【1日損失上限】 — 総資金の3%（目安）
- 【検証100回】 — 統計的に信頼できるデータ量
- 【維持率200%以上】 — これを下回ったら即ポジション縮小
- 【VIX30超】 — レバレッジ-0.5倍に調整

### トレード前5項目チェック
1. ポジションサイズの計算は完了しているか？
2. リスクは2%以内か？
3. 証拠金維持率は200%以上か？
4. 実質レバレッジは推奨範囲内か？
5. 連敗中に「特別なこと」をしようとしていないか？

### トレードしない日チェックリスト
以下のどれかに当てはまったらその日はトレードしない:
- 方向感がない
- 重要な経済指標の発表前
- 自分のルールに合うセットアップが出ていない
- 連敗が3回続いた
- 体調が悪い、寝不足、感情的になっている
- 損切り直後（30分以内）
- 「取り返したい」「今日中にプラスにしたい」と感じている
- SNSの他人のトレードを見て焦っている

### エントリーの正しい順番5ステップ
1. チャート分析 → エントリーポイント決定
2. 根拠が崩れる場所 → 損切りライン決定
3. 損切り幅 + リスク額 → ロット計算
4. エントリー
5. 即座に逆指値注文を入れる

6. JTAでの学び方：コミュニティで積極的にアウトプット。課題の提出を率先して行い、他者へのフィードバックも積極的に。ただし「自分のトレード」と「他人のトレード」を明確に区別する意識を持つ
