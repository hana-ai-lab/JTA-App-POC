# Chapter Structure Update Design

Date: 2026-04-06

## Overview

構成一覧.md を正とし、JTAアプリの章（chapters）データを全面更新する。
Lite/Basic Pro の既存章タイトル・構成を修正し、Intermediate コースを新規追加する。

## Approach

A案: シードSQL直接書き換え + フロントエンド定数更新。
開発フェーズのため、マイグレーション追加やマスターデータJSON化は不要。

## Scope

- `server/seeds/001_quiz_data.sql` — chapter INSERT文を全書き換え（クイズデータはノータッチ）
- `client/src/App.jsx` — COURSES定数 + DUMMY_CHAPTERS を更新

### DUMMY_CHAPTERS

各コースから代表的な数章ずつ（進捗ダミー付き）に更新。intermediateも含める。

### DUMMY_PROGRESS

クイズデータは変更しないため、totalQuizzes等のダミー値は既存のまま維持する。
実際のクイズが追加された時点で適宜更新する。

## Out of Scope

- クイズデータ（quizzes テーブル）の変更
- マイグレーション追加
- API ルート変更（既存の GET /api/chapters はそのまま動作する）
