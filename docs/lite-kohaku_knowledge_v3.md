# コハクくん ナレッジベース — JTA入門講座（駆け込み寺）完全版

【講座概要】
対象：Light / Basic / Pro 全コース公開
目的：負けているトレーダーのメンタル・心構えの矯正。技術ではなく「生存」と「自己認知」

全体構造：Week 1〜4（投資の現実と資金管理）→ Week 5〜8（感情と負けパターン）→ Week 9〜11（メタ認知・記録・ルール）→ Week 12（続ける仕組み）

## 第1部：なぜ負けるのか（Week 1〜4）

### Week 1：なぜ負けるのか — 9割が退場する本当の理由

【核心メッセージ】
・個人投資家の約7〜9割が負けている。退場する理由は「下手だから」ではなく「1回の負けが大きすぎたから」
・100万円で50万円負けると、元に戻すには100%のリターンが必要（資金を2倍にする必要がある）
・100万円で2万円の負けなら、約2%取り戻せばいい
・負けること自体は悪くない。負け方が問題
・メンタルは強い弱いではなく「理解しているかどうか」

【退場の3パターン】
①【一発退場型】レバレッジをかけすぎて一度の暴落で資金の大半を失う。「まさか自分が」と思っている間に口座がほぼゼロ
②【ジリ貧型】毎回少しずつ負けて気づいたら資金が半分以下。最も多い退場パターン。勝ちトレードの利益より負けトレードの損失がいつも大きい
③【休眠型】大きな負けを経験して精神的に相場に向き合えなくなる。チャートを開くのが怖くなり、半年、1年と離れてしまう

【ドローダウンと回復の非対称性】
・5%の損失 → 約5.3%で回復（まだ戻せる）
・10%の損失 → 約11.1%で回復（少し頑張れば）
・20%の損失 → 25%で回復（かなり厳しい）
・30%の損失 → 約42.9%で回復（非常に困難）
・50%の損失 → 100%で回復（資金を2倍にする必要）
・70%の損失 → 約233%で回復（ほぼ不可能）

【脳の仕組み】
・人間の脳には「プロスペクト理論」「損失回避バイアス」「現状維持バイアス」がある
・利益が出たら「早く確定したい」、損失が出たら「まだ戻るかも」と先延ばし → 利小損大パターン
・これは意志の弱さではなく、脳の標準的な反応

【知識と実行のギャップ】
・知っていることと実行できることはまったく別
・勉強しても勝てない理由：(1) 学ぶ順番が間違っている（手法より土台が先）(2) 知識が体験と結びついていない

【段階的成長】
負けトレーダー → 負けないトレーダー → 勝てるトレーダー
・いきなり「勝てるトレーダー」を目指さない。まず「負けないトレーダー」が最優先
・「勝ち方」より先に「負け方」を学ぶ

【Akiraの体験】
・投資を始めた頃は感覚だけで動いていた。勉強もせず、根拠もなく、勝てなかった
・2022年に転機。仕事後、夜中から朝4時まで毎日勉強。しかし勉強だけでは変わらなかった
・学んだことをアウトプットし、失敗を振り返り、また学ぶサイクルで「わかっている」が「できる」に変わった
・大きな負けをなくし、小さな負けだけにしたら一気に成績が向上した

【Week 1の宿題】
・退場3パターンで自分に一番近いものを選び理由を書く
・「負け方」を先に学ぶべき理由を説明
・「50%負けたら100%必要」を自分の資金で計算

→ ここまでが「なぜ退場するのか」の全体像。同じ経験をした仲間がJTAにはたくさんいるよ🐬

### Week 2：敵はマーケットではなくあなた自身です

【プロスペクト理論】
・人間は損失の痛みを利益の喜びより約2倍強く感じる
・1万円もらった嬉しさ＝「50」とすると、1万円失った痛み＝「100」
・含み損 → 「この痛みを早く消したい」→ 損切り先延ばし
・含み益 → 「この利益を失いたくない」→ 早すぎる利確
・結果：利小損大パターンの完成

【確証バイアス】
・自分がすでに信じていることを裏付ける情報ばかり集め、反する情報を無視する傾向
・「買い」と決めてから分析すると、買いの根拠しか目に入らなくなる
・対策：分析は「チャートが何を語っているか」から始める。先に方向を決めてから根拠を探さない

【サンクコストの罠】
・すでに失ったお金や時間に引きずられて判断が歪む
・「もうここまで耐えたのだから」と損切りを先延ばしにする
・過去に使ったコストと、これからの判断は本来無関係

【保有効果の罠】
・自分が持っているものを実際より高く評価してしまう
・マグカップの実験：持っているグループは持っていないグループの約2.5倍の値段をつけた
・ポジションを持つと「手放す＝価値を失う」と感じて損切りできなくなる

【破滅パターンの連鎖】
含み損 → サンクコスト（ここまで耐えた）→ 保有効果（手放したくない）→ 損失拡大 → リベンジ心理（取り返さないと）→ 暴走

【4つの暴走感情（概要）】
①恐怖 — 損を認められない。損切りラインに達しても「もう少し待てば戻る」と祈る
②欲望 — もっと欲しくなる。連勝後にロットを上げる。「ついている」と感じた瞬間が最も危険
③焦り — 乗り遅れる恐怖（FOMO）。動いてから入ると大半は天井か底をつかむ
④後悔 — 過去に囚われる。損切り後の反転で「切らなければよかった」→ 次の損切りを遅らせる

【メタ認知（初出）】
・自分が今何を感じているかを一歩引いて観察する技術
・3ステップ：(1) 気づく（感情にラベルを貼る）(2) 距離を置く（感情は一時的な脳の反応）(3) ルールに戻る

【最重要の問いかけ】
「もし今、ポジションを持っていないとしたら、このレートでエントリーするか？」
・答えが「いいえ」なら、そのポジションは手放すべき
・サンクコスト・保有効果・確証バイアスを一発で突破できる

【Week 2の宿題】
・「含み益→すぐ確定」「含み損→先延ばし」が起きる理由を説明
・「今ポジションを持っていなかったら入るか？」が有効な理由
・サンクコストと保有効果の違いを日常例で説明

→ 認知バイアスは「知っている」だけでは防げない。仲間と一緒に振り返ることで気づきが深まるよ🐬

### Week 3：資金管理 — 生き残りの数学

【ロットが大きすぎるサイン】
・エントリーした瞬間に心臓がバクバクする
・チャートが気になって仕事が手につかない
・夜何度もスマホを見てしまう
→ 分析が下手でもメンタルが弱いわけでもない。ロットが大きすぎるだけ

【2%ルール】
「1回のトレードで失っていい金額は、総資金の2%まで」
・10連敗しても資金の約82%が残る。20連敗しても約67%が残る
・2%は「我慢」ではなく数学的に最も合理的な水準

リスク率別の連敗耐性：
・1%リスク → 5連敗後約95%、10連敗後約90%、20連敗後約82%
・2%リスク → 5連敗後約90%、10連敗後約82%、20連敗後約67%
・5%リスク → 5連敗後約77%、10連敗後約60%、20連敗後約36%
・10%リスク → 5連敗後約59%、10連敗後約35%、20連敗後約12%

【ロット計算の正しい考え方】
「資金の2%の金額でエントリーする」のではない。
「損切りになった時に失う金額が資金の2%になるようにロット数を調整する」
計算式：最大リスク額 ＝ 総資金 × 2% → ロット ＝ 最大リスク額 ÷ 損切り幅

具体例（資金100万円）：
・最大リスク額 ＝ 100万 × 2% ＝ 2万円
・損切り幅500円 → 40株まで
・損切り幅1,000円 → 20株まで
・損切り幅2,000円 → 10株まで

【適正ロットの基準】
「夜、ぐっすり眠れるロット数」が今の適正値

【資金管理＝保険料】
・自動車保険の毎月の支払い＝「損」ではなく安心を買っている
・2%ルール＝トレード人生における安全ベルト

【連敗は「異常」ではなく「正常」】
・勝率60%でも5連敗する確率は約1%。年200回トレードすれば2回は起きる
・連敗中こそルールを守れるかが分かれ目
・連敗中はロットを下げる。流れが変わった可能性がある

【Week 3の宿題】
・2%ルールとは何か、なぜ2%か
・自分の口座残高で「1回で失ってよい金額」を計算
・「ぐっすり眠れるロット数」の意味

→ 資金管理の計算は、コハクくんと一緒に実際の数字でやってみよう。ライブ講義でも扱うよ🐬

### Week 4：損切りは技術、ナンピンは感情

【損切り＝保険料】
・小さな損失（保険料）を払うことで、大きな損失（破産）を防ぐ
・損切りは「損した」ではなく「次のチャンスのために資金を守る行為」
・ルール通りに損切りできたときは自分を褒める

【損切りできない心理の3段階】
①否認 — 「大丈夫、すぐ戻る」。まだ余裕がある段階で問題に気づけない
②期待 — 「ここまで耐えたのだから報われるはず」。サンクコストの罠が全力作動。チャートではなく「願望」を見ている
③麻痺 — 損切りボタンを押す気力もない。「もうどうにでもなれ」

【損切りを動かしてはいけない理由】
・損切りを動かす＝根拠のないトレードに変わる
・損切りを動かす＝2%ルールが崩壊する
・損切りを動かす＝「祈りのトレード」の始まり
・損切りラインはエントリー前の冷静な自分が決めた判断。含み損の中にいる自分より正しい

【正しい損切りの置き方】
損切りラインは「エントリー根拠が崩れる場所」に置く

【エントリーの正しい順番】
① エントリー根拠を見つける → ② 根拠が崩れる場所を特定（＝損切りライン）→ ③ 損切り幅から2%ルールでロット計算 → ④ エントリー

【ナンピンの2種類】
・【計画的ナンピン（＝戦略）】エントリー前にシナリオとして組み込み済み。各ポイントに根拠あり。全体リスク額が2%ルール内
・【感情的ナンピン（＝祈り）】含み損が出てから衝動的に追加。根拠は「助かりたい」だけ。リスク計算なし
・見分け方：エントリーする前に「ここで追加する」と決めていたか

【よくある負けパターン（初出5種）】
①損切りできない病 — 対策：エントリー前に逆指値を入れておく
②リベンジトレード — 対策：負けた後は最低30分チャートから離れる
③連勝後のロット上げ — 対策：ロットは感覚でなく計算で
④「今日だけ特別」症候群 — 対策：「今日だけ」と思ったら必ず見送り
⑤感情エントリー — 対策：「この判断は計画か感情か」と自問

【損切り後にやるべきこと】
・すぐ次のトレードに入らない
・最低30分〜1時間、チャートから離れる
・ルール通りに損切りできたときは自分を褒める

【Week 1〜4の総まとめ】
・9割が退場するのは下手だからではなく、脳の仕組みを知らないから
・敵はマーケットではなくあなた自身。仕組みを知れば対処できる
・2%ルールを守れば連敗しても退場しない
・損切りは失敗ではなく、次のチャンスへの保険料

【Week 4の宿題】
・損切りを「保険料」と考える意味
・計画的ナンピンと感情的ナンピンの違い
・エントリーから損切りの正しい4ステップ

→ 損切りは「技術」。感覚ではなく仕組みで守ることが大事。ライブ講義で実際のケースを一緒に見てみよう🐬

### Week 5：4つの暴走感情 — 恐怖・欲・焦り・後悔

【恐怖 — 「損を認めたくない」】
・損失を確定させることへの恐怖。脳は「失うこと」を極端に嫌う
・含み益が出ていても「逆行したらどうしよう」→ 利確を早める
・対処：「もし今ポジションを持っていなかったら、このレートでエントリーするか？」→ Noなら切る

【欲 — 「もっと欲しい」】
・利確目標に達しても「もっと伸びるかも」と持ち続ける
・連勝後に「今がチャンスだ」とロットを上げたくなる
・「今回だけ」と思ったとき、それは欲が暴走しているサイン
・対処：利確目標とロットは事前に決めて機械的に実行

【焦り — 「乗り遅れたくない」（FOMO）】
・チャートが動き始めた瞬間に飛び乗る → 高値掴み
・SNSの「爆益」投稿を見て焦る
・対処：「このトレードを見送ったら何を失うか？」→ 答えは「何も失わない」

【後悔 — 「あのとき、ああしていれば」】
・損切り直後の反転 → 「切らなければよかった」→ 次の損切りを遅らせる → 大損
・対処：「結果」ではなく「プロセス」で評価。ルール通りなら正しい行動

【感情の連鎖パターン】
焦り（FOMO）で飛び乗る → 含み損 → 恐怖で損切りできない → 損失拡大 → 後悔 → 「取り返したい」欲が暴走 → リベンジトレード → さらに負ける

【メタ認知の3ステップ】
①気づく — 「今、自分は何を感じているか？」
②距離を置く — 感情は自分そのものではなく一時的な脳の反応
③ルールに戻る — 感情で動くのではなくルールで動く

【トレード前の10秒チェック】
「今、冷静か？」→ 答えがNoならその日はトレードしない

【感情は敵ではなく「気づくべきシグナル」】
・感情をゼロにする必要はない。怖くてもいい。悔しくてもいい
・大切なのは「今、自分が感情的になっている」と気づくこと

【Week 5の宿題】
・4つの感情で自分が一番振り回されやすいもの
・4つの感情が「連鎖」する具体的な流れ
・メタ認知の3ステップ

→ 感情の暴走は1人で戦うと辛い。JTAの仲間と共有すると「自分だけじゃない」と気づけるよ🐬

### Week 6：リベンジトレードの罠

【典型的な流れ】
①負ける → ②「取り返したい」思考が支配 → ③根拠のないエントリー → ④ロットを上げる → ⑤さらに負ける → ⑥繰り返す → 退場への片道切符

【3つの心理的原因】
①サンクコスト — 「あの3万円を取り戻さないと」
②保有効果 — ポジションを手放せない
③怒り — 「不快感を早く解消したい」脳の報酬系の反応

【対処法】
・【30分ルール】損切り後は最低30分チャートを閉じる。物理的に離れる
・【1日の損失上限ルール】連敗3回→終了 / 1日の損失が総資金の3%→終了 / 2連敗→ロット半分
・【損切り後の感情メモ】「悔しい」「焦ってる」「取り返したい」など一言書く
・次のトレードは「まったく別の判断」としてゼロから組み直す

【Week 6の宿題】
・リベンジトレードとは何か、なぜ危険か
・3つの心理の説明
・30分ルールが有効な理由

### Week 7：「やらない」という最強の判断

【ポジポジ病の正体】
・常にポジションを持っていないと落ち着かない＝FOMO＋脳のドーパミン報酬系
・エントリーボタンを押した瞬間にドーパミンが放出される
・ギャンブル依存と同じメカニズム

【トレード回数を減らすと成績が上がる理由】
・月20回・勝率50%・RR1:1 → 手数料分負け
・月5回・勝率60%・RR1:2 → 大きくプラス
・「本当に良いセットアップ」だけに絞られる → 勝率向上

【「待つ」ことの経済的価値】
・資金が守られる（余計な損失がない）
・次のチャンスにフルの資金で臨める
・感情がニュートラルな状態で判断できる

【トレードしない日のチェックリスト】
以下のどれかに当てはまったらトレードしない：
・方向感がない / 重要指標発表前 / ルールに合うセットアップなし / 連敗3回 / 体調不良・寝不足 / 損切り直後（30分以内）/ 「取り返したい」と感じている / SNSで焦っている

【「やらない日」の過ごし方】
・過去トレードの振り返り / パターン観察の練習 / 日誌見返し / 来週の計画

【Week 7の宿題】
・ポジポジ病の正体
・「回数を減らすと成績が上がる」理由
・チェックリストで自分に当てはまる2つ

### Week 8：負けパターン22種 — あなたはどれ？

【全22種一覧】

エントリー判断系（5種）：
①追いかけエントリー — 上がっているのを見て飛び乗る→天井。対策：待つべき場所で待つ
②逆張り病 — 「そろそろ反転するだろう」。トレンドは想像より長く続く
③トリガーなしエントリー — 到達しただけで確認なしに飛び込む
④感情エントリー — 「ワクワクする」で入る。対策：5秒ルール
⑤分析麻痺 — 情報集めすぎて何もできない。「わからない」なら見送り

ポジション管理系（3種）：
⑥損切りできない病 — サンクコスト＋保有効果。対策：「今ノーポジなら入るか？」
⑦リベンジトレード — 最も破壊力がある。対策：30分チャート閉じる
⑧シナリオ変更病 — エントリー後の変更はほぼ100%感情

ロット管理系（3種）：
⑨連勝後のロット上げ — 「ついてる！」→次の1敗で全消え
⑩連敗後のロット上げ — リベンジの変形。連敗後はロット半分に
⑪証拠金維持率無視 — 強制決済リスク

### Week 9：メタ認知 — 感情に気づく技術

【メタ認知とは】
・「自分の思考を見ている、もう一人の自分」
・感情そのものではなく「感情を感じている自分」を見る技術
・「感情をなくす」のではなく「感情に気づく」ことが核心
・日常例：お腹が空いてる時のスーパー→「今お腹空いてるから判断狂ってるな」と気づけたらカゴを見直せる

【5秒ルール】
・エントリーボタンを押す前にたった5秒止まる
・声に出して「今、冷静か？」と自分に聞く
・答えがNoならその日はトレードしない
・5秒で逃すチャンスは、そもそもあなたのチャンスではない

【メタ認知の3ステップ（完全版）】
①気づく — 「今、自分は何を感じているか？」最初はトレード後に気づくだけでOK
②名前をつける（ラベリング）— 具体的に：「イライラ」→「損切りが悔しくて取り返したいと思っている」。脳科学で扁桃体の活動が抑制されると証明
③記録する — トレード前に感情を一言メモ。書く行為が客観性を生む

【感情日記】

①トレード前の感情（一言）
②トレード中に感じた変化
③トレード後の感情と感情スコア（1＝完全に冷静〜5＝感情的だった）

【メタ認知が難しい瞬間への対処】
・感情が暴走する「前」に物理的な歯止めを作る
・「連敗3回で終了」「損切り後30分はチャート閉じる」

### Week 10：トレード日誌の力

【なぜ日誌が必要か】
・人間の記憶は不正確。勝ちは「実力」、負けは「運が悪い」で処理してしまう

・完璧主義 → 1行でもいい
・効果がすぐ見えない → 1か月で変化、3か月で確実に変わる

【「勝ち負け」で評価しない — ルール遵守度で評価する】

④来週の改善点を1つだけ決める — 1つに絞る

## 資金管理カウンセリング用知識

---

### JTA資金管理運用方針（v2確定版）

【基本思想】
JTAの資金管理は「実質レバレッジ倍率の管理」を中核に据える。個別商品の利用を制限するのではなく、すべての運用を合算した実質レバレッジ倍率が基準値以内に収まっているかどうかで管理する。

【計算の基本ルール】
・分母：総資産のうち投資に回している資金（生活防衛資金・待機現金は除外）
・計算式：実質レバレッジ ＝ 全ポジションの総額 ÷ 投資資金

【重要な区別】
・「運用資産」と「総資産」は違う。総資産に対して何%を投資に回しているかで管理する
・ユーザーに必ず「総資産はどのくらい？🐬」と確認する

### レベル別レバレッジ上限（JTA確定基準）

・【Conservative】1.25倍以下 — レバの仕組み学習中、CFD/FX経験浅い人
・【Standard】1.5倍以下 — STRADA基礎習得済み、CFD/FX経験あり。JTA基本推奨ライン
・【Advanced】2.0倍以下 — STRADA上級者、実績とDD管理経験十分な人

・レベル選択は自己申告制
・Advancedの2.0倍上限はAkira実運用データに安全マージンを加えた数値

### 年齢×資産規模の方向性

投資に回す割合の目安：
・20代→総資産の50〜70%
・30〜40代→総資産の40〜60%
・50代以降→総資産の30〜50%

レバレッジレベルの目安：
・20代・少額（〜300万）→ Advanced水準まで可
・30〜40代・中規模（300〜1,000万）→ Standard基本推奨
・50代以降 or 大規模（3,000万〜）→ Conservative〜Standard
・資産規模が大きいほど守り重視（失う金額の絶対値が大きい）
・年齢が高いほど守り重視（リカバリーに使える時間が短い）

### ポジションサイジング（JTA確定基準）

・1回のトレードの最大損失許容額：投資資金の1%（全レベル共通）
・計算式：ロット数 ＝ 最大リスク額（投資資金×1%）÷ 損切り幅

### 月次ドローダウン上限（JTA確定基準）

・月次ドローダウン上限：-10%（全レベル共通）
・上限到達時：Advanced→Standardに引き下げ / Standard→Conservativeに / Conservative→ポジション大幅縮小 or 一時休止
・月が替わったら元のレベルに戻してよい

### 証拠金維持率と実際の運用額の計算

【計算式（最重要）】
実際の運用額 ＝ 証拠金 × 最大レバレッジ ÷ 証拠金維持率(%) × 100

具体例：
・FX口座500万円・維持率200% → 500万×25÷200×100 ＝ 6,250万円
・CFD口座1,000万円・維持率150% → 1,000万×10÷150×100 ＝ 約6,667万円
・FX口座500万円・維持率500% → 500万×25÷500×100 ＝ 2,500万円

維持率の意味：
・100% ＝ 最大枠フル使用（ロスカット寸前）
・200% ＝ 最大枠の半分
・300% ＝ 最大枠の1/3
・500% ＝ 最大枠の1/5（余裕あり）
維持率が高いほど安全、低いほど危険

### 商品別の基礎知識

現物株式：レバ基本1倍（信用最大約3.3倍）
FX：国内最大レバ25倍。最大運用額＝入金額×25倍
CFD：個別株5倍、株価指数10倍、商品20倍。概算で入金額×10倍

⚠️「最大運用額」は「最大まで使って良い額」ではない。あくまで理論上の最大。実際に使うべき額はJTA基準（1.25〜2.0倍）から逆算。

### ボラティリティ高騰時の動的調整
・FOMC/CPI/雇用統計前後：レバレッジ1段階下げ
・VIX 30超：新規禁止 or ロット半減
・地政学リスク：通常の50%以下に
