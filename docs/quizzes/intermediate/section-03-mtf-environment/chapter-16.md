# 第16章: 時間軸の衝突 — 上位足と下位足が矛盾するとき

## Q1
**問題:** 時間軸の衝突パターンのうち「最頻出」とされるのはどれですか？
- A) 上位足レンジ中の下位足トレンド
- B) 上位足トレンド中の下位足調整
- C) 上位足トレンド転換の初動
- D) 全時間足がレンジ
**正解:** B
**解説:** 上位足トレンド中に下位足が逆行する現象は、トレンドの中に必ず修正波が含まれるため日常的に発生し、最も頻出する衝突パターンです。フィボ38.2〜61.8%＋MA反発で押し目買いを狙います。

## Q2
**問題:** 週足レンジ中に日足が上昇している場合、適切な対応として正しいものはどれですか？
- A) 週足を無視して日足のトレンドに乗る
- B) レンジ上限でのブレイクアウトを狙ってフルロットで買う
- C) 利確目標をレンジ上限の手前に設定し、ポジションサイズを50%に縮小する
- D) 日足の上昇が週足に反映されるまで待つ
**正解:** C
**解説:** 上位足がレンジの場合、下位足のトレンドはレンジ上限で止まる可能性が高いため、利確目標を手前に設定し、ポジションサイズを縮小するのが適切です。この局面は「罠が多い」パターンです。

## Q3
**問題:** 「どの時間足でも方向感がないのに、スキャルピングならいけるかも」という思考が危険な理由として最も適切なものはどれですか？
- A) スキャルピングは上級者しかできない手法だから
- B) 手数料コストが高くなるから
- C) スキャルピングはMTF分析と相性が悪いから
- D) 方向感のない相場は、短い時間足でも方向感がないから
**正解:** D
**解説:** 方向感のない相場では、時間足を短くしても方向感は現れません。「短い時間足なら答えが見つかる」という幻想を捨てることが重要です。

## Q4
**問題:** 上位足トレンド転換の初動を判断する際、「迷ったら」取るべき行動として正しいものはどれですか？
- A) 小さなロットで逆方向にエントリーする
- B) 日足の200MAが下を向くまで待つ
- C) フィボ戻し率、出来高、週足安値の3つの手がかりで判断し、それでも迷えば見送る
- D) 4時間足のトリガーが出たら即座にエントリーする
**正解:** C
**解説:** トレンド転換の初動は判断が最も難しい局面です。フィボ戻し率・出来高・週足安値の3手がかりで総合判断し、迷った場合は見送るのが正解です。

## Q5
**問題:** 全時間足（週足・日足・4時間足）がレンジの状態になった場合、適切な対応はどれですか？
- A) トレードしない（「何もしない」が正解）
- B) レンジの上下限でデイトレードを繰り返す
- C) レンジ幅が狭くなるのを待ってからエントリーする
- D) 4時間足以下で方向感を探す
**正解:** A
**解説:** 全時間足がレンジの状態は方向感が全くなく、トレードすべきでない局面です。「何もしない」ことが正しい判断です。
