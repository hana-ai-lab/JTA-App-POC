# 第19章: 環境認識チェックシート — 毎日のルーティン

## Q1
**問題:** 環境認識チェックシートの「週次チェック」で確認する5項目に含まれないものはどれですか？
- A) 週足のトレンド方向
- B) 週足のサイクル位置
- C) ファンダメンタルズ概況
- D) エントリーのトリガー判定
**正解:** D
**解説:** 週次チェックは週足のトレンド方向、サイクル位置、主要水平線、ファンダメンタルズ概況、今週の戦略メモの5項目です。トリガー判定はエントリー前チェック（3分）で行う項目です。

## Q2
**問題:** 環境認識チェックシートの「日次チェック」における5番目の項目は何ですか？
- A) トレード判断（「本日は〇〇を狙う/やらない」の記述）
- B) 日足のトレンド方向
- C) 4時間足の状態
- D) 本日の経済指標
**正解:** A
**解説:** 日次チェックの5番目は「トレード判断」で、「本日は〇〇を狙う/やらない」を1行で記述します。これによりその日の方向性を事前に明確にします。

## Q3
**問題:** エントリー前チェック（トリガー出現時）で確認する6項目のうち、リスクリワード比の最低基準はいくつ以上ですか？
- A) 1:1以上
- B) 1:2以上
- C) 1:1.5以上
- D) 1:3以上
**正解:** C
**解説:** エントリー前チェックの5番目の項目は「リスクリワード比は最低1:1.5以上あるか」です。この基準を満たさないエントリーは見送ります。

## Q4
**問題:** エントリー前チェックの6項目で、「1つでもNOがあるなら」取るべき行動はどれですか？
- A) 見送り（エントリーしない）
- B) ポジションサイズを50%に縮小してエントリーする
- C) NOの項目が重要でなければエントリーしてよい
- D) トリガーが強いパターンなら例外としてエントリーする
**正解:** A
**解説:** 6つのチェックすべてに「YES」と答えられるときだけエントリーします。1つでも「NO」がある場合は見送りです。基準を一部緩めることはしません。

## Q5
**問題:** 環境認識チェックシートにおいて、エントリー条件の事前設計として記述すべき内容として正しいものはどれですか？
- A) 「今日の損益目標は〇〇円」
- B) 「今週は買い中心でいく」
- C) 「〇〇の水準で〇〇のトリガーが出たらエントリー」
- D) 「ロットを〇〇枚にする」
**正解:** C
**解説:** エントリー条件の事前設計では、「〇〇の水準で〇〇のトリガーが出たらエントリー」という具体的な条件を記述します。これにより、トリガー出現時に冷静に判断できます。
