# 第17章: 環境認識×ファンダメンタルズ — 二刀流の判断法

## Q1
**問題:** 環境認識に組み込むべきファンダメンタルズ4要素として正しい組み合わせはどれですか？
- A) 金利の方向、GDP成長率、貿易収支、住宅着工件数
- B) 為替介入、財政収支、消費者信頼感、小売売上高
- C) 金利の方向、インフレ動向、雇用統計、リスクオン/オフ
- D) 政策金利、鉱工業生産指数、PMI、失業保険申請件数
**正解:** C
**解説:** 環境認識に組み込むべきファンダ4要素は、金利の方向、インフレ動向（CPI）、雇用統計（NFP）、リスクオン/オフ（VIX・株式市場）です。これらはテクニカル分析の確度を測る基準になります。

## Q2
**問題:** テクニカルは明確だがファンダメンタルズが中立・不明の場合、適切なアクションはどれですか？
- A) エントリーせず、ファンダが明確になるまで待つ
- B) 通常のポジションサイズで積極的にエントリーする
- C) ファンダメンタルズ分析をやめてテクニカルのみで判断する
- D) エントリー可だが、ポジションサイズを50〜70%に縮小する
**正解:** D
**解説:** テクニカルが明確でもファンダが中立の場合は「中確度」と判定し、エントリーは可能ですがポジションサイズを50〜70%に縮小してリスクを抑えます。

## Q3
**問題:** テクニカルとファンダメンタルズが逆方向のときの対応として正しいものはどれですか？
- A) テクニカルを優先して通常サイズでエントリーする
- B) ファンダメンタルズを優先して逆方向にエントリーする
- C) ポジションサイズを30%に縮小すればエントリーしてよい
- D) 原則エントリーせず、方向が一致するまで待つ
**正解:** D
**解説:** テクニカルとファンダが逆方向の場合は「低確度」と判定し、原則としてエントリーしません。両方が一致するまで待つのが適切です。

## Q4
**問題:** 週次ファンダチェックリストに含まれる項目として含まれないものはどれですか？
- A) VIXの現在の水準確認
- B) ドルインデックス（DXY）の方向確認
- C) 主要中央銀行の直近のスタンス確認
- D) 200週移動平均線の傾き確認
**正解:** D
**解説:** 週次ファンダチェックリストは経済指標の日程、主要中央銀行のスタンス、VIX水準、主要株価指数の方向、DXYの方向、リスクオン/オフの記述の6項目です。200週MAの確認はテクニカル分析の環境認識チェックに含まれます。

## Q5
**問題:** VIXが「15以下」の水準である場合、一般的な市場の状態として正しいものはどれですか？
- A) 市場がパニック状態にあり、リスクオフが強まっている
- B) トレンド転換の直前であり警戒が必要
- C) ボラティリティが低く、リスクオンの穏やかな相場環境
- D) 出来高が急増しており、ブレイクが近い
**正解:** C
**解説:** VIXが15以下の場合は市場の恐怖感が低く、リスクオンの穏やかな環境を示しています。VIXは30以上でリスクオフの警戒レベル、20〜25で注意レベルとされます。
