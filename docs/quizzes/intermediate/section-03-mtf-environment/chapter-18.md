# 第18章: 実戦シナリオ分析 — 5つのケーススタディ

## Q1
**問題:** 実戦シナリオ分析のケーススタディで「週足上昇×日足押し目×4H反転」の教科書通りの押し目買いにおいて、損切りの適切な位置はどこですか？
- A) 直近高値のすぐ上
- B) 週足の200MAの下
- C) フィボ61.8%の下
- D) 4時間足の直近安値
**正解:** C
**解説:** 5レイヤー全てが買いを支持する高確度の場面では、損切りはフィボ61.8%の下に設定します。この位置は修正波の限界ラインとして機能しやすいためです。

## Q2
**問題:** 価格は上昇しているのに出来高が直近3週間で明らかに減少している状況は、何を示唆していますか？
- A) 上昇トレンドがさらに加速する前兆
- B) トレンドの「息切れ」のサインで、新規買いは見送りすべき
- C) 大口が新たにポジションを構築している
- D) レンジ入りする前に一旦調整が入る合図
**正解:** B
**解説:** 価格上昇に出来高が伴わないのは、上昇への参加者が減っている証拠です。トレンドの「息切れ」サインとして、新規買いは見送り、既存ロングはトレーリングストップで保護するのが適切です。

## Q3
**問題:** テクニカル的には完璧な押し目買いのセットアップが揃っているが、翌日にCPI発表を控えている場合、適切な対応はどれですか？
- A) 発表前のエントリーは見送り、発表後にセットアップが崩れていなければエントリーする
- B) CPI発表前にエントリーし、損切りを広めに設定する
- C) セットアップが完璧ならCPI前にエントリーすべき
- D) ポジションサイズを50%に縮小してエントリーする
**正解:** A
**解説:** Tier 1指標の発表前は、結果次第で方向が急変するリスクがあります。「待つ勇気」を持ち、発表後にセットアップが維持されていればその時点でエントリーするのが適切です。

## Q4
**問題:** 日足で直近の押し安値を割り込んだが、週足の200MAはまだ上向きの局面で取るべき対応として最も適切なものはどれですか？
- A) 日足の押し安値割れは売り転換の確定信号なので即ショート
- B) 「わからない」と認めて、週足の直近安値まで待って反発を見るか、別のペアを探す
- C) 週足の200MAが上向きだから買いで問題ない
- D) フィボナッチ78.6%までの deeper な調整を待って買い
**正解:** B
**解説:** この局面は「最も判断が割れる」トレンド転換の見極め場面です。無理に答えを出さず、「わからない」と認めて、より明確な状況を待つか、別の通貨ペアを探すのが適切です。

## Q5
**問題:** 週足レンジ×日足上昇のケースで、日足と4時間足だけを見て「買い継続」と判断した場合の落とし穴は何ですか？
- A) 日足の上昇トレンドがフェイクである可能性が高い
- B) 週足のレンジ上限までの残り距離が近く、上値が限定的であることを見落とす
- C) 4時間足のトリガーが信頼できない
- D) 出来高が常にレンジ相場では機能しない
**正解:** B
**解説:** 下位足だけで判断すると、週足レンジの上限が目前に迫っていることを見落とします。上位足のレンジ上限に到達するまでの距離を考慮せずにエントリーすると、上値が限定的で不利なトレードになります。
