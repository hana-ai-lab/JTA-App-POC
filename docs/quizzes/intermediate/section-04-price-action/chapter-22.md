# 第22章: 「足の質」の判定 — パターンのグレード分け

## Q1
**問題:** 複数の根拠が重なる重要なサポートゾーン内でフィボ61.8%と合流し、長い下ヒゲのピンバー（実体の4倍のヒゲ）が出現し、出来高が20日平均の2.3倍だった場合の適切な判断は何か？
- A) 三位一体のうち2つ揃っているので、ロットを半分にしてエントリーする
- B) 出来高が多すぎるため見送る
- C) さらに1週間待ってから判断する
- D) 三位一体すべて高評価なので、迷わずエントリーする
**正解:** D
**解説:** 場所（重要なサポートゾーン＋フィボ合流）、足（実体の4倍のヒゲ・反対側ヒゲなし）、出来高（20日平均の2.3倍）の3要素すべてが最高評価の場合、三位一体の全条件が充足されており高確度のエントリーポイントです。

## Q2
**問題:** 4時間足のサポートゾーンで教科書通りのピンバーが出たが、出来高が平均以下だった場合の適切な対応はどれか？
- A) 場所と足は合格だが出来高が欠けているため、次の足で出来高を伴った続伸を確認してからエントリーする
- B) ピンバーがきれいなので通常通りエントリーする
- C) 出来高は無視してピンバーだけで判断する
- D) 出来高が低いので逆方向にエントリーする
**正解:** A
**解説:** 三位一体のうち場所と足は合格でも、出来高が欠けていると「見せかけの反転」のリスクがあります。次の足で出来高を伴った続伸を確認することで、フェイクを回避しつつエントリーの精度を上げられます。

## Q3
**問題:** どのゾーンにも属していない「チャートの途中」にきれいなピンバーが出た場合の判断として正しいものはどれか？
- A) ピンバーの形が完璧ならエントリーしてよい
- B) 出来高が平均以上ならエントリーしてよい
- C) 場所が不合格なので見送る
- D) より下位の時間足で確認してからエントリーする
**正解:** C
**解説:** 「何もない場所のきれいなパターン」は最も多いノイズです。三位一体の優先順位で最も高い「場所」が不合格の場合、足の形や出来高にかかわらず見送るのが正解です。

## Q4
**問題:** デュクシ発生後の反転足で、ヒゲ突破時は出来高が低く、戻り時に出来高が急増していた場合、何を示しているか？
- A) 本物のブレイクが起きた
- B) 個人投資家が追随している
- C) 大口のストップ狩り後の反転であり、高確度のエントリーポイント
- D) レンジが継続する
**正解:** C
**解説:** 出来高の「低→高」の変化は、突破時には大口以外の参加が少なく（フェイク）、戻り時に大口が本気で反転方向に注文を入れた証拠です。デュクシの裏付けとして信頼性の高いパターンです。

## Q5
**問題:** 日足のレジスタンスゾーン上限付近で大きな陰線の後にはらみ足が3本連続し、出来高が徐々に減少している場合、どのような戦略が適切か？
- A) 直ちに買いエントリーする
- B) はらみ足の方向性が不明なため見送る
- C) 上位足が下降トレンドなら下方向のブレイクを待ち、ブレイク足の出来高急増でエントリーする
- D) レンジ取引として上限と下限で往復トレードする
**正解:** C
**解説:** はらみ足3本連続＋出来高減少はエネルギー蓄積中のサインです。はらみ足自体は方向を示しませんが、上位足が下降トレンドであれば下方向のブレイクが想定され、ブレイク時に出来高が急増したらエントリーします。

## Q6
**問題:** エントリー前の確認項目が全てクリアされた場合の適切なアクションはどれか？
- A) 通常のポジションサイズでエントリーする
- B) ロットを2倍にしてエントリーする
- C) さらに2時間待ってから判断する
- D) 見送って次のチャンスを待つ
**正解:** A
**解説:** 全ての確認項目がクリアされた状態は最高確度のシグナルです。三位一体の全条件が揃っているため、通常のポジションサイズで積極的にエントリーします。ロットを増やすのは連勝バイアスのサインなので避けます。

## Q7
**問題:** エントリー前チェックで確認項目の多くが満たされなかった場合の適切な判断は何か？
- A) ロットを50%に縮小してエントリーする
- B) 確度不足として見送り、次のチャンスを待つ
- C) 確認項目の半分以上が揃っているのでエントリーする
- D) 満たされた項目だけを条件にエントリーする
**正解:** B
**解説:** 確認項目が多く揃わない場合は確度不足と判定します。無理にエントリーするより見送って次の高確度のチャンスを待つ方が、長期的な損益を改善します。条件が高確度で揃う場面だけを狙うのが基準です。

## Q8
**問題:** 三位一体チェックシートの項目に「上位足のトレンド方向と一致しているか」が含まれる理由は何か？
- A) 上位足の分析だけで勝てるから
- B) 上位足のトレンドは常に正しいから
- C) 下位足のパターンは意味がないから
- D) 上位足のトレンドに逆らうパターンは機能しにくいから
**正解:** D
**解説:** 日足が下降トレンド中に4時間足のサポートで買いのピンバーが出ても、上位足の戻り売りに飲まれるリスクが大きいです。上位足と同方向のパターンだけを拾うことで、順張りの確度を高められます。

## Q9
**問題:** 三位一体チェックシートを使い始めた結果として報告されている変化はどれか？
- A) 月間トレード回数が増加し、勝率も上昇した
- B) 月間トレード回数が減少し、勝率が大幅に低下した
- C) 月間トレード回数が減少したが、勝率と月間損益が改善した
- D) トレード回数と勝率は変わらず、損益だけが改善した
**正解:** C
**解説:** チェックシートの基準を満たす場面だけに絞ることで、月間のトレード回数は大幅に減少しますが、エントリーの質が上がるため勝率と月間損益が改善します。「たくさんトレードすること」と「たくさん稼ぐこと」は別物です。

## Q10
**問題:** 三位一体チェックシートにおいて「重要経済指標の発表30分前ではないか？」という項目が含まれている理由は何か？
- A) 指標結果で全てがリセットされる可能性があり、パターンの信頼度が極端に下がるから
- B) 指標発表前はスプレッドが広がりエントリーに不利だから
- C) 指標発表前は必ずトレンドが発生するから
- D) 指標発表前は出来高が必ず減少するから
**正解:** A
**解説:** 経済指標の発表結果によって相場環境が一変する可能性があります。どんなに三位一体が揃っていても、指標発表直前は結果次第でパターンが無効になるリスクが高いため、エントリーを避けるべきです。
