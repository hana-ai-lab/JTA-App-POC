# 第26章: 大口の仕掛けを「読む」技術 — なぜダマシは繰り返されるのか

## Q1
**問題:** 大口投資家がダマシを仕掛ける構造的理由として最も適切なものはどれか？
- A) 個人投資家の注目を集めて取引量を増やすため
- B) 大きな注文を一度に執行すると価格が動いてしまうため、損切り注文を「燃料」として流動性を確保するため
- C) テクニカル指標を意図的に歪めてチャートパターンを崩すため
- D) 中央銀行の政策判断に影響を与えるため
**正解:** B
**解説:** 大口は何十億円もの注文を一度に執行すると自分の注文で価格が動いてしまうため、個人投資家の損切り注文を「天然の流動性プール」として利用します。

## Q2
**問題:** ダマシのパターンが過去も現在も未来も繰り返される理由として正しい組み合わせはどれか？
- A) 大口の構造的制約、個人投資家の心理的癖、教科書通りのトレードを逆手に取れること
- B) アルゴリズム取引の普及、高頻度取引の増加、投資信託の拡大
- C) テクニカル分析の進化、SNSの普及、グローバル化の進展
- D) 中央銀行の政策変更、通貨制度の変更、取引所の規制変更
**正解:** A
**解説:** 大口の大きな注文は分散執行の必要があり、個人は直近の高値・安値のすぐ外側に損切りを置く傾向があり、「サポート割れで売り」等の定石を逆手に取れるため、この構造は永遠に変わりません。

## Q3
**問題:** ストップ狩り（Stop Hunt）の際に大口が狙う場所として、最も標準的な水準はどれか？
- A) 移動平均線の交差ポイント
- B) ボリンジャーバンドの中央線
- C) 直近の明確な高値または安値のすぐ外側
- D) フィボナッチ23.6%リトレースメント
**正解:** C
**解説:** 直近の明確な高値・安値のすぐ外側には多くの投資家の損切り注文が集中しており、大口にとって最も「狩りやすい」流動性プールになります。

## Q4
**問題:** ベアトラップ（Bear Trap）の説明として正しいものはどれか？
- A) サポートを一瞬割り込んで売りを誘ってから急反転し、その売り注文で買い集める動き
- B) 上位足のトレンドに逆行して下落を加速させる動き
- C) 出来高を伴わないまま支持線を徐々に下抜けていく動き
- D) レジスタンスを一瞬超えてから急反落し、買いを誘ってから売り抜ける動き
**正解:** A
**解説:** ベアトラップはサポート割れで個人に「下落開始」と思わせて売らせ、その売り注文を燃料に買いポジションを構築する大口の仕掛けです。名前に「ベア（弱気）」とありますが、実際は大口が買いを入れる局面です。

## Q5
**問題:** ストップ狩りか本物のブレイクかを見極める際、最も信頼度が高いシグナルの組み合わせはどれか？
- A) ボリンジャーバンドの拡大＋RSIの乖離
- B) 実体での突破＋直近3本のローソク足が同方向
- C) ヒゲでの突破＋突破時の出来高が少ない
- D) 出来高の急増＋上位足のトレンドとの一致
**正解:** C
**解説:** ヒゲだけで高値・安値を超え、実体は元のレンジ内にあり、突破時の出来高が少ない場合、ストップ狩りの可能性が高いです。本物のブレイクは出来高の急増（20日平均の1.5倍以上）を伴います。

## Q6
**問題:** リアルタイムでストップ狩りを検知する5つの特徴のうち、「即座の反転」が意味するものはどれか？
- A) 突破後、1〜3本の足以内に元の方向に戻る
- B) 突破後、新しいレンジを形成してから戻る
- C) 突破後、損切りラインに向かって一気に進む
- D) 突破後、10本以上のローソク足をかけてゆっくり戻る
**正解:** A
**解説:** ストップ狩りでは、損切りを刈り取った直後に本来の方向に戻るため、1〜3本の足以内に反転が起きます。本物のブレイクは突破後も方向を維持する点が決定的に異なります。

## Q7
**問題:** ストップ狩りが起きやすい時間帯として挙げられるのは、日本時間でどの時間帯か？
- A) 12:00〜13:00（昼休み）
- B) 3:00〜5:00（欧米市場クローズ直後）
- C) 16:00前後と22:00前後（ロンドンオープン・NYオープン）
- D) 6:00〜9:00（東京オープン直後）
**正解:** C
**解説:** ロンドンオープン（16:00 JST前後）とNYオープン（22:00 JST前後）は流動性が急激に変化する時間帯で、大口のストップ狩りが起きやすくなります。

## Q8
**問題:** ストップ狩りの判定基準で「5つのうち3つ以上該当＝ストップ狩りの可能性大」とされるが、これはなぜ全て該当を求めないのか？
- A) 実戦では全ての条件が同時に揃うことは稀であり、3つ以上で十分に高い確度が得られるため
- B) 条件が多すぎると判断が遅くなりエントリーチャンスを逃すため
- C) 大口は常に3つの条件だけを満たすように仕掛けるため
- D) 5つ全て該当することは統計的にあり得ないため
**正解:** A
**解説:** リアルタイムの相場で5つの条件が全て同時に揃うことは稀ですが、3つ以上揃えば「ストップ狩りの可能性が高い」と実用的に判断できます。特にヒゲ突破＋出来高不足の組み合わせが信頼度を高めます。

## Q9
**問題:** 毎朝の環境認識でストップ狩りが起きそうな場所を事前にマークする意味として最も適切なものはどれか？
- A) マークした場所で必ずストップ狩りが起きることを確認するため
- B) 大口の注文の正確な規模を推定するため
- C) その場所でのみエントリーを許可するため
- D) 構えていれば狩りに引っかかるのではなく、狩りの後の反転を狙えるため
**正解:** D
**解説:** 事前に「狩り場」をマークしておけば、価格が接近したときに「狩りが来るかも」と構えられます。準備があることで、狩りの犠牲になるのではなく、狩りの後の反転を利益に変えられます。

## Q10
**問題:** ストップ狩りが起きやすい場所として「キリ番のすぐ外側」が挙げられる理由は何か？
- A) キリ番は中央銀行が介入する水準だから
- B) キリ番では自動売買システムが停止するため
- C) キリ番は過去の高値・安値と必ず一致するため
- D) 多くのトレーダーがキリ番の数pips外側に損切りを置く傾向があるため
**正解:** D
**解説:** 100円、145円などのキリ番は多くのトレーダーが意識する水準であり、その数pips外側に損切りを集める傾向があります。大口はその集まった損切り注文を狙ってヒゲで一瞬だけ突き抜けます。
