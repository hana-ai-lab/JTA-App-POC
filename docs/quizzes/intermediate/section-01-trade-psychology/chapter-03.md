# 第3章: 成功体験への執着 — 「前はこれで勝てた」の呪縛

## Q1
**問題:** 過去に大きく取れた形に似たチャートを見て、条件確認前に入りたくなった。最初にすべきことはどれか？
- A) 形ではなく、現在の相場環境・流動性・根拠の合流を確認する
- B) 過去に勝てた形なので素早く入る
- C) 同じ形ならロットを上げて再現を狙う
- D) 過去の勝ちを忘れるため別手法に切り替える
**正解:** A
**解説:** 成功体験は「似た形」を特別に見せます。しかし相場は背景条件まで同じにはなりません。現在の環境、ゾーンの鮮度、出来高、上位足との整合を確認してから判断します。

## Q2
**問題:** トレンド相場で成功したブレイク手法を、レンジ相場でも使い続けて連敗した。根本原因はどれか？
- A) ブレイク手法はどの環境でも使えない
- B) 相場環境の変化より過去の成功記憶を優先してしまった
- C) レンジ相場では全ての手法が必ず負ける
- D) 昨年の利益が全て偶然だった
**正解:** B
**解説:** 過去に機能した手法でも、環境が変われば期待値は変わります。成功体験が強いほど「また通用する」と思い込み、切り替えが遅れます。手法は過去ではなく今の環境で評価します。

## Q3
**問題:** 「3回連続で勝ったパターンだから今回も同じ」と考えた。危険な理由はどれか？
- A) 3回勝ったパターンは次に必ず負ける
- B) 同じパターンは一度しか機能しない
- C) 少数の成功に特別な意味を持たせ、背景条件の違いを見落とす
- D) 勝った回数を数えること自体が不要
**正解:** C
**解説:** 3回の成功は強い記憶になりますが、統計的には十分ではありません。形が似ていても参加者、ボラティリティ、流動性は違います。毎回の成功はサンプルの1つとして扱う必要があります。

## Q4
**問題:** 以前の押し目買い成功が忘れられず、安値更新後も「押し目」と見て買った。防ぐ認識はどれか？
- A) 押し目買いは一度成功したら使い続ける
- B) 安値更新後も買い目線を固定する
- C) 過去の成功を完全に記録から消す
- D) 押し目か転換かは過去の記憶ではなく現在の構造で判断する
**正解:** D
**解説:** 成功記憶が強いと、転換の兆候まで押し目に見えます。判断基準は今の高安値、出来高、波動、上位足です。過去の記憶より現在の構造を優先することで執着を切れます。

## Q5
**問題:** 勝てた手法の成績が2ヶ月悪化している。適切な対応はどれか？
- A) 過去に勝てたので何も変えず耐える
- B) 現環境で機能しにくい理由を検証し、必要なら修正や休止を選ぶ
- C) 半年は無条件に同じ手法を続ける
- D) 心理面だけが原因と決めつける
**正解:** B
**解説:** 成績低下には、環境変化か運用ミスか原因があります。過去の成功実績があるほど見直しを遅らせがちです。現在のデータで検証し、修正や休止を選べる柔軟性が生存につながります。

## Q6
**問題:** 「前にこのゾーンで反転したから今回も反転する」と考えた。何を見落としている可能性が高いか？
- A) 過去に反転した場所は二度と機能しない事実
- B) ゾーン分析は成功体験と無関係であること
- C) 到達回数や注文の残り、上位足の環境が変わっている可能性
- D) 反転した場所では必ず出来高が増えること
**正解:** C
**解説:** 同じ価格帯でも、何度も試されると注文が減り、環境も変わります。前回の反転は参考情報であり、今回の根拠そのものではありません。鮮度や上位足の流れを改めて確認します。

## Q7
**問題:** 成功した1回のトレードを「自分の型」として固定したくなった。なぜ危険か？
- A) 成功した型は必ず次から使えなくなるから
- B) 型を持つこと自体が中級者には不要だから
- C) 固定すると記録が取りにくくなるから
- D) 1回の結果に過剰な意味を持たせ、統計的な検証を飛ばすから
**正解:** D
**解説:** 印象的な勝ちは、自分だけの発見に見えやすいものです。しかし1回の結果はサンプルにすぎません。100回規模の記録で評価する姿勢が、成功体験への執着を防ぎます。

## Q8
**問題:** 月次の手法棚卸しで最も確認すべきことはどれか？
- A) 先月の勝ちパターンが今月の環境でも同じ条件で機能しているか
- B) 一番利益が出たトレードだけを再現できるか
- C) 負けたトレードを記録から外せるか
- D) 毎月必ず新しい手法を追加できるか
**正解:** A
**解説:** 棚卸しの目的は手法探しではなく、環境との適合確認です。先月の勝ち筋が今月も同じ条件で機能しているかを見ることで、成功体験に縛られず微調整できます。

## Q9
**問題:** 「初心者の目」でチャートを見る練習が役立つ理由はどれか？
- A) 初心者の方が必ず正しい判断をするため
- B) 難しい分析を全て捨てるため
- C) 過去の成功記憶を一度脇に置き、今のチャートを先入観なく読むため
- D) 過去の検証を使わないため
**正解:** C
**解説:** 初心者の目とは、知識を捨てることではありません。過去の勝ち負けの記憶をいったん横に置き、今の価格、構造、出来高をそのまま見る姿勢です。これが執着のフィルターを弱めます。

## Q10
**問題:** 成功体験への執着が学習停止につながる場面はどれか？
- A) 勝ったトレードを記録に残す
- B) 負けた手法を一度休止する
- C) 環境に合わせて微調整する
- D) 「自分はもう分かっている」と感じ、新しい相場変化を検証しなくなる
**正解:** D
**解説:** 過去の成功は自信になりますが、過信になると学びを止めます。「もう分かっている」と感じた時ほど、現在の相場に合っているかを確認する必要があります。成長は成功後の検証で続きます。
