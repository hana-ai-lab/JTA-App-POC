# 第33章: メインシナリオとサブシナリオの設計法

## Q1
**問題:** シナリオの切り替え条件を「リアルタイムで判断」するときに最も起こりやすい問題は何か。
- A) エントリーのタイミングが遅れること
- B) スプレッドが広がってエントリーしにくくなること
- C) 感情が介入し、「もう少し待てばメインが復活するかも」と損失を拡大すること
- D) 出来高が減少して判断材料が不足すること
**正解:** C
**解説:** リアルタイムでの判断には必ず感情が介入し、「かも」という希望が損失を拡大させる。だからこそ切り替え条件は事前に定義し、機械的に実行する必要がある。

## Q2
**問題:** シナリオの切り替えトリガーのうち、VIXが30を超えて急上昇した場合の対応として正しいものはどれか。
- A) 通常通りメインシナリオを実行する
- B) サブシナリオに切り替えて逆方向にエントリーする
- C) 全シナリオを一時停止し、VIXが落ち着くまで待機する
- D) ロットを半分にしてエントリーする
**正解:** C
**解説:** VIXが30超で急上昇した場合、市場環境が急変しており通常のテクニカルが効きにくくなる。全シナリオを一時停止し、落ち着くまで待機するのが正しい対応。

## Q3
**問題:** 切り替えトリガーのうち、「価格がサポートゾーンを終値で下抜けた」場合のアクションとして正しいものはどれか。
- A) 損切りラインを広げてポジションを維持する
- B) 即座に逆方向にエントリーする
- C) エントリー条件を見直してロットを増やす
- D) サブシナリオのサポートゾーン（より深い位置）に移行する
**正解:** D
**解説:** メインのサポートゾーンが終値で下抜けた＝ゾーンが機能しなかったことを意味する。ここではメインを破棄し、サブシナリオで設定したより深い位置のサポートゾーンに移行する。

## Q4
**問題:** 「計画的な切り替え」と「リベンジトレード」の最も本質的な違いは何か。
- A) エントリーする方向が同じか逆かの違い
- B) 事前に定義した条件に基づいているか、損切り後の感情に基づいているかの違い
- C) ポジションサイズの大小の違い
- D) 使用する時間足の違い
**正解:** B
**解説:** 切り替えは事前に定義した条件に基づく計画的な方向転換であり、リベンジは損切り後の感情的な方向転換。見た目は似ていても本質は全く異なる。書き出していない条件で方向を変えたらリベンジの可能性が高い。

## Q5
**問題:** ゾーンに到達したがトリガーが出ないまま価格が反対方向に動いた場合、適切な対応はどれか。
- A) ゾーンに到達した事実だけでエントリーする
- B) 即座に逆方向にエントリーする
- C) トリガーなしの見送りに切り替え、次のゾーンまで待機する
- D) 損切りラインを広げて待機を続ける
**正解:** C
**解説:** ゾーン到達だけでトリガーが出ない場合は「待つ→見送り」に切り替える。トリガーなしにエントリーすれば確度の低いトレードになる。次の機会を待つのが正しい判断。

## Q6
**問題:** If-Then-Elseの条件分岐構造をトレードに応用する最大のメリットは何か。
- A) 「想定外」がなくなり、どんな値動きにも冷静に対応できること
- B) エントリー回数を最大化できること
- C) テクニカル指標の種類を減らせること
- D) 利確目標を自動的に計算できること
**正解:** A
**解説:** If-Then-Elseで全ての分岐を事前に定義しておけば、どんな値動きが来ても「これはIf-Aのケースだな」「これはElseのケースだな」と冷静に対応でき、「想定外」がなくなる。

## Q7
**問題:** 3段階のIf-Then-Elseにおいて、「If D：サブシナリオのサポートゾーンも下抜けた」に該当した場合の正しいアクションはどれか。
- A) 新しいゾーンを探して第3のシナリオを作る
- B) ロットを縮小してもう一度エントリーを試みる
- C) 損切りラインを大幅に広げて待機する
- D) 全シナリオを無効化し、その通貨ペアは当日トレードしない
**正解:** D
**解説:** メインもサブも機能しなかった場合、全シナリオを無効化する。無理に新しいシナリオを作ろうとすると焦りからバイアスのかかった分析になる。その日は休むのが最も賢い判断。

## Q8
**問題:** 「Else：上記のいずれにも該当しない状態」に該当した場合、どのようなアクションが適切か。
- A) チャートを閉じてトレードを完全にやめる
- B) アラートを設定して待つ——まだ何も起きていない状態
- C) 直感でエントリーする
- D) ロットを増やしてエントリーの機会を逃さない
**正解:** B
**解説:** どのIf条件にも該当しない＝まだ何も起きていない状態。アラートを設定して待機するのが適切。分岐図のElseは「待つ」という立派な判断を明示的に定義している。

## Q9
**問題:** If-Then-Elseの分岐図をノートに書くべき理由として最も重要なものはどれか。
- A) 日中の判断を全て「計画の実行」に変え、地図なしで歩く状態を避けるため
- B) SNSで共有して他者の意見を聞くため
- C) 過去のトレード記録として保存するため
- D) エントリーのタイミングを秒単位で指定するため
**正解:** A
**解説:** 分岐図をノートに書くことで、日中のすべての判断が「計画の実行」になる。分岐図がない状態は「地図なしで知らない街を歩く」のと同じであり、感情に流されやすくなる。

## Q10
**問題:** 「If B：価格がサポートゾーンに到達したが、トリガーが出ない」の場合、見送りに切り替える具体的な目安はどれか。
- A) 10本のローソク足が経過してもトリガーが出ない場合
- B) 2本のローソク足以内にトリガーが出ない場合
- C) 1時間経ってもトリガーが出ない場合
- D) 出来高が減少し始めた場合
**正解:** B
**解説:** ゾーン到達後に2本のローソク足以内にトリガーが出なければ見送りに切り替える。いつまでも待ち続けると「待つ」ことが「執着」に変わるため、明確な期限を設定する。
