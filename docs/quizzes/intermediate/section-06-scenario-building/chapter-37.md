# 第37章: シナリオ管理シート — 毎日の準備ルーティン

## Q1
**問題:** ケーススタディで「メインシナリオのサポートゾーンで完璧なピンバーが確定し出来高も急増した」場合（If A）の対応として、やってはいけないことはどれか？
- A) 計画通りの損切り・利確を実行する
- B) 「もっと利益を取れる」と利確目標を途中で延ばす
- C) エントリーを実行する
- D) 計画した損切りラインを維持する
**正解:** B
**解説:** If Aが発動した最良の場面でも、「もっと行けるはず」と利確目標を途中で変えることは「エントリー後のシナリオ変更」にあたる。感情的な判断が紛れ込む典型的なパターン。計画通りに実行することが規律。

## Q2
**問題:** ケーススタディで「メインのゾーンに到達したがトリガーが出ず見送った」（If B）場合の正しい心構えはどれか？
- A) ゾーン到達時に早めにエントリーしておけばよかった
- B) 「トリガーなしの見送り」は正しい判断であり、規律が機能している証拠
- C) より下位足でトリガーを探せばよかった
- D) シナリオが間違っていた
**正解:** B
**解説:** If Bで見送ることができた事実は「基準を守れた」ことを意味する。結果論で「早く入ればよかった」と思うのは間違い。基準を守ることが長期的な成功の根拠になる。

## Q3
**問題:** 「メインのサポートゾーンを下抜け→サブシナリオの深い位置のゾーンに移行」した後、そのサブゾーンでもトリガーが出なかった場合の正しい対応はどれか？
- A) さらに深い第3のゾーンを探してエントリーを試みる
- B) ノートレード（全シナリオを一時停止）にして翌日に改めて環境認識をする
- C) 逆方向にエントリーする
- D) 損切りラインを大幅に広げてエントリーを維持する
**正解:** B
**解説:** メインもサブも機能しなかった場合は「相場が自分のシナリオと合っていない日」。無理に第3のシナリオを作ろうとすると焦りからバイアスのかかった分析になりやすい。当日は休んで翌日に冷静な目で再スタートする。

## Q4
**問題:** ケーススタディで全シナリオが無効化された後、「今日は休む」と決めることが難しい心理的な理由は何か？
- A) チャートを見る習慣があるため
- B) 「せっかくチャートを開いたのだから何かしたい」「今日中に取り返したい」という心理が無理なトレードを促すから
- C) 休むと技術が低下するから
- D) 他のトレーダーが稼いでいる焦りから
**正解:** B
**解説:** 「チャートを開いた後に何もしない」は心理的に難しい。しかし根拠のないトレードは期待値がマイナスになりやすい。「トレードしない」も立派な判断で、口座を守ることが最優先。

## Q5
**問題:** 重要な経済指標発表でファンダメンタルズの前提が変わった場合、全シナリオを一時停止する判断が正しい理由はどれか？
- A) 指標後は必ず相場が急変するから
- B) テクニカルシナリオはファンダメンタルズの前提の上に立っており、その前提が変わると全シナリオの根拠が失われるから
- C) 指標発表後は流動性がなくなるから
- D) 大口が一時的に相場から退場するから
**正解:** B
**解説:** テクニカル分析は現在の相場環境を前提にしている。重要指標で方向感が大きく変わると、その前提が崩れる。新しいファンダメンタルズの前提で環境認識を再構築してからシナリオを立て直すのが合理的。

## Q6
**問題:** シナリオを立てる通貨ペア数を絞ること（毎日2〜3ペア程度）で得られる最も本質的なメリットはどれか？
- A) 分析にかかる時間が短くなるから
- B) ペアを絞ると各シナリオへの集中力が高まり、判断の質が向上するから
- C) 少ないペアほど相場予測の精度が上がるから
- D) ブローカーの推奨に合わせるため
**正解:** B
**解説:** 複数のシナリオを高い品質で管理するには集中力が必要。管理対象が増えるほど各シナリオへの注意が分散して判断が雑になる。「少なく深く」に絞ることで、1つひとつのエントリーの精度を維持できる。

## Q7
**問題:** シナリオ管理シートの習慣を1週間続けた後の変化として最も自然に期待できるものはどれか？
- A) 全てのトレードが勝ちに変わる
- B) シナリオの分岐構造が頭に染み込み、チャートを開いた瞬間に「どのIfに当たるか」が自然に浮かぶようになる
- C) 環境認識に必要な時間が大幅に短縮される
- D) 損切りが一切なくなる
**正解:** B
**解説:** 繰り返しによって分岐構造が思考習慣になる。意識的に考えなくても「今はElseの状態だな」と自然に判断できるようになる。この「自動化」が精神的な余裕を生み、より冷静なトレードを可能にする。

## Q8
**問題:** 「エントリー後にシナリオを変更しない」ルールの実行が難しい理由はどれか？
- A) 市場の動きを予測できないから
- B) ポジションを持った状態では「ポジションを有利に見たい」という無意識の欲求が判断を歪めるから
- C) チャートが見づらくなるから
- D) ブローカーが制限するから
**正解:** B
**解説:** ポジションを持つと「願望的解釈」が始まる。「この動きはシナリオ通りでは？」「まだいけるかも」という歪んだ解釈が、エントリー後のシナリオ変更を合理化する。エントリー後の変更は感情由来と考えて基準を守る。

## Q9
**問題:** シナリオ管理シートの「朝のルーティン（わずか数分の追加）」が日中の精神的な安定に直結する理由はどれか？
- A) 運動後のリラックス効果と同じ原理だから
- B) 朝に全ての分岐と対応を書き出しておくことで、日中の全ての出来事が「計画の実行」に変わり迷いがなくなるから
- C) 朝の分析は夜の分析より精度が高いから
- D) 習慣化すると相場予測が上達するから
**正解:** B
**解説:** 「何が起きても計画通り」という状態は精神的な安定をもたらす。朝に分岐図を書いておくと、日中は「計画の実行者」として動けるため、不確実な相場でも焦りを感じにくくなる。

## Q10
**問題:** 複数週にわたってシナリオ管理シートを継続したトレーダーが「損益の質が改善した」と感じる最も大きな要因はどれか？
- A) シナリオが当たる頻度が増えたから
- B) 計画なきエントリー（感情的・衝動的）が減り、根拠のある場面だけでトレードするようになったから
- C) 勝率が統計的に上がるから
- D) 相場の予測力が向上したから
**正解:** B
**解説:** シナリオ管理により「計画にないトレード」が自然と減少する。衝動的なトレードは期待値が低いため、それが減るだけで損益の質が改善する。予測の精度が上がるのではなく、低品質なトレードが減ることが改善の正体。
