# Week 8: 負けパターン22種 — 自己診断

## Q1
**問題:** 「上がっているのを見て飛び乗ったら天井だった」という経験に最も当てはまる負けパターンの名称はどれですか？
- A) 追いかけエントリー
- B) トリガーなしエントリー
- C) 感情エントリー
- D) 逆張り病
**正解:** A
**解説:** 追いかけエントリーは、すでに大きく動いた後についていき、天井で掴まされる典型パターンです。対策は「待つべき場所で待つ」ことです。

## Q2
**問題:** 逆張り病の本質的な問題として正しいものはどれですか？
- A) トレンドの転換点を正確に予測することが技術的に不可能だから
- B) 逆張りは常にトレンド追随よりもリスクが高いと数学的に証明されているから
- C) 逆張りエントリーはブローカーの約定システムに不利なスリッページが発生するから
- D) トレンドは想像よりも長く続くため、「そろそろ反転するだろう」という予想が外れやすいから
**正解:** D
**解説:** 逆張り病は「そろそろ反転するだろう」という主観的な予想に基づきますが、トレンドは想像より長く続くのが相場の性質であり、逆張りは損失を膨らませやすい手法です。

## Q3
**問題:** 「価格がラインに到達したから」という理由だけで確認なく飛び込む負けパターンを何と呼びますか？
- A) 感情エントリー
- B) 追いかけエントリー
- C) トリガーなしエントリー
- D) 分析麻痺
**正解:** C
**解説:** トリガーなしエントリーは、価格が目標地点に到達したことだけで飛び込み、ローソク足の確定や追加の確認条件を待たない失敗パターンです。

## Q4
**問題:** 感情エントリーの対策として提示されている「5秒ルール」とは具体的にどのような方法ですか？
- A) エントリー後に5秒待って損失が拡大しないか確認する
- B) 5秒間チャートを見つめ続けてエントリー根拠を探す
- C) エントリーボタンを押す前に5秒止まり、「今、冷静か？」と自分に問う
- D) 5秒以内にエントリーを完了させないとチャンスが消えるという認識を持つ
**正解:** C
**解説:** 5秒ルールはエントリー前に5秒止まって冷静さを確認する方法です。「ワクワクする」だけで飛び込む感情エントリーを防止する具体的な歯止めです。

## Q5
**問題:** 「情報を集めすぎて結局何もできない」状態を表す負けパターンの名称はどれですか？
- A) 確証バイアス
- B) ルーティン無視
- C) 感情エントリー
- D) 分析麻痺
**正解:** D
**解説:** 分析麻痺は情報を集めすぎて判断できなくなる状態です。「わからない」と判断できるなら見送りが正解であり、無理にエントリーしないことが重要です。

## Q6
**問題:** 損切りできない病の背景にある心理的効果として、「サンクコスト」と「保有効果」が挙げられています。このうちサンクコスト効果とはどのような心理ですか？
- A) ポジションを保有している期間が長いほど愛着が湧いて手放せなくなる心理
- B) すでに投入した資金や時間を回収しようとして、損失を手放せない心理
- C) 過去に大きな利益を出した銘柄に何度もエントリーしてしまう心理
- D) 損切りした瞬間に相場が反転するのではないかと恐れる心理
**正解:** B
**解説:** サンクコスト効果は、すでに投入したコスト（時間・資金）を無駄にしたくない心理で、損失を抱えたポジションを手放せない原因になります。

## Q7
**問題:** 損切りできない病に対する対策として提示されている「今ノーポジなら入るか？」という問いかけが効果的な理由はどれですか？
- A) 現在のポジションを一旦決済してエントリーし直すことで心理がリセットされるから
- B) 含み損のサンクコスト感情を排除し、純粋にエントリー根拠だけを客観的に評価できるから
- C) ノーポジ状態で考えると常にエントリーしない方がよいという結論になるから
- D) 問いかけることで時間が経過し、自然と相場が好転する確率が上がるから
**正解:** B
**解説:** 「今ノーポジなら入るか？」と問うことで、含み損のサンクコスト感情を排除し、純粋にエントリー根拠の有無だけを客観的に評価できます。

## Q8
**問題:** 22種の負けパターンのうち「最も破壊力がある」とされているものはどれですか？
- A) 追いかけエントリー
- B) 連勝後のロット上げ
- C) 経済指標無視
- D) リベンジトレード
**正解:** D
**解説:** リベンジトレードは「最も破壊力がある」とされる負けパターンです。損失を取り返そうとする感情が全てのルールを吹き飛ばし、連鎖的な大損を招きます。対策は30分間チャートを閉じることです。

## Q9
**問題:** エントリー後にシナリオを変更する行為について、「エントリー後の変更はほぼ100%何に基づく」と説明されていますか？
- A) 新しいテクニカル根拠
- B) 相場環境の変化
- C) 感情
- D) 経済指標の発表結果
**正解:** C
**解説:** エントリー後のシナリオ変更はほぼ100%感情に基づきます。含み損を避けたい、利益を伸ばしたいなどの感情が、事前に立てた計画を書き換える原因になります。

## Q10
**問題:** 連勝後のロット上げが危険な理由として最も的確なものはどれですか？
- A) 連勝後はブローカーが約定レートを不利に操作する仕組みになっているため
- B) 連勝中は相場の環境が良いだけで技術が向上しているわけではないため、次の1敗でそれまでの利益を全て消し去る可能性がある
- C) ロットを上げると強制決済のリスクが高まる仕組みになっているため
- D) 連勝後のエントリーは勝率が統計的に下がることが証明されているため
**正解:** B
**解説:** 連勝中は「ついてる！」と勢いに乗りやすく、ロットを上げた状態で1回負けるとそれまでの利益を全て消し去るリスクがあります。ロットは事前計画で固定すべきです。

## Q11
**問題:** 連敗後のロット上げはどの負けパターンの変形とされていますか？
- A) 感情エントリー
- B) シナリオ変更病
- C) 追いかけエントリー
- D) リベンジトレード
**正解:** D
**解説:** 連敗後のロット上げはリベンジトレードの変形です。損失を取り返すためにロットを増やす行為は、リベンジ心理そのものです。連敗後は逆にロットを半分にするのが推奨されています。

## Q12
**問題:** 5分足で「上昇トレンド」と判断して買いエントリーしたが、日足では下降トレンドだったという失敗は、どの負けパターンに当てはまりますか？
- A) 下位足だけ見る病
- B) 確証バイアス
- C) フェイクブレイク
- D) 分析麻痺
**正解:** A
**解説:** 下位足だけ見る病は、短い時間足の局部的な動きに反応してエントリーし、上位足の大局的なトレンドと矛盾していることに気づかない失敗パターンです。

## Q13
**問題:** 確証バイアスのトレードにおける典型的な現れ方として正しいものはどれですか？
- A) 自分のポジション方向に有利な情報だけを集め、不利な情報を無視する
- B) 複数の時間足を確認せずに一つのチャートだけで判断する
- C) エントリー後に損切りラインを何度も下方修正する
- D) 他人のトレード結果を見て自分の判断を変える
**正解:** A
**解説:** 確証バイアスは「自分が信じたいこと」を裏付ける情報だけを集め、矛盾する情報を無視する認知バイアスです。買いポジションを持っていれば上昇の根拠ばかり探してしまいます。

## Q14
**問題:** フェイクブレイクに引っかかるトレーダーの典型的な行動として正しいものはどれですか？
- A) 一瞬だけラインを抜けたのを見て飛び込むが、すぐに戻されて損切りになる
- B) レンジ相場ではエントリーせず、明確にブレイクした後だけエントリーする
- C) ブレイクした後に数本ローソク足を待ってからエントリーする
- D) 複数の時間足でブレイクを確認してから慎重にエントリーする
**正解:** A
**解説:** フェイクブレイクは一瞬だけラインを抜ける現象で、これに飛び込むとすぐに戻されて損切りになります。ブレイクの確認にはローソク足の確定や再テストを待つことが重要です。

## Q15
**問題:** 利確早すぎ病の対策として提示されているものはどれですか？
- A) 利確後にすぐ次のエントリーをして利益を積み重ねる
- B) 含み益が減る不安をなくすために常に両建てにする
- C) 利確目標を事前に設定し、そこに到達するまでポジションを維持する
- D) 利確のタイミングは直感に頼り、含み益が出たらすぐに手放す
**正解:** C
**解説:** 利確早すぎ病は少しの含み益で焦えて確定してしまう癖です。対策は利確目標を事前に設定し、目標に到達するまでポジションを維持するルールを守ることです。

## Q16
**問題:** 「今日だけ特別」症候群について、「今日だけ」と思った瞬間に何が最も危険だと言われていますか？
- A) その瞬間がその日の最もエントリーに適したタイミングである可能性が高いから
- B) 「今日だけ」と思った瞬間こそが最も危険な状態であり、ルールを破る入口だから
- C) 「今日だけ」と考えたトレードは勝率が統計的に低いことが分かっているから
- D) 特別な日は相場のボラティリティが高くなり予測不能になるから
**正解:** B
**解説:** 「今日だけルールを少し緩めてもいい」「今日だけロットを上げてもいい」と思った瞬間が最も危険です。この思考がルール崩壊の入口であり、一度崩すと戻りにくくなります。

## Q17
**問題:** 22種の負けパターンのうち「最も地味で最も根深い」とされているものはどれですか？
- A) ルーティン無視
- B) 損切りできない病
- C) 記録しない
- D) 確証バイアス
**正解:** C
**解説:** 「記録しない」は最も地味で最も根深い負けパターンです。記録がないと自分の失敗パターンに気づけず、同じ過ちを無限に繰り返すことになります。対策は1トレード3行でも構わないので書くことです。

## Q18
**問題:** 事前分析なしにエントリーしてしまう負けパターンの名称はどれですか？
- A) ルーティン無視
- B) 経済指標無視
- C) トリガーなしエントリー
- D) 記録しない
**正解:** A
**解説:** ルーティン無視は事前の分析手順を飛ばしてエントリーする習慣です。毎日のルーティンを守ることが、感情に流されないトレードの土台になります。

## Q19
**問題:** トレンド終盤に参入したトレーダーが巻き込まれやすい現象はどれですか？
- A) スプレッドが極端に狭くなり利益が出にくくなる
- B) 大口の売り抜けに巻き込まれて天井で掴まされる
- C) 相場のボラティリティが低下してポジションが膠着する
- D) ブローカーのシステムトラブルで決済できなくなる
**正解:** B
**解説:** トレンド終盤への参入は、すでに利益を確保した大口投資家の売り抜けに巻き込まれるリスクが高いです。長く続いたトレンドの最後に飛び乗るのは典型的な負けパターンです。

## Q20
**問題:** 「追いかけエントリー＋損切りできない＋リベンジ」の複合パターンが引き起こす結果として最も正しいものはどれですか？
- A) 手数料負けしてコツコツ損失が積み重なる
- B) 一つのトレードで大きな損失を出すが、そこで止まる
- C) 感情的なエントリーになるが、損切りで最小限の損失に抑えられる
- D) 連鎖的な大損に繋がる
**正解:** D
**解説:** 追いかけエントリーで天井を掴み、損切りできずに含み損を抱え、最後にリベンジトレードでさらに損失を拡大する連鎖は、最も破壊的な負けパターンの組み合わせです。一つの失敗が次の失敗を呼ぶ連鎖的大損に繋がります。
