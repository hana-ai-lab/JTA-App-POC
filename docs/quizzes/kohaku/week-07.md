# Week 7: 「やらない」という最強の判断

## Q1
**問題:** あるトレーダーが「根拠が薄い」と感じながらもエントリーボタンを押すことが止められない。勝負の瞬間に得られる快感がこの行動を繰り返させている。この「止められない」仕組みの正体は何ですか？
- A) エントリーの瞬間に脳内でドーパミンが放出され、結果に関わらず報酬回路が活性化するから
- B) ポジションを持つことで相場への集中力が高まり、良い判断ができるようになるから
- C) 取引回数が多いほど統計的に勝率が安定していくという期待があるから
- D) エントリーしないと相場の流れに乗り遅れるという正当な危機感があるから
**正解:** A
**解説:** エントリーの瞬間にドーパミンが放出され、結果に関係なく「報酬を得た感覚」が生まれます。この快感回路が根拠のないエントリーを繰り返させます。「押したい衝動」を感じたとき、それ自体が警戒のサインです。

## Q2
**問題:** あるトレーダーは明確なセットアップがないのに「何かしなければ」と焦りを感じてエントリーしてしまう。翌日振り返ると「なぜ入ったのか分からない」という状況が毎週続いている。この行動の根本原因は何ですか？
- A) チャートを長時間見ることで、実際にはないパターンが見えるようになるから
- B) 常にポジションを持っていないと不安になる心理が、根拠のないトレードを駆り立てているから
- C) エントリーしないと資金が時間とともに目減りするという誤った認識があるから
- D) 集中力が高まると根拠が薄くても直感的に正しいエントリーができると思っているから
**正解:** B
**解説:** 「ポジションがないと落ち着かない」感覚はFOMOとドーパミン報酬系が組み合わさって生じます。この衝動に気づかずエントリーを繰り返すと手数料が積み重なり口座を削ります。「焦り」を感じたらまずチャートから離れることが最初の対策です。

## Q3
**問題:** 月20回トレードして勝率50%、リスクリワード比1:1の場合、手数料を含めた最終的な損益はどうなりますか？
- A) 勝率50%なのでプラマイゼロで終わる
- B) トレード回数が多いほどスプレッドが優遇されてプラスになる
- C) 手数料は利益からのみ引かれるので、損失は増えない
- D) 勝敗が同数でも手数料分だけマイナスになる
**正解:** D
**解説:** 勝率50%・RR1:1では利益と損失が相殺され、全取引にかかる手数料だけが確実に差し引かれます。回数が多いほど手数料負担が膨らみ、結果的にマイナスになります。

## Q4
**問題:** あるトレーダーが「今度こそ当たる」という期待が止まらず、負けても翌日また根拠の薄いエントリーをしてしまう。この「止められない」感覚を生み出している仕組みは何ですか？
- A) 損失が続くと次は当たるという確率論的な期待が正当化されるから
- B) 不確実な結果に対してドーパミンが放出される脳の仕組みが、繰り返しを促しているから
- C) 相場に参加し続けることで統計的に勝率が収束していくという誤解があるから
- D) ポジションを持つことで市場の動きが体感的に把握しやすくなると感じているから
**正解:** B
**解説:** 「当たるかもしれない」という不確実性こそがドーパミンを放出させ、快感を生みます。スロットと同じ原理です。この仕組みを知ることで、衝動的なエントリーに「待て」をかける理由が腹落ちします。

## Q5
**問題:** 月5回・勝率60%・RR1:2のトレードが月20回・勝率50%・RR1:1より成績が良い最大の理由は何ですか？
- A) トレード回数が少ないほどブローカーのスプレッドが狭くなるから
- B) エントリー基準を厳しくすることで低確率のトレードを排除できるから
- C) 勝率60%を維持すれば回数に関係なく常にプラスになるから
- D) 月5回以下なら手数料が無料になる仕組みがあるから
**正解:** B
**解説:** 回数を絞ることで「本当に良いセットアップ」だけを厳選するようになり、勝率とリスクリワード比が同時に向上します。質の低いトレードを削り切ることが成績向上の鍵です。

## Q6
**問題:** 「待つこと」に経済的価値がある理由として正しいものはどれですか？
- A) トレードしない期間が長いほどブローカーから特別なレートが提供されるから
- B) 待てば待つほどエントリーポイントの精度が上がるから
- C) 余計な損失を防ぎ、次のチャンスにフルの資金で臨める状態を保てるから
- D) 待っている間に相場のトレンドが自動的に予測できるようになるから
**正解:** C
**解説:** 無駄なトレードで資金を削らずに済むことで、本当に良いチャンスが来たときに最大限のポジションサイズで臨めます。資金を温存すること自体が戦略的な行動です。

## Q7
**問題:** あるトレーダーがSNSで他のトレーダーの大きな利益を見た瞬間、「このチャンスを逃すわけにいかない」と感じてルール外のエントリーをしてしまいました。この感情が繰り返しトレードを誤らせる理由は何ですか？
- A) SNSの利益報告は実績データなので、同じ方向に乗らないと機会損失になるから
- B) 相場は多くのトレーダーが同じ方向を向いたときに動くため、追従することが有効だから
- C) 「見逃すことへの恐怖」が冷静な判断より強く働き、根拠のないエントリーを引き起こすから
- D) チャンスを見逃すと次のセットアップが来るまで長く待つことになるから
**正解:** C
**解説:** 「見逃すことへの恐怖」は実際のリスクより「チャンスを逃す痛み」を大きく感じさせます。この感情に従うと根拠のないエントリーになり損失を重ねます。SNSを見て焦りを感じたとき、それ自体がトレードを止めるサインです。

## Q8
**問題:** 重要な経済指標の発表前にトレードを見送るべき理由として正しいものはどれですか？
- A) ボラティリティが急拡大し、予測不能な値動きになるため
- B) 指標発表前はブローカーが注文を制限するため
- C) 指標発表前にエントリーするとスプレッドが3倍になるため
- D) 指標の結果がすでに価格に織り込まれており動きがないため
**正解:** A
**解説:** 重要指標の発表前後はボラティリティが急拡大し、スプレッドも広がりやすくなります。予測不能な値動きの中では技術的な根拠が通用しないため、見送るのが安全です。

## Q9
**問題:** 連敗3回した状態でさらにトレードを続けると陥りやすい危険な行動はどれですか？
- A) トレードルールを厳格にしてエントリー基準を高くする
- B) ポジションサイズを半分にして慎重にトレードする
- C) エントリーする銘柄を変えて気分を切り替える
- D) 損失を取り返そうとルールを無視したエントリーに走る
**正解:** D
**解説:** 連敗が続くと損失を取り返そうとする心理が強く働き、冷静さを失ってルールを無視したトレードに走りやすくなります。この「傾斜」に気づいて手を引くことが重要です。

## Q10
**問題:** 損切り直後30分以内にトレードすべきではない理由は何ですか？
- A) 損切りの執行に手数料が二重にかかるため
- B) 損切りの痛みから「すぐ取り返したい」という衝動に駆られやすいため
- C) ブローカーのシステムが30分間注文を制限するため
- D) 損切りした銘柄は30分以内に必ず反発するため、二度負ける確率が高い
**正解:** B
**解説:** 損切りの直後は損失の痛みが最も強く、「すぐに取り返したい」という衝動に駆られます。この感情は冷静な判断を奪うため、最低30分は距離を置くことが推奨されます。

## Q11
**問題:** 「取り返したい」と感じている状態でトレードすると、どのような結果になりやすいですか？
- A) 過去の損失に執着した根拠のないエントリーになり、さらに損失が膨らむ
- B) 怒りが原動力になるため、かえって集中力が上がって良い結果が出る
- C) 損失を取り返す目的があるため、自然的に慎重なトレードになる
- D) 無意識に損切りラインが厳しくなり、小さな負けで済む
**正解:** A
**解説:** 「取り返したい」という感情はサンクコスト効果の表れで、過去の損失に執着した根拠のないトレードを誘発します。冷静な判断が失われ、損失がさらに拡大しやすくなります。

## Q12
**問題:** 体調不良や寝不足の日にトレードを見送るべき本質的な理由は何ですか？
- A) 体調が悪いとチャートの色が見分けにくくなるから
- B) ブローカーが体調不良時の取引を推奨していないから
- C) 寝不足だとエントリーのタイミングが遅れて約定価格がずれるから
- D) 認知機能と感情制御が低下し、冷静な判断とルールの遵守が難しくなるから
**正解:** D
**解説:** 体調不良や寝不足は認知機能と感情制御を低下させます。トレードに必要な冷静な判断力とルール遵守が難しくなることが、見送るべき本質的な理由です。

## Q13
**問題:** SNSで他のトレーダーの利益画像を見て焦りを感じたとき、最も適切な対応はどれですか？
- A) 自分もすぐにエントリーして同じように利益を出す
- B) そのトレーダーの銘柄をコピーして同じポジションを取る
- C) トレードを見送り、SNSから離れて冷静になる
- D) 小額のポジションで気分を落ち着かせてから本番に戻る
**正解:** C
**解説:** SNSでの焦りはFOMOの典型的な引き金です。他人の結果は自分のトレードとは無関係であり、焦ってエントリーすると根拠のないトレードになるため、まずは画面から離れるべきです。

## Q14
**問題:** 「やらない日」の過ごし方として最も効果的なものはどれですか？
- A) デモトレードでエントリーの感覚を維持する
- B) 小額のリアルトレードで感覚を衰えさせない
- C) 過去トレードの振り返りやパターン観察の練習をする
- D) SNSでトレーダーの意見を収集して次の戦略を練る
**正解:** C
**解説:** トレードしない日は過去の振り返り、パターン観察、日誌の見返し、来週の計画などに使うのが最も効果的です。相場から離れつつも学習を深める有意義な時間になります。

## Q15
**問題:** 自分のトレードルールに合うセットアップが見つからない日があります。このとき取るべき行動はどれですか？
- A) エントリーせず、ルールに合うセットアップが現れるのを待つ
- B) チャートの形が良さそうならルール外でもエントリーする
- C) ルールをその場で少し緩めてエントリーの機会を作る
- D) 新しいテクニカル指標を試すために小額でエントリーする
**正解:** A
**解説:** 自分のルールに合うセットアップがない日は堂々とトレードしない日です。ルールを妥協してまでエントリーすることが、長期的な成績を悪化させる最大の要因になります。

## Q16
**問題:** 月20回トレードしていたトレーダーが月5回に絞ったところ、勝率が50%から60%に上がり口座の減少が止まりました。トレード回数を減らしただけで成績が改善した主な理由は何ですか？
- A) 取引回数が少ないとブローカーのスプレッドが優遇されるから
- B) 回数を絞ることで各トレードへの集中力が増し、技術的な精度が上がるから
- C) 根拠の薄いエントリーが排除され、手数料負担と低質なトレードが同時に消えたから
- D) 月5回以下のトレーダーは参加者が少ない有利な時間帯を自然に選べるから
**正解:** C
**解説:** 回数を絞ると「本当に良いセットアップだけ」に自然に絞られ、根拠の薄いエントリーが消えます。同時に毎回かかる手数料も激減します。「やらない判断」は消極的ではなく、成績を改善する積極的な戦略です。

## Q17
**問題:** 「待つこと」の経済的価値のうち、「感情がニュートラルな状態で判断できる」とはどういう意味ですか？
- A) 前のトレードの勝敗に影響されず、冷静に次のエントリー根拠を組み立てられるということ
- B) 待っている間に感情的な記憶が消去されるということ
- C) トレードしない期間が長いほど感情が麻痺して何も感じなくなるということ
- D) 待つことで相場に対する恐怖心が完全に消滅するということ
**正解:** A
**解説:** 前回のトレードで勝っても負けても感情が揺さぶられます。適切に待つことで感情がリセットされ、次のトレードを独立した冷静な判断で臨めるようになります。

## Q18
**問題:** 「連敗が3回続いている」「体調が悪い」「相場の方向感がない」の3条件が重なっているとき、このトレーダーが取るべき判断はどれですか？
- A) 体調不良だけでは理由として弱いので、方向感が出るのを待ってエントリーする
- B) 連敗が続いているため、損失を取り返すために早めにエントリーする
- C) 3条件すべてが「やらない」基準に該当するため、今日は完全にトレードを見送る
- D) 小額のポジションならリスクが低いため入っても問題ない
**正解:** C
**解説:** 方向感のなさ・連敗3回・体調不良はいずれもトレードを見送るべきサインです。複数の条件が重なっている日は判断力・冷静さ・相場環境のすべてが整っていません。「やらない」がこの日の最善のトレードです。

## Q19
**問題:** あなたが損切りした直後、SNSで他のトレーダーの利益画像を見て焦りを感じました。この状況で取るべき最善の判断はどれですか？
- A) 損切りの直後なので30分待ってから、SNSの銘柄を参考にエントリーする
- B) SNSの利益画像は信用できないので、自分の分析ですぐに別の銘柄を探す
- C) 焦りを感じているうちはエントリーせず、翌日までトレードを休む
- D) 損切り後の30分ルールを守りつつSNSから離れ、感情が落ち着くまで待つ
**正解:** D
**解説:** 損切り直後は「取り返したい」衝動、SNSはFOMOの引き金という二重の感情リスクがあります。30分ルールを守りSNSから離れることで、両方の感情に対応できます。

## Q20
**問題:** トレード回数を月20回から月5回に減らしたトレーダーが、勝率50%から60%に向上しました。この変化が起きた理由として最も説明できるものはどれですか？
- A) 手数料が減った分が利益に反映され、勝率の計算結果が上がっただけだから
- B) 月5回しかエントリーできないという制約がプレッシャーになり、集中力が上がるから
- C) 厳しい基準を満たすセットアップだけに絞ることで、低確率の見送り案件を排除できたから
- D) トレード回数が少ないほど運が良くなる傾向があるから
**正解:** C
**解説:** 回数を絞ることで「本当に良いセットアップ」だけを厳選するようになり、根拠の薄いトレードを排除できます。選択と集中によって勝率が向上するのが本質的な理由です。
