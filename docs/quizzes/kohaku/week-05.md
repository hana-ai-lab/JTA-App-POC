# Week 5: 4つの暴走感情 — 恐怖・欲・焦り・後悔

## Q1
**問題:** トレードで含み損を抱えた際、「損失を確定させたくない」という心理を引き起こす4つの暴走感情のうちどれに該当しますか？
- A) 焦り
- B) 恐怖
- C) 欲
- D) 後悔
**正解:** B
**解説:** 恐怖は「損を認めたくない」という心理から損切りの実行を阻害します。脳が「失うこと」を極端に嫌うため、損失の確定を避けようとしてポジションを持ち続けてしまいます。

## Q2
**問題:** トレードにおける「欲」の暴走として典型的な行動はどれですか？
- A) 利確目標に達しても「もっと伸びるかも」と持ち続けること
- B) 含み損が出た瞬間に損切りすること
- C) チャートが動き始めた瞬間に飛び乗ること
- D) 損切り直後の反転を気にして次の損切りを遅らせること
**正解:** A
**解説:** 欲の暴走は「もっと欲しい」という心理で、利確目標に達しても利益を伸ばそうとして持ち続ける行動に現れます。事前に決めた目標を超えても満足できず、結果的に利益を取りこぼす原因になります。

## Q3
**問題:** FOMO（Fear Of Missing Out）がトレードに与える影響として最も典型的なものはどれですか？
- A) 利確を早めすぎて利益を伸ばせないこと
- B) 損切りラインを必要以上に広げてしまうこと
- C) チャートが動き始めた瞬間に飛び乗り、高値掴みになること
- D) 連勝後にロットを下げて保守的になりすぎること
**正解:** C
**解説:** FOMOは「乗り遅れたくない」という焦りから、チャートが動き始めた瞬間にエントリーしてしまう心理です。エントリーの根拠がないタイミングで飛び乗るため、高値掴みになりやすい特徴があります。

## Q4
**問題:** 損切り直後にレートが反転したのを見て「切らなければよかった」と思い、次の損切りを遅らせる原因となる感情はどれですか？
- A) 恐怖
- B) 焦り
- C) 欲
- D) 後悔
**正解:** D
**解説:** 後悔は「あのとき、ああしていれば」と過去の行動を悔やむ感情です。損切り直後の反転を見ると「損切りしなければよかった」と後悔し、その体験が次の損切りを躊躇させる原因になります。

## Q5
**問題:** 感情の暴走を防ぐための「メタ認知の3ステップ」として正しい順番はどれですか？
- A) ルールに戻る → 気づく → 距離を置く
- B) 距離を置く → ルールに戻る → 気づく
- C) 気づく → 距離を置く → ルールに戻る
- D) 距離を置く → 気づく → ルールに戻る
**正解:** C
**解説:** メタ認知のステップは、まず「今、自分は何を感じているか」に気づき、次に「感情は一時的な脳の反応」と距離を置き、最後に「感情ではなくルールで動く」ように戻る、の順番です。

## Q6
**問題:** 含み益が出ているにもかかわらず「逆行したらどうしよう」と利確を早めてしまう心理の背景にある感情はどれですか？
- A) 恐怖
- B) 欲
- C) 焦り
- D) 後悔
**正解:** A
**解説:** 含み益がある場面でも「せっかくの利益が消えてしまうかもしれない」という恐怖が働き、計画通りの利確ポイントを待てずに早めに確定させてしまいます。恐怖は含み損だけでなく含み益の場面でも損失回避を引き起こします。

## Q7
**問題:** 連勝後に「今がチャンスだ」とロットを上げたくなる心理を、4つの暴走感情のどれで説明できますか？
- A) 恐怖
- B) 焦り
- C) 後悔
- D) 欲
**正解:** D
**解説:** 連勝による高揚感が「もっと稼げるかもしれない」という欲を刺激します。調子がいいときほど「今がチャンス」とロットを上げたくなりますが、これが欲の暴走による危険なサインです。

## Q8
**問題:** 4つの暴走感情が連鎖して大損に至る典型的なパターンとして正しいものはどれですか？
- A) 焦りで飛び乗る → 含み損 → 後悔 → 欲が暴走 → 恐怖で損切りできない
- B) 焦りで飛び乗る → 含み損 → 恐怖で損切りできない → 損失拡大 → 後悔 → 欲が暴走 → リベンジトレード
- C) 恐怖で損切りできない → 後悔 → 焦りで飛び乗る → 欲が暴走 → 大損
- D) 欲が暴走 → 高値掴み → 焦り → 恐怖で利確 → 後悔 → ナンピン
**正解:** B
**解説:** 感情の連鎖は「焦り（FOMO）で根拠なく飛び乗る」ことを起点に、含み損→恐怖で損切りできず→損失拡大→後悔→「取り返したい」欲が暴走→リベンジトレード→さらに負ける、という悪循環に陥ります。

## Q9
**問題:** トレード前の「10秒チェック」で行うべきことは何ですか？
- A) 「今、冷静か？」と自問し、Noならその日はトレードしない
- B) 直近の経済指標発表のスケジュールを確認する
- C) 過去の類似チャートパターンを確認する
- D) 今の資金残高と許容損失額を計算する
**正解:** A
**解説:** トレード前の10秒チェックは「今、冷静か？」と自分に問いかける行動です。答えがNoであれば、その日はトレードを見送ることが大きな損失を防ぐ最も確実な方法です。

## Q10
**問題:** SNSで他者の「爆益」投稿を見て「自分も乗らなければ」と焦ってエントリーする行動の根本にある感情はどれですか？
- A) 焦り
- B) 恐怖
- C) 欲
- D) 後悔
**正解:** A
**解説:** 他人の成果を見て「自分だけ取り残されている」と感じるのは焦りの典型的なパターンです。SNSの情報は結果の一部だけを切り取ったものであり、それに反応してエントリーするのは根拠のない感情トレードになります。

## Q11
**問題:** 「今回だけ」と思ったときの心理状態について正しい説明はどれですか？
- A) 「今回だけ」と思ったとき、それは欲が暴走しているサインである
- B) 「今回だけ」は損切りタイミングを見極めた慎重な行動の表れである
- C) 「今回だけ」は過去の検証に基づく例外的なエントリーの合図である
- D) 「今回だけ」は冷静にリスクを評価した上での合理的な判断である
**正解:** A
**解説:** 「今回だけ特別」「今回だけはルールを逸脱してもいい」という思考は、欲が理性を上回っている状態です。ルールを破る言い訳を自分に許している時点で、すでに感情が判断を支配しています。

## Q12
**問題:** 恐怖に対する有効な対処法として提示されている問いかけはどれですか？
- A) 「このポジションの含み損は資金の何%にあたるか？」
- B) 「もし今ポジションを持っていなかったら、このレートでエントリーするか？」
- C) 「過去に同じチャートパターンで勝ったことはあるか？」
- D) 「損切りした後にレートが反転する確率は何%か？」
**正解:** B
**解説:** 「もし今ポジションを持っていなかったら、このレートで新規にエントリーするか？」と自問し、答えがNoであれば即座に損切りするのが恐怖への対処法です。保有している事実から離れて純粋に相場を評価することで、恐怖による判断の歪みを修正できます。

## Q13
**問題:** 焦り（FOMO）に対する対処法として提示されている問いかけと、その答えの正しい組み合わせはどれですか？
- A) 「このエントリーの根拠は何か？」→「チャートの勢い」
- B) 「このトレードを見送ったら何を失うか？」→「大きな利益を失う」
- C) 「このトレードを見送ったら何を失うか？」→「何も失わない」
- D) 「今エントリーしないとどうなるか？」→「二度とチャンスが来ない」
**正解:** C
**解説:** 焦りへの対処は「このトレードを見送ったら何を失うか？」と自問することです。答えは「何も失わない」であり、1回のトレードを見送っても口座の資金は減りません。この事実を思い出すことで焦りを冷静に処理できます。

## Q14
**問題:** 後悔の感情に対する正しい対処法はどれですか？
- A) 次は損切りしないという方針に切り替える
- B) 「結果」ではなく「プロセス」で評価し、ルール通りなら正しい行動とみなす
- C) 過去のトレードを詳細に振り返り、最適なタイミングを探る
- D) 反転の確率が高い場面では損切りを見送る戦略に変更する
**正解:** B
**解説:** 後悔への対処は、結果ではなくプロセスで自分を評価することです。ルール通りに損切りを実行したのであれば、その後レートがどう動いたかにかかわらず正しい行動でしたと評価すべきです。

## Q15
**問題:** 欲の暴走を防ぐための対処法として正しいものはどれですか？
- A) 利確目標に達したら、さらに上値を狙えるか1分だけ検討する
- B) 連勝時はロットを段階的に上げて勢いに乗る
- C) 利確目標とロットを事前に決め、機械的に実行する
- D) 利確目標は相場の動きに合わせて毎回柔軟に変更する
**正解:** C
**解説:** 欲への対処は、利確目標とロットをトレード前に決めておき、機械的に実行することです。トレード中の判断は感情に左右されるため、事前の冷静な状態で決めた数値に忠実に従うことが重要です。

## Q16
**問題:** トレードにおける「感情」について正しい考え方はどれですか？
- A) 感情はトレードの敵であり、感情を完全に排除して初めて勝てるようになる
- B) 感情をゼロにする必要はない。大切なのは「自分が感情的になっている」と気づくことである
- C) 感情は無視すべきで、気づく必要もなくただルールに従えばよい
- D) 感情は初心者だけの問題であり、経験を積めば自然に消えるものである
**正解:** B
**解説:** 感情は敵ではなく「気づくべきシグナル」です。怖くてもいい、悔しくてもいい。感情をゼロにする必要はなく、大切なのは「今、自分が感情的になっている」と自覚し、感情にコントロールされないことです。

## Q17
**問題:** メタ認知の3ステップのうち、「感情は一時的な脳の反応である」と認識するステップはどれですか？
- A) 気づく
- B) 振り返る
- C) ルールに戻る
- D) 距離を置く
**正解:** D
**解説:** 「距離を置く」ステップでは、自分の感情を客観視し、「これは一時的な脳の反応にすぎない」と認識します。感情そのものを否定せず、感情と自分を切り離して俯瞰することで冷静さを取り戻します。

## Q18
**問題:** 「今、自分は何を感じているか？」と自問するのは、メタ認知の3ステップのうちどれに該当しますか？
- A) 距離を置く
- B) ルールに戻る
- C) 気づく
- D) 振り返る
**正解:** C
**解説:** 「今、自分は何を感じているか？」に気づくことがメタ認知の最初のステップです。自分の感情状態を言語化することで、感情に飲まれるのではなく感情を客体として認識できるようになります。

## Q19
**問題:** 損切りした後にレートが反転した経験が、次のトレード行動に与える悪影響として最も典型的なものはどれですか？
- A) 次のトレードで利確を早めすぎて利益を取りこぼすこと
- B) 次のトレードで過剰にロットを上げて一発逆転を狙うこと
- C) 次のエントリーを見送り続けてトレード機会を失うこと
- D) 次の損切りを遅らせて、結果的に大損すること
**正解:** D
**解説:** 損切り直後の反転を見ると「切らなければよかった」と後悔し、その経験から次の損切りを躊躇するようになります。損切りのタイミングを遅らせることで小さな損失が大きな損失に育つのが最も典型的な悪影響です。

## Q20
**問題:** 感情の連鎖パターンにおいて、大損後に「取り返したい」という欲が暴走して起こる行動はどれですか？
- A) 損切りラインを厳しくしてリスクを減らすこと
- B) しばらくトレードを休んで冷静になること
- C) ロットを下げて慎重に取り戻そうとすること
- D) リベンジトレードをしてさらに負けること
**正解:** D
**解説:** 感情の連鎖では、大損による後悔が「取り返したい」という欲を暴走させ、リベンジトレードを引き起こします。感情的な状態でのエントリーは根拠が乏しく、さらに負ける確率が高くなる悪循環を生みます。
