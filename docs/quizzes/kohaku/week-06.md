# Week 6: リベンジトレードの罠

## Q1
**問題:** リベンジトレードの典型的な流れとして正しい順序はどれですか？
- A) 根拠のないエントリー → 負ける → ロットを上げる → さらに負ける → 「取り返したい」思考が支配
- B) 「取り返したい」思考が支配 → 負ける → ロットを上げる → 根拠のないエントリー → さらに負ける
- C) 負ける → 「取り返したい」思考が支配 → 根拠のないエントリー → ロットを上げる → さらに負ける
- D) ロットを上げる → 負ける → 「取り返したい」思考が支配 → さらに負ける → 根拠のないエントリー
**正解:** C
**解説:** リベンジトレードは「負ける」ことを起点に、「取り返したい」感情が思考を支配し、根拠のないエントリーやロット増加へつながり、悪循環が繰り返されます。

## Q2
**問題:** リベンジトレードに陥るプロセスの最終的な帰結として最も適切なものはどれですか？
- A) 一時的な含み損を抱えるが、相場が戻れば回復できる状態
- B) 悪循環を繰り返し、退場への片道切符となる状態
- C) 精神的な疲労は増すが、資金管理さえ守れば退場は免れる状態
- D) メンタルが鍛えられ、結果的に冷静なトレードができるようになる状態
**正解:** B
**解説:** リベンジトレードは負け→感情→無謀な行動→さらに負けるのサイクルが繰り返され、最終的には退場に直結します。

## Q3
**問題:** 「あの3万円を取り戻さないと」という思考を引き起こす心理的要因の名称は何ですか？
- A) サンクコスト効果
- B) 保有効果
- C) 確証バイアス
- D) アンカリング効果
**正解:** A
**解説:** サンクコストとは、すでに失ったもの（埋没費用）を取り戻そうとしてさらに損失を拡大してしまう心理的バイアスです。

## Q4
**問題:** 損失が出ているポジションをどうしても手放せない心理的要因の名称は何ですか？
- A) 現状維持バイアス
- B) 保有効果
- C) サンクコスト効果
- D) 確証バイアス
**正解:** B
**解説:** 保有効果とは、自分が所有しているものを過大評価し、手放すことをためらってしまう心理的傾向です。損失ポジションを持ち続ける原因の一つです。

## Q5
**問題:** 損切り後に「不快感を早く解消したい」と衝動的にエントリーしてしまう原因として、脳のどの系の反応が関係していますか？
- A) 記憶系 — 過去の失敗を忘れようとする防衛反応
- B) 報酬系 — 不快感を即座に解消する行動を促す反応
- C) 免疫系 — ストレスホルモンによって判断力が低下する
- D) 運動系 — 身体的なストレス反応として手が動いてしまう
**正解:** B
**解説:** 負けた不快感を早く消したいという衝動は、脳の報酬系による反応です。怒りと結びついて、即座の行動を促してしまいます。

## Q6
**問題:** リベンジトレードを引き起こす3つの心理的原因として正しい組み合わせはどれですか？
- A) 確証バイアス・現状維持バイアス・後悔回避
- B) アンカリング効果・バンドワゴン効果・損失回避
- C) 沉没コスト・過剰自信・恐怖
- D) サンクコスト・保有効果・怒り
**正解:** D
**解説:** リベンジトレードの3つの心理的原因は「サンクコスト」「保有効果」「怒り」です。それぞれ「取り戻したい」「手放せない」「不快感を解消したい」という行動を引き起こします。

## Q7
**問題:** サンクコストの考え方がトレードにおいて危険な理由として最も適切なものはどれですか？
- A) 過去の利益に固執して利益確定が遅くなるため
- B) 取引手数料が増えてコストパフォーマンスが悪化するため
- C) 失った金額を取り戻そうと、本来見送るべきエントリーをしてしまうため
- D) 過去の成功体験に縛られて新しい手法を試せなくなるため
**正解:** C
**解説:** サンクコストは「すでに失った金額」に執着し、その回収を目的とした根拠のないトレードをしてしまう点が危険です。過去の損失は現在の判断には無関係であるべきです。

## Q8
**問題:** 保有効果がトレードに与える悪影響として最も典型的なものはどれですか？
- A) 損失が拡大しているポジションを損切りできず、含み損が膨らむ
- B) 利益確定を急ぎすぎて、本来得られたはずの利益を取りこぼす
- C) エントリータイミングが遅くなり、良い価格で入れない
- D) 他の銘柄に目移りして、ポジションを頻繁に乗り換えてしまう
**正解:** A
**解説:** 保有効果によって「自分の持っているポジション」に愛着が生まれ、損切りの判断が遅れて含み損が拡大してしまいます。

## Q9
**問題:** 損切り後に怒りが生じるメカニズムとして最も正しい説明はどれですか？
- A) 自己評価が下がったことに対する防衛機制として怒りが生まれる
- B) 市場の不公平さに対する正当な怒りであり、合理的な感情反応である
- C) 脳が損失を物理的な脅威として認識し、闘争反応を引き起こす
- D) 損切りそのものが間違いだったと無意識に認識している証拠である
**正解:** C
**解説:** 脳は損失を物理的な脅威と同様に処理し、不快感を素早く解消しようとする報酬系の反応が怒りとして表れます。これが衝動的なリベンジエントリーを引き起こします。

## Q10
**問題:** 【30分ルール】の目的として最も重要なものはどれですか？
- A) 相場の変動が落ち着くのを待つため
- B) 次のトレードチャンスを冷静に探すための時間を確保するため
- C) 他のトレーダーの動きを観察してからエントリーするため
- D) 感情の高ぶりが自然に収まる物理的なクーリングタイムを設けるため
**正解:** D
**解説:** 30分ルールは、損切り後の感情的な高ぶりが収まるまで物理的にチャートから離れることで、衝動的なリベンジトレードを防ぐことが目的です。

## Q11
**問題:** 1日の損失上限ルールの内容として正しいものはどれですか？
- A) 連敗2回で終了 / 1日の損失が総資金の2%で終了 / 1連敗でロット半分
- B) 連敗5回で終了 / 1日の損失が総資金の5%で終了 / 3連敗でロット半分
- C) 連敗3回で終了 / 1日の損失が総資金の5%で終了 / 1連敗でロット半分
- D) 連敗3回で終了 / 1日の損失が総資金の3%で終了 / 2連敗でロット半分
**正解:** D
**解説:** 損失上限ルールは「連敗3回で終了」「1日の損失が総資金の3%で終了」「2連敗でロットを半分にする」の3本柱で構成され、損失の拡大を体系的に防ぎます。

## Q12
**問題:** 損切り後に感情メモを書く意義として最も重要なものはどれですか？
- A) 自分の感情を客観視することで、リベンジトレードの衝動を弱めるため
- B) トレード日記として後から振り返る記録を残すため
- C) 他のトレーダーと感情を共有して共感を得るため
- D) 感情のパターンを分析して、負けやすい時間帯を特定するため
**正解:** A
**解説:** 「悔しい」「焦っている」「取り返したい」など感情を一言書き出すことで、自分の状態を客観視でき、衝動的な行動へのブレーキがかかりやすくなります。

## Q13
**問題:** 損失上限ルールにおいて「2連敗でロットを半分にする」理由として最も適切なものはどれですか？
- A) 連敗中は判断力が低下しており、3連敗目以降の損失を小さく抑えるため
- B) 2連敗した時点で市場の環境が悪化している可能性が高いため
- C) ロットを半分にすることで利益幅が広がり、損失を早く回復できるため
- D) 資金の3%ルールと連動しており、自動的にロットを調整するため
**正解:** A
**解説:** 連敗中は感情が乱れており、さらなる判断ミスのリスクが高い状態です。事前にロットを減らすことで、仮に3連敗目が来ても被害を最小限に抑えられます。

## Q14
**問題:** 100万円の資金でトレードしている人が1日の損失上限ルール（総資金の3%）を設定した場合、その日の損失限度額はいくらですか？
- A) 1万円
- B) 3万円
- C) 2万円
- D) 5万円
**正解:** B
**解説:** 総資金100万円の3%は3万円です。この額に達した時点でその日のトレードを終了することで、1日で致命的な損失を防ぎます。

## Q15
**問題:** 損切りをした直後に取るべき行動として最も適切なものはどれですか？
- A) すぐにチャートを分析して、同じ銘柄の再エントリーポイントを探す
- B) 損失を取り戻すため、ロットを上げて別のエントリーチャンスを探す
- C) 損切りの判断が正しかったか検証するため、すぐにトレード履歴を見直す
- D) 30分間チャートを閉じて物理的に離れ、感情メモを書く
**正解:** D
**解説:** 損切り直後は感情が高ぶっているため、30分ルールで物理的に離れ、感情メモを書くことで冷静さを取り戻すことが最適な行動です。

## Q16
**問題:** 次のトレードで「まったく別の判断としてゼロから組み直す」とはどういうことですか？
- A) 前回と同じ銘柄には絶対にエントリーしないという意味
- B) 前回使用したテクニカル指標をすべて変更して、新しい分析方法を使うという意味
- C) 前回のトレード結果（勝敗・損益）を一切引きずらず、独立的にエントリー根拠を組み立てるという意味
- D) 前回の損失を考慮して、今回の目標利益を大きく設定し直すという意味
**正解:** C
**解説:** 前回の損失を取り戻すという目的ではなく、市場の現在の状況だけを見て独立した根拠でエントリーを組み立てることが「ゼロから組み直す」ことの本来の意味です。

## Q17
**問題:** あるトレーダーが1回目のトレードで2万円の損失を出しました。すぐに「取り返すぞ」と2回目のエントリーをして、ロットを1.5倍に増やしました。この行動に当てはまる心理的原因の組み合わせとして最も適切なものはどれですか？
- A) 保有効果 + 損失回避
- B) 確証バイアス + アンカリング効果
- C) 現状維持バイアス + 後悔回避
- D) サンクコスト + 怒り
**正解:** D
**解説:** 「取り返すぞ」という思考はサンクコスト（失った2万円への執着）、ロットを増やす行動は怒り（不快感を早く解消したい）によるものです。両方が同時に作用しています。

## Q18
**問題:** 30分ルールを実践する際、最も効果的な具体的な行動はどれですか？
- A) スマホのアプリも含めてすべての取引ツールを閉じ、別の部屋に行く
- B) チャート画面は開いたままにして、別の銘柄の分析だけをする
- C) デモトレードで練習してから、本番に戻る
- D) 取引履歴を確認しながら、次の作戦を頭のなかで組み立てる
**正解:** A
**解説:** 30分ルールの効果は「物理的にチャートから離れる」ことにあります。アプリも含めてツールを閉じ、別の部屋に行くなど、視覚的な刺激を完全に遮断することが最も効果的です。

## Q19
**問題:** 次のうち、リベンジトレードを予防するルールとして機能しないものはどれですか？
- A) 連敗3回でその日のトレードを終了する
- B) 損失を出した直後に、損失分を回復できる銘柄をすぐに探す
- C) 1日の損失が総資金の3%に達したらトレードを終了する
- D) 2連敗した時点でロットを半分にする
**正解:** B
**解説:** 損失直後に「回復できる銘柄を探す」行為は、まさにリベンジトレードそのものです。感情的なエントリーを促すだけで予防になりません。

## Q20
**問題:** 連敗中に「今は調子が悪い」と自覚しながらもエントリーを止められないトレーダーに対して、最も有効な対策はどれですか？
- A) メンタルを強化するために、あえて連敗を続けて耐性をつける
- B) エントリー前に自分に「本当に根拠があるか」を問いかける習慣をつける
- C) 連敗の原因をテクニカル分析の不足だと割り切り、勉強時間を増やす
- D) あらかじめ損失上限ルールを自動的に执行できる仕組みを用意しておく
**正解:** D
**解説:** 連敗中は感情が理性を上回り、自力でのコントロールが困難です。損失上限ルールを事前に決めておき、ルール通りの行動を仕組み化することが最も確実な予防策です。
