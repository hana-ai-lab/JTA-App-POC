# Week 11: ルールを作る、ルールを守る

## Q1
**問題:** ルールブックが「冷静な自分から暴走中の自分への手紙」である理由として最も適切なものはどれですか？
- A) 暴走中は感情的になり、その場で正しい判断を下す力を失っているから
- B) ルールブックは他人に見せるものではなく、自分だけの秘密の文書だから
- C) 暴走中の自分は他人のアドバイスしか受け付けない状態になっているから
- D) 冷静な自分と暴走中の自分では好むトレード手法が変わるから
**正解:** A
**解説:** トレード中に感情が暴走している最中は、自力で正しい判断を下すことができません。だからこそ、冷静な状態の自分が事前にルールを書き残しておく必要があります。

## Q2
**問題:** ルールブック7項目のうち、「トレードする市場と時間軸」を1つに絞る理由は何ですか？
- A) 複数の市場を同時に監視すると手数料が増えるから
- B) 焦点を絞ることでその市場のクセや特徴を深く理解できるから
- C) 証券会社の口座が1つの市場しか対応していない場合が多いから
- D) 複数市場のトレードは法律で制限されているから
**正解:** B
**解説:** 市場と時間軸を1つに絞ることで、その環境特有の値動きのパターンやリズムを体感として蓄積でき、判断の精度が向上します。

## Q3
**問題:** エントリー条件を「具体的に」書くべき理由として正しいものはどれですか？
- A) 曖昧な条件は感情に「今回だけは特別」という言い訳を与えてしまうから
- B) 具体的であれば他のトレーダーと条件を共有しやすくなるから
- C) 証券会社の注文画面には数値で入力する必要があるから
- D) エントリー条件は過去データと照合するために数式化が必要だから
**正解:** A
**解説:** エントリー条件に曖昧さが残っていると、感情が「これも条件を満たしている」と勝手に解釈し、ルールを曲げる言い訳を生んでしまいます。

## Q4
**問題:** 損切りルールの正しい位置として最も適切な記述はどれですか？
- A) 含み損が資金の5%に達した時点で損切りする
- B) エントリーの根拠が崩れる場所で、最大でも資金の2%以内に設定する
- C) チャートのサポートラインより少し下で、余裕を持って幅広く設定する
- D) 損切り幅は利益目標の3分の1以内に収まるように設定する
**正解:** B
**解説:** 損切りは「根拠が崩れる場所」に設定し、1回の損失を資金の2%以内に抑えます。損切りラインは一度決めたら動かしてはいけません。

## Q5
**問題:** 資金管理において、1回のトレードで許容すべきリスクの目安は資金の何%ですか？
- A) 0.1%〜0.5%
- B) 1%〜2%
- C) 5%〜10%
- D) 10%〜20%
**正解:** B
**解説:** 1回のトレードで失う金額を資金の1〜2%に抑えることで、連敗が続いても口座を壊滅させることなく続けられます。

## Q6
**問題:** ルールブックの6項目目「トレードしない条件」を客観的な数字で区切る理由は何ですか？
- A) 主観的な判断では、手が震えているときでもエントリーしてしまう可能性が高いから
- B) 数字で区切ることで自動売買システムにルールを組み込みやすくなるから
- C) トレードしない条件は法律で数値基準を定めることが義務付けられているから
- D) 数字で区切ると他のトレーダーのトレード頻度と比較しやすくなるから
**正解:** A
**解説:** 「今日は調子が悪いからやめておこう」という感覚頼りの判断は、感情に左右されます。「過去10トレードの勝率が〇%以下ならエントリーしない」のように数字で客観的に線を引くことで、感情の介入を防ぎます。

## Q7
**問題:** 振り返りルールで「曜日と時間を決める」ことが重要な理由はどれですか？
- A) 市場が開いている時間帯に振り返るとリアルタイムのデータが使えるから
- B) 週末はチャートデータが更新されないため、平日に振り返る必要があるから
- C) トレード日記を書くために最低2時間の連続した時間が必要だから
- D) 曜日と時間を固定することで、振り返りを習慣化しサボれなくなるから
**正解:** D
**解説:** 振り返りは「気が向いたときにやる」では続きません。「毎週日曜の朝10時」のように日時を決めることで、振り返りがルーティンになり継続できます。

## Q8
**問題:** トレード中に「今回だけは特別だ」という考えが浮かんだとき、何が起きていると考えるべきですか？
- A) 直感が研ぎ澄まされており、例外的なチャンスを見つけた状態
- B) ほぼ確実に感情が判断を歪めているサイン
- C) ルールが不完全で、この状況に対応する条件が抜けている証拠
- D) 市場がいつもと違う動きをしており、柔軟な対応が求められている合図
**正解:** B
**解説:** 「今回だけは特別」という考えが浮かんだ時点で、感情が理性を上回っています。この言葉が出たときは原則としてルールを守るべき警告シグナルです。

## Q9
**問題:** 含み損を抱えている最中に「この状況はルールに当てはまらない」と考えたときの正しい対応はどれですか？
- A) 自分の判断を信じて、状況に応じて臨機応変に対応する
- B) ルールを一時的に見直して、現在の状況に合った新しい基準を適用する
- C) 含み損の中の自分を信用せず、事前に決めたルール通りに損切りを実行する
- D) 少し待って含み損が縮まるのを見てから、改めて判断し直す
**正解:** C
**解説:** 含み損を抱えているときの自分は「まだ大丈夫」と根拠なく信じ込みやすくなります。損失を抱えている状態での判断は信用せず、冷静な自分が決めたルールに従います。

## Q10
**問題:** 「ルールを破って勝つ」ことと「ルールに従って負ける」ことのうち、なぜ後者の方が価値があると言われるのですか？
- A) ルールを守って負ければ、損失の金額が必ず小さくなるから
- B) ルールを破って勝つと税金が高くなり、結果的に利益が減るから
- C) ルールに従って負ける経験は再現性のあるプロセスを示しており、改善できるから
- D) ルールを破って勝つと、次のトレードで必ず大きな損失が出る統計があるから
**正解:** C
**解説:** ルールを破って勝ってもそれは運であり再現性がありません。ルールに従って負けた場合は「プロセスは正しかったか」を検証でき、改善の材料になります。

## Q11
**問題:** ルールブックの見直しは何ヶ月ごとに行うことが推奨されていますか？
- A) 1ヶ月ごと
- B) 3ヶ月ごと
- C) 6ヶ月ごと
- D) 1年ごと
**正解:** B
**解説:** 3ヶ月に一度の頻度で見直しを行います。短すぎるとデータが足らず、長すぎると悪い癖が定着してしまうため、3ヶ月が適切な間隔です。

## Q12
**問題:** ルールの見直し手順として正しい順番はどれですか？
- A) 修正を3つ加える → 検証する → 共通パターンを探す
- B) ルール通りのトレードで勝ったものを集める → 勝因を分析する → 新しいルールを追加する
- C) ルール通りにやって負けたトレードを集める → 共通パターンを探す → 修正を1つだけ加える
- D) 全トレードの損益を計算する → 利益の大きいトレードを再現する → エントリー条件を増やす
**正解:** C
**解説:** 見直しでは「ルールを守って負けたトレード」に焦点を当て、共通の負けパターンを見つけ、修正は一度に1つだけ加えます。複数同時に変えると何が効果あったか分からなくなります。

## Q13
**問題:** トレーダーAは「1回の損失を資金の5%まで許容」し、トレーダーBは「1回の損失を資金の1%に制限」しています。両者が5連敗した場合、資金への影響として正しいものはどれですか？
- A) トレーダーAは約22%減、トレーダーBは約5%減になる
- B) トレーダーAは25%減、トレーダーBは5%減になる
- C) トレーダーAは約35%減、トレーダーBは約10%減になる
- D) 両者ともに損失率は同じで、回数が増えるほど差が出るわけではない
**正解:** A
**解説:** 5%リスクで5連敗すると (0.95)^5 ≒ 0.774、約22.6%の減少です。1%リスクなら (0.99)^5 ≒ 0.951、約4.9%の減少です。資金管理の1〜2%ルールは、連敗時の被害を最小限に抑えるためにあります。

## Q14
**問題:** 利確ルールを事前に決めておくべき理由として最も適切なものはどれですか？
- A) 利確ルールを決めておけば、利益が必ず最大化されるから
- B) 利確は損切りより重要であり、エントリーよりも先に決めるべきだから
- C) 税金対策のために利確のタイミングを年度末に合わせる必要があるから
- D) 含み益を見ると「まだ伸びるかも」と欲張ったり「今のうちに確定させよう」と焦ったりするから
**正解:** D
**解説:** 含み益がある状態では感情が揺れ動きやすく、冷静な判断ができません。利確ポイントを事前に決めておくことで、感情に判断を委ねず機械的に実行できます。

## Q15
**問題:** 次のルールブックの記述のうち、最も改善が必要なものはどれですか？
- A) 「ダウ理論で明確な上昇トレンドを確認後、押し目でエントリーする」
- B) 「1回の損失は口座資金の2%以内に抑え、損切りラインはエントリー後に動かさない」
- C) 「なんとなく上がりそうな気配を感じたらエントリーする」
- D) 「直近5トレードの勝率が30%を下回った日は、それ以上エントリーしない」
**正解:** C
**解説:** 「なんとなく上がりそうな気配」は曖昧すぎる条件です。エントリー条件は具体的で客観的でなければならず、曖昧さは感情の言い訳を許してしまいます。

## Q16
**問題:** 損切りラインを「動かさない」ことが重要な理由はどれですか？
- A) 損切りラインを動かすと手数料が発生するから
- B) 損切りラインは計算で求めたものだから、途中で変更すると計算が合わなくなるから
- C) 損切りラインを動かすと、自動売買のプログラムがエラーを起こすから
- D) 一度動かし始めると「もう少し待てば戻る」と期待して、最終的に被害が拡大するから
**正解:** D
**解説:** 損切りラインを少し下にずらすと、さらに下にずらしたくなります。この「損切りの先延ばし」が口座を壊滅させる最大の原因の一つです。

## Q17
**問題:** ルールの見直しで「修正を1つだけ加える」とされている理由はどれですか？
- A) 1回の見直しで複数の修正を加えると時間がかかりすぎるから
- B) 複数同時に変えると、どの修正が結果に影響したか特定できなくなるから
- C) ルールブックの項目は最大7つまでと決まっているから
- D) 1度に2つ以上変更すると、証券会社のシステムに反映されないから
**正解:** B
**解説:** 一度に複数の条件を変えると、勝率が上がっても「どの変更が効いたのか」が分かりません。1つずつ変更・検証を繰り返すことで、精度の高いルールを構築できます。

## Q18
**問題:** あなたはルールブックを作成しました。ある日、エントリー条件を満たしてポジションを持ったものの、想定と逆に動き始めました。損切りラインまであと少しのところで、「ここで損切りするより、もう少し様子を見よう」と考えました。この状況で取るべき行動はどれですか？
- A) 損切りラインに到達する前に、より安全な位置にラインを引き下げてから待つ
- B) チャートパターンを確認し、反転の兆しがあれば損切りを見送る
- C) 事前に決めた損切りラインに到達した時点で、理由に関わらず損切りを実行する
- D) 同じ方向のポジションを追加して平均取得価格を改善する
**正解:** C
**解説:** 損切りラインは「根拠が崩れる場所」です。到達した時点でエントリーの前提が崩れているため、感情的理由で見送らず機械的に実行します。

## Q19
**問題:** ルールブックを作る「タイミング」として最も適切なものはどれですか？
- A) トレード中に気がついたことをその都度メモして、後で整理する
- B) 大きな損失を出した直後に、反省を込めて一気に書き上げる
- C) ポジションを持たず冷静な状態のときに、時間をかけて作成する
- D) セミナーに参加した直後に、教わった内容をそのまま書き写す
**正解:** C
**解説:** ルールブックは感情が介在しない冷静な状態で作成する必要があります。ポジションを持っている最中や損失直後は感情が高ぶっており、適切なルールを作れません。

## Q20
**問題:** 次のうち、ルールブックに含めるべき「トレードしない条件」として最も適切なものはどれですか？
- A) 「利益目標に到達するまでは絶対にエントリーしない」
- B) 「その日の最初のトレードが損失だった場合、午後のセッションはトレードしない」
- C) 「ルールに従って損切りした日は、翌日までエントリーしない」
- D) 「勝っている日は調子に乗らないよう、全くトレードしない」
**正解:** B
**解説:** 「1回負けたら午後はやらない」のような客観的な数字による条件は、負けが連鎖するのを防ぎます。トレードしない条件は感情に左右されない明確な基準で設定します。
