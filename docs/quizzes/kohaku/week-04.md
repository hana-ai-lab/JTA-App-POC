# Week 4: 損切りは技術、ナンピンは感情

## Q1
**問題:** 損切りを「保険料」と考える理由として最も適切な説明はどれですか？
- A) 小さな損失を意図的に受け入れることで、資金を全失するリスクを防ぐ仕組みだから
- B) 損切りは証券会社の手数料を支払う行為だから
- C) 損失を確定することで税金の控除が受けられるから
- D) 損切りを繰り返すと損失に慣れて精神的ダメージがなくなるから
**正解:** A
**解説:** 損切りは小さな損失（保険料）を払うことで、大きな損失による口座破綻を防ぐ仕組みです。保険と同じく、支払う金額は限定的ですが防げるリスクは壊滅的です。

## Q2
**問題:** 損切りができなくなっていく心理状態の変化を、正しい順番で並べたものはどれですか？
- A) 期待 → 否認 → 麻痺
- B) 麻痺 → 否認 → 期待
- C) 否認 → 麻痺 → 期待
- D) 否認 → 期待 → 麻痺
**正解:** D
**解説:** 損切りできない心理は「まだ余裕がある」と問題を認めない否認、「ここまで耐えたのだから報われるはず」というサンクコストの期待、そして最後に損切りボタンを押す気力すら失う麻痺の3段階で進行します。

## Q3
**問題:** 含み損が拡大しているとき「ここまで耐えたのだから報われるはず」と考える心理現象を何と呼びますか？
- A) 確証バイアス
- B) サンクコスト効果
- C) バンドワゴン効果
- D) ダニング＝クルーガー効果
**正解:** B
**解説:** サンクコスト効果は、すでに投入した時間や金銭を取り返したいという心理から、合理的でない選択を続けてしまう現象です。トレードでは「ここまで耐えた」という執着が損切りを遅らせます。

## Q4
**問題:** 損切りラインをエントリー後に動かしてはいけない理由として間違っているものはどれですか？
- A) 損切りラインは市場の変動に合わせて柔軟に調整すべきだから
- B) エントリー後は含み損のストレスで冷静な判断ができなくなるため
- C) ラインを動かすと根拠のないトレードに変わるため
- D) 損切りラインを広げると2%ルールが崩壊するため
**正解:** A
**解説:** 損切りラインはエントリー前の冷静な自分が決めた判断であり、エントリー後に動かすべきではありません。ラインを動かすと根拠のない「祈りのトレード」に変わり、リスク管理が崩壊します。

## Q5
**問題:** 「計画的ナンピン」と「感情的ナンピン」を見分ける基準として最も重要なものはどれですか？
- A) 追加エントリーの回数が3回以内かどうか
- B) ナンピンした時の含み損が資金の5%以内かどうか
- C) エントリーする前に「ここで追加する」と決めていたかどうか
- D) 追加エントリーの間隔が1時間以上あいているかどうか
**正解:** C
**解説:** 計画的か感情的かの違いは「エントリー前に追加ポイントを決めていたか」で判断します。事前にシナリオとして組み込んでいれば戦略、含み損が出てから衝動的に追加すれば祈りです。

## Q6
**問題:** 正しいトレードのエントリー手順として適切な順番はどれですか？
- A) エントリー → 根拠の確認 → 損切りライン設定 → ロット計算
- B) ロット計算 → エントリー → 損切りライン設定 → 根拠の確認
- C) 損切りライン特定 → 根拠を見つける → エントリー → ロット計算
- D) 根拠を見つける → 損切りライン特定 → 2%ルールでロット計算 → エントリー
**正解:** D
**解説:** 正しい順番は「根拠 → 損切りライン → ロット計算 → エントリー」です。まずエントリーの根拠を確認し、その根拠が崩れる場所に損切りを置き、リスク額からロットを計算してからエントリーします。

## Q7
**問題:** 損切りラインを置くべき正しい場所はどこですか？
- A) 直近の高値・安値から一定の距離を空けた場所
- B) エントリー価格から逆方向に1%下がった場所
- C) 全体資金の2%に相当する価格
- D) エントリーの根拠が崩れる価格帯
**正解:** D
**解説:** 損切りラインは「エントリー根拠が崩れる場所」に置くのが正解です。根拠が通用しなくなった地点で撤退するのが論理的な損切りであり、パーセンテージや距離だけで機械的に決めるべきではありません。

## Q8
**問題:** 損切りの心理3段階のうち、「まだ余裕がある段階で問題に気づけない」状態を指すのはどれですか？
- A) 期待
- B) 否認
- C) 麻痺
- D) パニック
**正解:** B
**解説:** 否認は「大丈夫、すぐ戻る」と自分に言い聞かせる段階です。まだ含み損が小さく余裕があるため、問題の深刻さに気づけず、結果的に損切りのタイミングを逃してしまいます。

## Q9
**問題:** 損切りができなくなる心理3段階の最終段階「麻痺」の特徴として正しいものはどれですか？
- A) 損切りラインをさらに広げて、より大きな含み損に耐えようとする状態
- B) チャートを激しく分析して損切りすべきかどうかを延々と迷い続ける状態
- C) 損切りボタンを押す気力すらなくなり、「もうどうにでもなれ」と投げやりになる状態
- D) 過去の類似チャートパターンを検証して、反転の確率を計算し続ける状態
**正解:** C
**解説:** 麻痺は心理的な限界点であり、損切りボタンを押す気力すら残っていない状態です。「もうどうにでもなれ」と投げやりになり、ここまで来ると自力での損切りはほぼ不可能です。

## Q10
**問題:** 「計画的ナンピン」が戦略として成立するための条件として、不要なものはどれですか？
- A) エントリー前に追加ポイントをシナリオとして組み込んでいること
- B) 各追加ポイントにそれぞれ根拠があること
- C) 追加エントリーのたびにロットを倍にして平均単価を下げること
- D) ナンピンを含めた全体のリスク額が2%ルールの範囲内であること
**正解:** C
**解説:** 計画的ナンピンは事前のシナリオ、各ポイントの根拠、全体リスクの管理が前提です。ロットを倍にして平均単価を下げるのは典型的なマーチンゲール手法であり、リスクを指数関数的に増大させるため推奨されません。

## Q11
**問題:** 感情的ナンピンをしてしまう人の根拠として、実際に動いている心理として最も近いものはどれですか？
- A) 「ここが底値だから反転する可能性が高い」という分析的判断
- B) 「平均単価を下げれば小さな反発で損益が戻る」という計算
- C) 「過去のチャートで同じパターンは反転している」という統計的根拠
- D) 「今の含み損をどうにかして減らしたい、助かりたい」という切迫感
**正解:** D
**解説:** 感情的ナンピンの根拠は「助かりたい」という切迫感のみです。分析や計算に基づくものではなく、含み損のストレスから逃れたいという感情が衝動的な追加エントリーを引き起こします。

## Q12
**問題:** よくある負けパターンのうち、「損切りできない病」に対する具体的な対策として正しいものはどれですか？
- A) 損切り幅を広めに設定して、含み損に耐えられる余裕を持たせる
- B) 小さなロットで取引すれば損切りしなくても被害が小さい
- C) 損切りラインに近づいたらチャートを見ないようにする
- D) エントリー前に逆指値注文（ストップロス）を入れておく
**正解:** D
**解説:** 逆指値注文をエントリーと同時に設定しておけば、迷いや感情が介在する前に自動的に損切りが執行されます。人間の意志に頼らずシステムで損切りを確実に実行するのが最も効果的な対策です。

## Q13
**問題:** 負けた直後にやりがちな「リベンジトレード」の対策として正しいものはどれですか？
- A) 負けを取り返すために、次は勝率の高い設定だけを選ぶ
- B) 負けた後は最低30分間チャートから離れる
- C) 負けた方向と逆のエントリーをして、トレンドに乗り直す
- D) 連続して負けた場合はロットを倍にして一発逆転を狙う
**正解:** B
**解説:** 負けた直後は感情的になっており、冷静な判断ができません。最低30分チャートから離れて感情を落ち着かせることが、リベンジトレードという負の連鎖を断ち切る最も確実な方法です。

## Q14
**問題:** 連勝が続いた後にやってしまいがちな負けパターンと、その対策の正しい組み合わせはどれですか？
- A) パターン：リベンジトレード — 対策：連勝中はロットを段階的に下げる
- B) パターン：連勝後のロット上げ — 対策：ロットは感覚でなく計算で決める
- C) パターン：感情エントリー — 対策：連勝中はエントリー間隔を空ける
- D) パターン：「今日だけ特別」症候群 — 対策：連勝数を記録して目に見える化する
**正解:** B
**解説:** 連勝が続くと「調子がいいから」と感覚的にロットを上げがちですが、これは破滅的なリスクにつながります。ロットは常に2%ルールなどの計算に基づいて決め、感情や勢いで調整してはいけません。

## Q15
**問題:** 「今日だけは特別だ」と思ったときに取るべき行動として正しいものはどれですか？
- A) 「今日だけ」と思った時点でそのトレードは見送る
- B) その日の市場環境を改めて分析し、特別な根拠があるか検証する
- C) 直感が合っていることが多いので、小さなロットで試してみる
- D) いつもより損切り幅を広げて、特別な値動きに対応できるようにする
**正解:** A
**解説:** 「今日だけ特別」症候群は、ルールを破る言い訳を自分に許してしまう危険な心理状態です。「今日だけ」と思った時点で感情が判断を支配しているサインであり、そのトレードは見送るのが正解です。

## Q16
**問題:** エントリー前に「この判断は計画か感情か」と自問するのは、どの負けパターンの対策ですか？
- A) 損切りできない病
- B) 感情エントリー
- C) 「今日だけ特別」症候群
- D) リベンジトレード
**正解:** B
**解説:** 感情エントリーはチャートを見た瞬間の衝動で飛び乗ってしまうパターンです。「この判断は計画に基づくものか、それとも感情か」と自問することで、衝動的なエントリーを事前に防ぐことができます。

## Q17
**問題:** 損切りを正しく実行した後にやるべきこととして、間違っているものはどれですか？
- A) すぐに次のトレードを見つけてエントリーする
- B) 最低30分〜1時間はチャートから離れる
- C) ルール通りに損切りできた自分を褒める
- D) 感情をリセットしてから次のトレードに備える
**正解:** A
**解説:** 損切り直後はまだ感情的な揺れがあり、冷静な判断ができていません。すぐに次のトレードに入ると「損失を取り戻そう」という心理が働きやすく、リベンジトレードにつながるため避けるべきです。

## Q18
**問題:** ルール通りに損切りができたとき、自分を褒めるべき理由として最も適切なものはどれですか？
- A) 損切りで失った資金は税金控除の対象になるから
- B) 損切りを褒めると次のトレードの勝率が上がるというデータがあるから
- C) 損切りは「損した」ではなく「次のチャンスのために資金を守る行為」だから
- D) 損切りを習慣にすれば、将来的に損切りが不要になるから
**正解:** C
**解説:** 損切りは損失を確定する行為ではなく、次のトレードチャンスに向けて資金を守る行為です。ルール通りに実行できたことを褒めることで、損切りに対するネガティブな感情を徐々に軽減できます。

## Q19
**問題:** エントリー前に損切りラインを決めるべき理由として最も重要なものはどれですか？
- A) 含み損を抱えた状態では冷静な判断ができず、損切りの必要性が見えなくなるため
- B) 証券会社の規定で損切りラインの設定が義務付けられているため
- C) 損切りラインを決めないと注文が約定しない仕組みになっているため
- D) 損切りラインを決めておけば、その価格に到達する確率が下がるため
**正解:** A
**解説:** エントリー前は冷静に市場を分析できますが、ポジションを持った後は含み損のストレスで判断が歪みます。冷静な自分が決めた損切りラインに従うことが、感情に左右されないトレードの基本です。

## Q20
**問題:** Week 1〜4の内容を踏まえた場合、トレードで「負けない」ために最初に確立すべきものとして最も重要なものはどれですか？
- A) 高い勝率を誇るエントリーシグナルの見つけ方
- B) 複数のテクニカル指標を組み合わせた分析手法
- C) 損切りルールを含めた資金管理の仕組み
- D) 市場の方向性を予測するファンダメンタル分析
**正解:** C
**解説:** Week 1〜4を通じて一貫しているのは、勝ち方より先に「負け方」を学ぶことの重要性です。高い勝率のシグナルを見つけるよりも、損切りルールと資金管理を確立することが、口座を守り長く生き残るための最優先事項です。
