# Week 12: 焦らなくて大丈夫です — 続けるための仕組み

## Q1
**問題:** 続く人と続かない人の違いを決める最も大きな要因は何ですか？
- A) 生まれつきの意志の強さ
- B) 続けるための仕組みの有無
- C) 一日にかけられる時間の長さ
- D) これまでの投資経験の豊富さ
**正解:** B
**解説:** 続く人と続かない人の違いは意志の強さではなく、続けるための仕組みを持っているかどうかで決まります。「気合いは3日、仕組みは3年」と言われるように、仕組み化が継続の鍵です。

## Q2
**問題:** 習慣化のプロセスにおいて、最初の21日間はどの段階にあたりますか？
- A) 自動化 — 考えなくても自然にできる段階
- B) 定着 — やらないと気持ち悪くなる段階
- C) 慣れ始める — 一番つらいが完璧を求めず小さく始める段階
- D) 振り返り — これまでの習慣を見直して改善する段階
**正解:** C
**解説:** 最初の21日間は「慣れ始める」段階で最もつらい時期です。この時期は完璧を求めず、小さく始めることが大切です。

## Q3
**問題:** 習慣化において「やらないと気持ち悪くなる」状態になるのは何日目以降ですか？
- A) 21日目以降
- B) 40日目以降
- C) 66日目以降
- D) 90日目以降
**正解:** D
**解説:** 90日目以降は「定着」の段階で、やらないと気持ち悪くなる状態になります。この頃には意志力をほとんど使わずに習慣を維持できます。

## Q4
**問題:** 習慣化の3段階を正しい順序で並べたものはどれですか？
- A) 慣れ始める → 自動化 → 定着
- B) 自動化 → 慣れ始める → 定着
- C) 定着 → 慣れ始める → 自動化
- D) 慣れ始める → 定着 → 自動化
**正解:** A
**解説:** 習慣化は「慣れ始める（21日）→ 自動化（66日）→ 定着（90日）」の順で進みます。まず一番つらい慣れの段階を小さく乗り越えることで、徐々に自動化・定着へと進みます。

## Q5
**問題:** 習慣化の66日目に到達した時点でどのような状態になっていますか？
- A) まだ意識的な努力が必要だが、少しずつ習慣になり始めている
- B) 完全に定着し、二度と習慣を崩すことはない
- C) 考えなくても自然にできる自動化の状態になっている
- D) 習慣の効果が数字に現れ始め、大きな成果を実感できる
**正解:** C
**解説:** 66日目は「自動化」の段階です。考えなくても自然に行動できるようになり、習慣を維持するための意志力の負担が大幅に減ります。

## Q6
**問題:** トレードを続けるためのコツとして推奨されている「小さく始める」行動はどれですか？
- A) 感情を一言メモするだけ（10秒）
- B) 毎朝ニュースサイトを5つ巡回する
- C) 毎日1時間チャート分析をする
- D) 毎日トレード日誌をA4用紙1枚分書く
**正解:** A
**解説:** 「感情を一言メモ」はたった10秒でできる行動です。小さく始めることで続けやすく、完璧を求めない姿勢が習慣化の第一歩になります。

## Q7
**問題:** 習慣化の過程で3日間サボってしまった場合、最も適切な対応はどれですか？
- A) 最初からやり直すため、カレンダーのチェックをリセットする
- B) サボったことを深く反省し、ペナルティを自分に課す
- C) 習慣化に失敗した証拠なので、別のアプローチに切り替える
- D) 3日サボっても4日目にまた始めればよく、気にしすぎない
**正解:** D
**解説:** 完璧を目指さないことが習慣化のコツです。3日サボっても4日目にまた始めればよく、途切れたことを理由に諦める必要はありません。

## Q8
**問題:** 習慣を「見える化」する方法として推奨されているものはどれですか？
- A) スプレッドシートに詳細なデータを毎日入力する
- B) 月に一度、詳しいレポートを書く
- C) カレンダーにチェックを付ける
- D) SNSに毎日の成果を投稿する
**正解:** C
**解説:** カレンダーにチェックを付けるだけのシンプルな見える化が推奨されています。手間が少ないからこそ続き、積み重ねが目で見て分かることがモチベーションに繋がります。

## Q9
**問題:** 朝のトレードルーティンで「シナリオを1つ書く」のに想定されている時間はどれですか？
- A) 15分
- B) 1分
- C) 10分
- D) 5分
**正解:** D
**解説:** 朝のルーティン全体で10〜15分のうち、シナリオを1つ書くのに5分を想定しています。指標確認（2分）、市場の流れ確認（5分）、シナリオ作成（5分）の構成です。

## Q10
**問題:** 日中の過ごし方として推奨されている行動はどれですか？
- A) アラートを設定してチャートから離れる
- B) 1時間ごとにチャートを確認して現在値をメモする
- C) 常にチャートを監視してエントリーチャンスを見逃さない
- D) トレード関連のニュースを常にストリーミングで流しておく
**正解:** A
**解説:** 日中はチャートに張り付かず、アラートを設定して離れることが推奨されています。張り付きすぎは感情による判断ミスを招きやすくなります。

## Q11
**問題:** 夜の振り返りルーティンで「トレードをしなかった日」に行うべき行動はどれですか？
- A) 翌日に必ずトレードする決意を固める
- B) 次の日のエントリーポイントを詳細に分析する
- C) 「やらなかった理由」を一言で書き留める
- D) チャートを見直して見逃したチャンスを確認する
**正解:** C
**解説:** トレードしなかった日は「やらなかった理由」を一言書くだけで十分です。無理にトレードしなかった自分を責めず、理由を残すことで自分の傾向が分かります。

## Q12
**問題:** 夜の振り返りでトレードをした日に日誌に書く分量はどれくらいが推奨されていますか？
- A) 項目数は決まっていないが最低500文字
- B) 10行
- C) 1ページ
- D) 3行
**正解:** D
**解説:** トレードをした日は日誌に3行書くだけで十分です。負担を小さくすることで続けやすくなり、継続することで豊富なデータが蓄積されます。

## Q13
**問題:** 検証回数と信頼性の関係について、30回の検証でどのような状態になりますか？
- A) 統計的に信頼できる結果が得られる
- B) たまたまの可能性が高く、まだ判断材料にならない
- C) 傾向が見え始める
- D) パターンを明確に確認できる
**正解:** C
**解説:** 10回ではたまたまの可能性が高く、30回で傾向が見え始め、50回でパターン確認、100回で統計的に信頼できるという段階があります。30回は傾向が見え始める段階です。

## Q14
**問題:** あるトレード手法の検証を50回行いました。この時点で期待できることはどれですか？
- A) パターンを確認できる段階に達している
- B) たまたまの可能性が高く、まだ何も判断できない
- C) 統計的に完全に信頼できる結論が出る
- D) 検証は不要になり、すぐに実戦投入できる
**正解:** A
**解説:** 50回の検証でパターンの確認ができる段階です。ただし統計的な信頼性を高めるには100回が目安であり、50回は中間的な確認地点と捉えます。

## Q15
**問題:** Akiraの3原則の1つ「わかる範囲で、わかることをやる」が意味していることはどれですか？
- A) 自分の理解度に合わせて少しずつ知識を広げていくこと
- B) 理解していないトレードで得た利益は同じ場所で返すため、理解している範囲で取引すること
- C) 初心者のうちは少額の取引だけに絞ること
- D) 難しい分析手法は避けてシンプルな指標だけ使うこと
**正解:** B
**解説:** 理解していないトレードで利益が出ても、同じ理解不足の場面で損失を出すことになります。自分がわかる範囲で取引することが、長期的な安定に繋がります。

## Q16
**問題:** Akiraの3原則のうち「1つを確実にできる人の方が強い」と説明されているのはどれですか？
- A) わかる範囲で、わかることをやる
- B) 完璧を目指さず、小さく続ける
- C) できる範囲で、できることをやる
- D) 自分の頭で考える
**正解:** C
**解説:** 「できる範囲で、できることをやる」は、何でもかんでもやろうとするのではなく、1つを確実にできる人の方が強いという原則です。無理な拡張より確実な積み上げを重視します。

## Q17
**問題:** Akiraの3原則にある「自分の頭で考える」とは、具体的にどのような態度を指していますか？
- A) SNSの情報に振り回されず、自分の基準で考える
- B) 他人の意見を一切参考にせず、自分だけで全てを判断する
- C) 常に最新の経済ニュースから自分なりの見解を導き出す
- D) 過去の自分のトレード履歴だけを根拠に判断する
**正解:** A
**解説:** SNSには情報があふれていますが、それに振り回されず自分の頭で考えることが重要です。他人の意見は参考にしても、最終的な判断は自分の基準で行います。

## Q18
**問題:** あなたはトレードを始めて3週間が経ちました。最初の習慣化の段階が一番つらいと知った今、どのように取り組むのが最も適切ですか？
- A) 毎日必ず完璧なルーティンをこなすと決めて、どんなに忙しくても時間を作る
- B) この時期はつらいのが当然なので、完璧を求めず小さく続ける
- C) 3週間が一番つらいなら、つらさが減る21日目まで一気に集中して乗り切る
- D) つらい期間を短縮するため、複数の習慣を同時に始めて一気に慣れる
**正解:** B
**解説:** 最初の21日間は「慣れ始める」段階で最もつらい時期です。この時期は完璧を求めず小さく続けることが推奨されており、無理な目標設定は挫折の原因になります。

## Q19
**問題:** あなたはあるトレード手法を20回検証しました。結果は悪くありませんが、実戦で使うべきか迷っています。正しい判断はどれですか？
- A) 20回でも結果が良ければ統計的に十分であり、すぐに実戦投入してよい
- B) まだ傾向が見え始める段階なので、少なくとも50回までは検証を続ける
- C) 10回でたまたまの可能性があるため、あと10回追加して判断する
- D) 検証回数に関わらず、自分が納得すればいつでも実戦投入してよい
**正解:** B
**解説:** 20回ではまだ傾向が見え始める段階（30回）にも達していません。パターンを確認するには50回、統計的な信頼性には100回が目安なので、焦らず検証を続けることが重要です。

## Q20
**問題:** あなたは毎朝のトレードルーティンを始めようとしています。全体で10〜15分のルーティンを構成する3つのステップを正しい順序で選んでください。
- A) シナリオを1つ書く → 市場の大きな流れ確認 → 経済指標カレンダー確認
- B) 経済指標カレンダー確認 → 市場の大きな流れ確認 → シナリオを1つ書く
- C) 経済指標カレンダー確認 → シナリオを1つ書く → 市場の大きな流れ確認
- D) 市場の大きな流れ確認 → 経済指標カレンダー確認 → シナリオを1つ書く
**正解:** B
**解説:** 朝のルーティンは「①経済指標カレンダー確認（2分）→ ②市場の大きな流れ確認（5分）→ ③シナリオを1つ書く（5分）」の順です。事実確認から始めて、最後に自分なりのシナリオを構築する流れになります。
