# Week 2: 敵はマーケットではなくあなた自身です

## Q1
**問題:** プロスペクト理論によれば、人間は利益の喜びと損失の痛みをどのように感じますか？
- A) 損失の痛みを利益の喜びより約2倍強く感じる
- B) 金額が大きくなるほど利益の喜びが損失の痛みを上回る
- C) 利益の喜びを損失の痛みより約2倍強く感じる
- D) 利益の喜びも損失の痛みもほぼ同じ強さで感じる
**正解:** A
**解説:** プロスペクト理論によれば、人間は損失から受ける痛みを同等額の利益から得る喜びの約2倍強く感じます。これが損失を先延ばしする行動の根本原因です。

## Q2
**問題:** 含み損を抱えたトレーダーが「損切りしたいが、まだ戻るかもしれない」と損切りを先延ばしする心理的メカニズムとして最も適切なものはどれですか？
- A) 確証バイアス — 都合の良い情報だけを集めて判断が歪む
- B) 保有効果 — 持っているものを過大評価して手放せなくなる
- C) サンクコスト — すでに使ったコストを無駄にしたくない心理
- D) 損失回避 — 損失の痛みを確定させたくない心理
**正解:** D
**解説:** 含み損を確定させる＝損失の痛みを確定させるため、脳はその痛みを先送りしようとします。これが「まだ戻るかも」という損切り先延ばしにつながります。

## Q3
**問題:** 「1万円もらった嬉しさ」を「50」とした場合、「1万円失った痛み」はプロスペクト理論に基づくとどの程度に相当しますか？
- A) 約200
- B) 約25
- C) 約100
- D) 約50
**正解:** C
**解説:** 損失の痛みは利益の喜びの約2倍強く感じるため、嬉しさが「50」なら痛みは「100」に相当します。この非対称性がトレード行動を大きく歪めます。

## Q4
**問題:** 含み益が出ているポジションで「利益を失いたくない」と早すぎる利確をしてしまう原因は何ですか？
- A) 損失回避性によって利益を失う痛みを恐れ、確定させて安心を得ようとするため
- B) 確証バイアスによって利益確定の根拠ばかりが目に入るため
- C) サンクコスト効果によってすでに得た利益を無駄にしたくないため
- D) 保有効果によってポジションの価値を過小評価してしまうため
**正解:** A
**解説:** 含み益はまだ確定していない利益ですが、脳はすでに「自分のもの」と感じ、それを失う痛みを恐れます。その結果、本来の利確ポイントに達する前に利益を確定させたくなります。

## Q5
**問題:** 損失回避性により「利小損大」のパターンが完成する理由として正しいものはどれですか？
- A) 含み損では損切りを先延ばし、含み益では早く利確したくなるため、結果的に利益より損失が大きくなるため
- B) テクニカル分析の精度が不足しており、エントリーポイントが悪くなるため
- C) 利益を最大化しようとする欲求と損失を最小化しようとする理性が競合するため
- D) 人間が本能的に高リスク・高リターンを好むため、利確を急いで大きな賭けに向かうため
**正解:** A
**解説:** 含み損では「まだ戻るかも」と損切りを先延ばし、含み益では「失いたくない」と早めに利確する傾向が組み合わさることで、結果的に小さな利益と大きな損失が繰り返されるパターンが完成します。

## Q6
**問題:** 「この銘柄は上がる」と決めてからチャートを分析すると、上がる根拠ばかりが目に入ってしまう現象を何と呼びますか？
- A) 保有効果
- B) 確証バイアス
- C) サンクコストの罠
- D) 損失回避性
**正解:** B
**解説:** 確証バイアスとは、自分が信じていることを裏付ける情報ばかりを集め、反する情報を無視してしまう認知バイアスです。先に結論を決めてから分析すると、その結論を支持する情報だけが目に入ります。

## Q7
**問題:** 確証バイアスに対する有効な対策として正しいものはどれですか？
- A) 複数のテクニカル指標を組み合わせて分析の精度を上げる
- B) 自分の分析に反対する意見を持つトレーダーと議論する
- C) 過去のトレード履歴を振り返ってパターンを見つける
- D) 分析は「チャートが何を語っているか」から始める
**正解:** D
**解説:** 「買いたい」から分析するのではなく、「チャートが何を語っているか」から分析を始めることで、結論ありきの見方を避け、より客観的な判断ができるようになります。

## Q8
**問題:** 「もうここまで耐えたのだから、ここで損切りしたらこれまでの我慢が無駄になる」という思考は、どの心理的罠に分類されますか？
- A) サンクコストの罠
- B) 確証バイアス
- C) プロスペクト理論
- D) 保有効果
**正解:** A
**解説:** サンクコストの罠とは、すでに失ったお金や時間に引きずられて判断が歪む現象です。過去に使ったコストとこれからの判断は本来無関係ですが、「これまでの我慢を無駄にしたくない」という感情が損切りを先延ばしさせます。

## Q9
**問題:** サンクコストの罠について正しい説明はどれですか？
- A) 他人の投資行動に影響されて、自分の判断を見失う現象
- B) 過去に投じたコストとこれからの判断は本来無関係であるにもかかわらず、過去のコストに引きずられて判断が歪む現象
- C) 将来のリターンを過大評価して、現在の損失を軽く見てしまう現象
- D) 短期的な利益に目がくらんで、長期的なリスクを見落とす現象
**正解:** B
**解説:** サンクコスト（埋没費用）はすでに失われたものであり、これからの判断には関係ありません。しかし人間は「ここまで費やしたのだから」と過去のコストに縛られ、損切りを先延ばししてしまいます。

## Q10
**問題:** ある大学で行われたマグカップの実験で、マグカップを与えられたグループは、与えられなかったグループと比べてどのような結果になりましたか？
- A) マグカップの価値を約半分と評価した
- B) マグカップの価値をほぼ変わりなく評価した
- C) マグカップの価値を約2.5倍の値段と評価した
- D) マグカップの価値を約1.5倍の値段と評価した
**正解:** C
**解説:** マグカップの実験では、持っているグループは持っていないグループの約2.5倍の値段をつけました。自分が所有しているものを実際より高く評価する傾向を保有効果と呼びます。

## Q11
**問題:** 保有効果がトレードにおいて危険な理由として最も適切なものはどれですか？
- A) ポジションを持つと分析能力が物理的に低下するため
- B) ポジションを持つと「手放す＝価値を失う」と感じ、損切りできなくなるため
- C) 保有効果があるとエントリーのタイミングが遅くなるため
- D) 保有効果によって利益目標が低すぎる値に設定されてしまうため
**正解:** B
**解説:** ポジションを持った瞬間に保有効果が働き、そのポジションを実際の価値より高く評価してしまいます。その結果、「手放す＝価値を失う」と感じて損切りができなくなります。

## Q12
**問題:** 含み損を抱えた状態から損失が拡大していく「破滅パターンの連鎖」として正しい流れはどれですか？
- A) 含み損 → サンクコスト → 保有効果 → 損失拡大 → リベンジ心理 → 暴走
- B) 含み損 → 保有効果 → 確証バイアス → 損失拡大 → 諦め
- C) 含み損 → リベンジ心理 → サンクコスト → 損失拡大 → 保有効果 → 暴走
- D) 含み損 → 損失拡大 → 確証バイアス → サンクコスト → 保有効果 → 冷静化
**正解:** A
**解説:** 含み損を抱えると「これまで耐えたのだから」とサンクコストに縛られ、保有効果でポジションを過大評価し、損失が拡大するとリベンジ心理が生じて暴走に至るという連鎖が起きます。

## Q13
**問題:** トレードにおける4つの暴走感情の組み合わせとして正しいものはどれですか？
- A) 恐怖・怒り・焦り（FOMO）・嫉妬
- B) 恐怖・欲望・焦り（FOMO）・後悔
- C) 不安・欲望・焦り（FOMO）・後悔
- D) 恐怖・欲望・退屈・後悔
**正解:** B
**解説:** トレードにおける4つの暴走感情は「恐怖」「欲望」「焦り（FOMO）」「後悔」です。これらは冷静な判断を奪い、ルールを無視した行動を引き起こします。

## Q14
**問題:** 「メタ認知」とは何をすることですか？
- A) 過去のトレード結果を統計的に分析して弱点を見つけること
- B) 市場の参加者の心理を予測して先手を取る分析手法
- C) 自分が今何を感じているかを一歩引いて観察する技術
- D) 瞑想や深呼吸で感情を完全に消し去る訓練
**正解:** C
**解説:** メタ認知とは、自分自身の感情や思考を一歩引いて客観的に観察する技術です。「今、焦りを感じているな」と気づくこと自体がメタ認知の働きです。

## Q15
**問題:** メタ認知を使って暴走感情に対処する3ステップとして正しい順序はどれですか？
- A) (1)距離を置く (2)気づく (3)ルールに戻る
- B) (1)ルールに戻る (2)気づく (3)距離を置く
- C) (1)気づく (2)距離を置く (3)ルールに戻る
- D) (1)感情を抑え込む (2)深呼吸する (3)エントリーする
**正解:** C
**解説:** メタ認知の3ステップは「(1)気づく」「(2)距離を置く」「(3)ルールに戻る」の順です。まず自分の感情に気づき、その感情と距離を置いてから、自分のトレードルールに沿った行動に戻ります。

## Q16
**問題:** 含み損を抱えているトレーダーが「もうここまで損が拡大したのだから、さらにポジションを追加して平均単価を下げよう」と考える場合、作用している心理的罠の組み合わせとして最も適切なものはどれですか？
- A) 保有効果と確証バイアス
- B) 損失回避性と保有効果
- C) サンクコストと損失回避性
- D) 確証バイアスとサンクコスト
**正解:** C
**解説:** 「ここまで損が拡大した」と過去の損失（サンクコスト）に縛られつつ、損失を確定させる痛みから逃れるために（損失回避性）ポジションを追加する行動は、2つの罠が同時に作用しています。

## Q17
**問題:** ポジションを持っているかどうかにかかわらず、自分が下すべき判断を客観的に見極めるための問いかけとして最も有効なものはどれですか？
- A) 「このポジションの含み損はどこまで耐えられるか？」
- B) 「この銘柄の今後の見通しは強気か弱気か？」
- C) 「過去に同じパターンでどうなったか？」
- D) 「もし今、ポジションを持っていなかったら、このレートでエントリーするか？」
**正解:** D
**解説:** 「もし今、ポジションを持っていなかったら、このレートでエントリーするか？」と問うことで、保有効果やサンクコストの影響を排除し、純粋にチャートの現状から判断を下すことができます。

## Q18
**問題:** 「もし今ポジションを持っていなかったら、このレートでエントリーするか？」という問いに対して答えが「いいえ」だった場合、取るべき行動として正しいものはどれですか？
- A) 含み損が小さいうちに損切りする
- B) 損切りラインを少しだけ下方に修正する
- C) 新たな根拠が見つかるまで待機する
- D) ポジションを手放す
**正解:** D
**解説:** 答えが「いいえ」なら、現在そのレートで新規にエントリーする根拠はありません。保有しているポジションであっても、同じ基準で判断すれば手放すべきです。

## Q19
**問題:** FOMO（Fear Of Missing Out）に駆られてエントリーしたトレーダーの典型的なパターンとして最もよくあるものはどれですか？
- A) 相場が停滞している時にエントリーし、長期間膠着状態になる
- B) 相場が大きく動いた後にエントリーし、反落で含み損を抱える
- C) 相場の天井でショートを入れ、さらに上昇して含み損が拡大する
- D) 相場が反転した直後にエントリーし、トレンドに乗って利益が出る
**正解:** B
**解説:** FOMOは「乗り遅れる恐怖」から、すでに大きく動いた後の相場に飛び乗ってしまう行動です。タイミングが遅いため、その後の反落で含み損を抱えるケースが典型的です。

## Q20
**問題:** チャート分析を行う際に「この銘柄を買いたい」という先入観を持った状態で分析を始めると、どのような結果になりやすいですか？
- A) 買いの根拠と売りの根拠を同等に見つけ、判断に迷う
- B) チャートパターンの認識精度が向上し、エントリーポイントが良くなる
- C) 売りの根拠に敏感になり、エントリーを見送る可能性が高まる
- D) 買いの根拠ばかりが目に入り、リスクを見落とした判断をする
**正解:** D
**解説:** 先に「買いたい」という結論を持つと確証バイアスが働き、買いを支持する情報ばかりに注目し、売りを示唆する情報を無視してしまいます。これがリスクを見落とす原因になります。
