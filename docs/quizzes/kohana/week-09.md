# Week 9: ダウ理論の基本

## Q1
**問題:** 「HL（Higher Low）」とはどのような状態を指しますか？
- A) 前の谷よりも高い谷が形成された状態
- B) 前の山よりも低い山が形成された状態
- C) 前の谷よりも深い谷が形成された状態
- D) 前の山よりも高い山が形成された状態
**正解:** A
**解説:** HL（Higher Low＝安値切り上げ）は、直近の安値（谷）を下回らず、より高い位置に谷を形成した状態です。上昇トレンドを構成する2つの要素のうちの1つです。

## Q2
**問題:** 「LH（Lower High）」の意味として正しいものはどれですか？
- A) 安値切り上げ — 前の谷より高い谷
- B) 高値切り下げ — 前の山より低い山
- C) 安値切り下げ — 前の谷より深い谷
- D) 高値切り上げ — 前の山より高い山
**正解:** B
**解説:** LH（Lower High＝高値切り下げ）は、直近の高値（山）を上回れず、より低い位置に山を形成した状態です。下降トレンドを構成する要素の一つです。

## Q3
**問題:** チャート上でHHとHLが連続して形成されている場合、現在の相場環境はどれですか？
- A) 下降トレンド
- B) レンジ相場
- C) 上昇トレンド
- D) トレンド転換中
**正解:** C
**解説:** HH（高値切り上げ）とHL（安値切り上げ）の連続は、上昇トレンドの定義です。山も谷も切り上がっていることが、買い手が主導権を握っている証拠です。

## Q4
**問題:** 下降トレンドを構成する波の組み合わせはどれですか？
- A) HHとHLの連続
- B) LHとLLの連続
- C) HHとLLの交互
- D) LHとHLの交互
**正解:** B
**解説:** 下降トレンドは「LH（高値切り下げ）」と「LL（安値切り下げ）」の連続で定義されます。山も谷も切り下がっていくことが売り手優位の環境です。

## Q5
**問題:** チャート上で高値・安値を見つける際のコツとして推奨されていないものはどれですか？
- A) 明確な反転ポイントだけを拾う
- B) 終値ベースで判断する
- C) 1分足から練習を始める
- D) 日足チャートから練習する
**正解:** C
**解説:** 高値・安値を見つける3つのコツは「明確な反転ポイントだけを拾う」「終値ベースで判断する」「日足から練習する」です。短い時間足はノイズが多く、正しい波の判断が難しいため、まずは日足で練習します。

## Q6
**問題:** レンジ相場をHH/HL/LH/LLの観点から説明したものとして最も適切なものはどれですか？
- A) HHもLLも形成されず、高値と安値が同じ水準で推移する状態
- B) HLとLHが交互に現れる状態
- C) HHとLLが同時に連続して形成される状態
- D) HLだけが連続し、HHが出現しない状態
**正解:** A
**解説:** レンジ相場は「HHもLLも出ない」状態です。高値を更新できず、安値も更新できないため、価格が一定の範囲内で行ったり来たりします。

## Q7
**問題:** ダウ理論の第1原則「トレンドには3段階がある」において、正しい順番はどれですか？
- A) 追随期 → 先行期 → 利食い期
- B) 利食い期 → 先行期 → 追随期
- C) 先行期 → 利食い期 → 追随期
- D) 先行期 → 追随期 → 利食い期
**正解:** D
**解説:** トレンドの3段階は「先行期（賢い人が動く）→ 追随期（大衆が参加）→ 利食い期（賢い人が抜ける）」の順です。この流れを理解すれば、今がトレンドのどの段階にあるかを判断できます。

## Q8
**問題:** ダウ理論の第2原則「出来高がトレンドを確認する」について、正しい説明はどれですか？
- A) 出来高がトレンド方向に増加していることは、トレンドが健全であることを示す
- B) 出来高はトレンド転換時しか意味を持たない指標である
- C) 出来高が減少している時ほどトレンドが強い
- D) 出来高とトレンドの方向には関係がない
**正解:** A
**解説:** トレンド方向への出来高増加は、多くの参加者がその方向に同意している証拠です。価格が上昇しているのに出来高が減る場合は、トレンドの衰えを警告するサインとなります。

## Q9
**問題:** ダウ理論の第3原則として最も適切なものはどれですか？
- A) トレンドは必ず3日以内に転換する
- B) トレンドの長さは過去の値動きから正確に予測できる
- C) 明確な転換シグナルが出るまでトレンドは継続すると考える
- D) 出来高が減った時点で即座にトレンドが転換する
**正解:** C
**解説:** 第3原則は「疑わしきはトレンド継続」です。明確な転換シグナルが確認できるまでは、現在のトレンドが続くと仮定して行動します。これが逆張りを防ぐ根拠にもなります。

## Q10
**問題:** 上昇トレンドから下降トレンドへの「転換」を判断するために必要な2条件として正しい組み合わせはどれですか？
- A) 安値更新の成功 ＋ 戻り高値を上抜く
- B) 高値更新の成功 ＋ 安値更新の成功
- C) 出来高の急減 ＋ 200MAが横ばいになる
- D) 高値更新の失敗 ＋ 押し安値を割り込む
**正解:** D
**解説:** トレンド転換の判断には「高値更新の失敗」と「押し安値を割り込む」の2条件が揃う必要があります。片方だけでは転換と断定できず、単なる調整の可能性が残ります。

## Q11
**問題:** 次のケースのうち、「調整」ではなく「転換」の兆候として最も適切なものはどれですか？
- A) 出来高が減少し、押し安値は割っていない
- B) 出来高が急増し、上位足が崩れ始め、押し安値を明確に割った
- C) 上位足は上昇トレンドのままで、価格が直近の押し安値近くまで下落した
- D) 出来高が減少し、上位足の方向は変わっていない
**正解:** B
**解説:** 転換の兆候は「出来高急増」「上位足が崩れ始める」「押し安値を明確に割った」の3要素で判断します。出来高が減少している場合や押し安値が保たれている場合は調整の可能性が高いです。

## Q12
**問題:** 「200MA＝北極星」という比喩について、価格が200MAの下にある場合の正しい見方はどれですか？
- A) 売り目線でトレードを考えるべき
- B) 買い目線でエントリーを探すべき
- C) 200MAは関係ないので他の指標を使うべき
- D) レンジ相場なので様子見が適切
**正解:** A
**解説:** 200MAはトレンドの方向性を示す「北極星」として機能します。価格が200MAの下にあれば売り目線、上にあれば買い目線、200MAが横ばいならレンジとして捉えます。

## Q13
**問題:** 価格が200MAの上にあり、200MA自体も右肩上がりの場合、最も適切な目線はどれですか？
- A) 売り目線で戻り売りを探す
- B) レンジ目線で両方向のブレイクアウトを待つ
- C) 買い目線で押し目買いを考える
- D) 200MAの傾きに関係なく、常にニュートラル目線
**正解:** C
**解説:** 価格が200MAの上にあり、200MA自体も上向きの場合は明確な上昇トレンドの環境です。買い目線で押し目買いを検討するのが、トレンドに逆らわない順張りの基本です。

## Q14
**問題:** スイングトレードを行う場合の、推奨されるマルチタイムフレーム分析の組み合わせはどれですか？
- A) 月足 → 週足 → 日足
- B) 週足 → 日足 → 4H足
- C) 日足 → 4H足 → 1H足
- D) 4H足 → 1H足 → 15分足
**正解:** B
**解説:** スイングトレードでは「週足（上位足で方向）→ 日足（中位足でシナリオ）→ 4H足（下位足でタイミング）」の組み合わせが推奨されます。トレードスタイルに応じて時間軸の組み合わせを変えます。

## Q15
**問題:** 日足で上昇トレンド、1時間足で下降の波が見られる場合、最も適切な判断はどれですか？
- A) 1時間足の下降を重視して売りエントリーする
- B) 日足と1時間足が矛盾するため、両方無視して別の銘柄を探す
- C) 1時間足の下降が終わるまで待ち、反転したら日足に関係なく売りで入る
- D) 日足の上昇トレンドを信じて、1時間足の下降は押し目の可能性が高いと考える
**正解:** D
**解説:** 上位足と下位足が矛盾する場合、上位足の調整（押し目）の可能性が高いです。迷ったら上位足を信じるのが鉄則であり、日足の上昇トレンドが続く前提で対応すべきです。

## Q16
**問題:** 次のチャート状況のうち、「上昇トレンドが継続している」と最も自信を持って言えるものはどれですか？
- A) 直近の高値を更新し、直近の安値も切り上がっている
- B) 直近の高値を更新したが、直近の安値も下抜けた
- C) 直近の高値更新に失敗したが、直近の安値は保っている
- D) 直近の高値は更新していないが、直近の安値は切り上がっている
**正解:** A
**解説:** 上昇トレンドの定義は「HH（高値切り上げ）＋ HL（安値切り上げ）の連続」です。高値も安値も切り上がっているAが、上昇トレンド継続を最も明確に示しています。

## Q17
**問題:** 次のシチュエーションで、トレンド転換と判断できるものはどれですか？
- A) 上昇トレンド中に高値を更新したが、その後少し下落した
- B) 上昇トレンド中に直近の押し安値を下抜けたが、その後すぐに新高値を更新した
- C) 上昇トレンド中に高値更新に失敗し、さらに直近の押し安値を下抜けた
- D) 上昇トレンド中に高値更新に失敗し、その後も高値を更新し続けている
**正解:** C
**解説:** 転換の2条件「高値更新の失敗」と「押し安値を割り込む」が両方揃った場合にのみ、トレンド転換と判断できます。Aは通常の波、Bはダマシの可能性、Dはトレンド継続です。

## Q18
**問題:** ダウ理論の限界として指摘されていないものはどれですか？
- A) 転換シグナルが遅れて出ることがある
- B) レンジ相場では使いにくい
- C) ダマシ（フェイクシグナル）が存在する
- D) 出来高を全く考慮できない
**正解:** D
**解説:** ダウ理論の限界は「遅れのシグナル」「レンジ相場で使いにくい」「ダマシがある」の3点です。ダウ理論では出来量を重視しており（第2原則）、むしろ出来高でトレンドの健全性を確認します。

## Q19
**問題:** デイトレードを行う場合の、推奨されるマルチタイムフレーム分析の組み合わせはどれですか？
- A) 月足 → 週足 → 日足
- B) 週足 → 日足 → 4H足
- C) 4H足 → 1H足 → 5分足
- D) 日足 → 4H足 → 1H足
**正解:** D
**解説:** デイトレードでは「日足（上位足で方向）→ 4H足（中位足でシナリオ）→ 1H足（下位足でタイミング）」の組み合わせが推奨されます。長期投資やスイングとは異なる時間軸を使います。

## Q20
**問題:** ダウ理論に基づくトレードで「最も重要な教訓」とされるものはどれですか？
- A) レンジ相場では積極的にスキャルピングする
- B) 出来高が減少したら必ずエントリーする
- C) 200MAに触れたら必ず反転すると仮定する
- D) トレンドに逆らう逆張りトレードをしない
**正解:** D
**解説:** ダウ理論の最大の教訓は「逆張りをしない」ことです。トレンドが継続している限り、その方向に沿った順張りを心がけることが、ダウ理論を活かしたトレードの基本姿勢です。
