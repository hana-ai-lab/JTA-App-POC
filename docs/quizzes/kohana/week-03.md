# Week 3: 波の法則 — 200MAとエリオット波動

## Q1
**問題:** 推進波の特徴として正しいものはどれですか？
- A) トレンドと逆方向に動き、出来高が減少する
- B) トレンド方向に動き、ローソク足が大きく角度が急である
- C) トレンド方向に動くが、出来高は減少傾向にある
- D) トレンドと逆方向に動き、ローソク足が大きい
**正解:** B
**解説:** 推進波はトレンド方向に力強く動き、出来高が増加し、ローソク足が大きく、角度が急になるのが特徴です。これに対し修正波はトレンドと逆方向に弱く動きます。

## Q2
**問題:** 修正波の特徴として正しい組み合わせはどれですか？
- A) 出来高増加・ローソク足大きい・角度急
- B) トレンド方向・力強い・角度緩やか
- C) 出来高減少・ローソク足小さい・角度緩やか
- D) トレンド方向・出来高減少・ローソク足大きい
**正解:** C
**解説:** 修正波はトレンドと逆方向に動き、力が弱く、出来高が減少し、ローソク足が小さく、角度が緩やかになります。これらは推進波と対照的な特徴です。

## Q3
**問題:** エリオット波動の基本構造として正しいものはどれですか？
- A) 上昇3波＋下降2波の計5波
- B) 推進5波＋修正3波（A-B-C）の計8波
- C) 推進3波＋修正5波（A-B-C）の計8波
- D) 上昇5波＋下降5波の計10波
**正解:** B
**解説:** エリオット波動はトレンド方向に進む推進5波（1〜5波）と、トレンドに逆行する修正3波（A・B・C波）の計8波で一つのサイクルを構成します。

## Q4
**問題:** エリオット波動の第1波の特徴として最も適切なものはどれですか？
- A) 最も値幅が大きく、多くのトレーダーが参入する波
- B) 天井を示唆するダイバージェンスがよく出現する波
- C) 始まりの波だが、ほとんどの人が気づかず後からわかることが多い
- D) 直前のトレンドの反転を確認させる、最も恐怖の強い波
**正解:** C
**解説:** 第1波は新しいトレンドの始まりの波ですが、当時はほとんどのトレーダーが気づかず、後から振り返って「これが1波だった」と判明することが多いのが特徴です。

## Q5
**問題:** エリオット波動の第2波における「鉄則」とは何ですか？
- A) 3波の起点を割り込んではならない
- B) 5波の高値を超えなければならない
- C) 1波の起点（出発点）を割り込んではならない
- D) A波の安値を必ず更新しなければならない
**正解:** C
**解説:** 第2波はふるい落としの波で恐怖がピークに達しますが、1波の起点（出発点）を割り込まないことが鉄則です。起点を割り込んだ場合、エリオット波動のカウント自体が見直しの対象になります。

## Q6
**問題:** エリオット波動において「本流」と呼ばれ、最も狙うべきとされる波はどれですか？
- A) 第1波 — 最初に動くため最も安全
- B) 第5波 — 最後のひと押しで確実性が高い
- C) 第3波 — 確認済みの流れで最大の値幅を持つ
- D) B波 — 偽りの回復だが、順張りのチャンス
**正解:** C
**解説:** 第3波は「本流」と呼ばれ、最も大きな値幅を持ち、出来高が急増します。1波が確認済みであることから参入の根拠が立ちやすく、5波のような天井掴みリスクもないため最も狙いやすい波です。

## Q7
**問題:** 第5波の注意点として正しいものはどれですか？
- A) 出来高が最も急増し、1波の起点を割り込むリスクがある
- B) 3波よりも値幅が大きく、最も利益を狙いやすい波
- C) 勢いが鈍り、ダイバージェンスが出現し、天井掴みのリスクが高い
- D) 直前の4波と価格帯が重なるため、エントリーを避けるべき波
**正解:** C
**解説:** 第5波はトレンドの最後のひと押しですが、勢いが鈍っており、オシレーターとのダイバージェンスが出現しやすくなります。天井を掴むリスクが高いため慎重な対応が求められます。

## Q8
**問題:** エリオット波動の修正波A波・B波・C波について、正しい説明はどれですか？
- A) A波は本格的転換、B波はふるい落とし、C波は偽りの回復
- B) A波は転換の始まり、B波は偽りの回復で罠、C波は本格的転換でA波の安値を割り込む
- C) A波は最も値幅が大きい、B波は押し目確認、C波は新たなトレンドの始まり
- D) A波は利確の調整、B波は最後のひと押し、C波は1波の起点への回帰
**正解:** B
**解説:** A波は転換の始まりで「ただの押し目」と楽観視されがち、B波は偽りの回復で最も多い失敗パターン、C波は本格的な転換でA波の安値を割り込むという性格を持っています。

## Q9
**問題:** 200MA（200日移動平均線）の役割として正しいものはどれですか？
- A) エントリーの直接的な根拠となる指標
- B) 短期の売買タイミングを判定するオシレーター
- C) 方向を示すコンパスであり、ゾーンの信頼度を上げる補助ツール
- D) フィボナッチリトレースメントと組み合わせるための基準線
**正解:** C
**解説:** 200MAは方向を示すコンパスであり、エントリーの直接的な根拠ではなく、ゾーン分析などと組み合わせて信頼度を高める補助ツールとして使われます。価格が200MAの上にあれば上昇トレンド、下にあれば下降トレンドと判断します。

## Q10
**問題:** フラクタル構造とはどのような概念ですか？
- A) 同じ時間軸の中で波動が3回繰り返される構造
- B) 上位足の1波の中に下位足の3波構造があるという考え方
- C) 上位足の3波の中に下位足の5波構造があるという考え方
- D) すべての時間軸で波動の形が完全に一致するという法則
**正解:** C
**解説:** フラクタル構造とは、上位足の3波の中に下位足の5波構造が内包されているという概念です。これにより、上位足で方向を確認し、下位足で起点を見つけるという二段階のアプローチが可能になります。

## Q11
**問題:** 推進波と修正波の違いを実践で活かす際、正しいスタンスはどれですか？
- A) 推進波でも修正波でも、エントリーチャンスがあれば積極的に乗る
- B) 推進波に乗り、修正波では待つ。修正波でエントリーすると逆行に巻き込まれる
- C) 修正波は推進波よりリスクが低いため、修正波中心にエントリーする
- D) 推進波は値幅が大きすぎるため、修正波の反発を狙う方が安全
**正解:** B
**解説:** 推進波はトレンド方向に力強く動くため、そこに乗ることが推奨されます。修正波では待つのが正解であり、修正波でエントリーすると逆行に巻き込まれるリスクが高くなります。

## Q12
**問題:** 第3波を狙うべき理由として最も適切でないものはどれですか？
- A) 1波がすでに確認済みであり、トレンドの方向性が見えている
- B) 推進波の中で最も値幅が大きく、リスクリワードが良い
- C) 天井掴みのリスクがなく、エントリー後の逆行リスクが最も低い
- D) 出来高が急増しており、多くの参加者が同じ方向に動いている証拠がある
**正解:** C
**解説:** 「天井掴みリスクがない」のは5波と比較した場合の利点ですが、「逆行リスクが最も低い」は言い過ぎです。3波であってもトレードにリスクは伴います。3波を狙う主な理由は、1波が確認済みで流れが見えることと、最大の値幅でリスクリワードが良いことです。

## Q13
**問題:** 第4波の特徴として正しいものはどれですか？
- A) 1波の価格帯と重ならない範囲で調整する波
- B) 直前のトレンドの起点を割り込む、恐怖のピーク
- C) 最も値幅が大きく、出来高が急増する波
- D) 偽りの回復で、最も多い失敗パターンを生む波
**正解:** A
**解説:** 第4波は3波での利確による調整の波ですが、1波の価格帯と重ならないのが特徴です。このルールはエリオット波動のカウントを検証する際の重要な基準の一つになります。

## Q14
**問題:** B波が「最も多い失敗パターン」を生む理由として最も適切な説明はどれですか？
- A) B波は値幅が最も大きく、損失が拡大しやすいため
- B) B波は新たな上昇トレンドの始まりと錯覚させる偽りの回復であるため
- C) B波は1波の起点を割り込むため、カウントが崩壊するため
- D) B波は出来高が急増し、多くのトレーダーが参入してしまうため
**正解:** B
**解説:** B波は一時的に価格が回復する「偽りの回復」であり、多くのトレーダーが「トレンド再開だ」と錯覚して買い（売り）を入れてしまいます。その後C波で本格的に転換するため、罠にはまったトレーダーが大きな損失を被るのが最も多い失敗パターンです。

## Q15
**問題:** 次のシナリオを読んで、現在どの波にいる可能性が最も高いかを選んでください。

「上昇トレンドの途中で、直近の安値が前回の安値を下回らず、押し目から反発し始めた。出来高が急増しており、これまでの波の中で最も勢いがある。」
- A) 第1波
- B) 第5波
- C) 第3波
- D) A波
**正解:** C
**解説:** 前回の安値を下回らない押し目（2波の終わり）から反発し、出来高が急増して最も勢いがあるという特徴は、第3波の始まり〜進行中の典型的なパターンです。第3波は推進波の中で最も力強く、出来高が最も増加します。

## Q16
**問題:** 200MAを使った相場環境の判断として正しいものはどれですか？
- A) 価格が200MAにタッチしたら、必ず反発する
- B) 価格が200MAの上にあれば上昇トレンド、下にあれば下降トレンドと判断する
- C) 200MAの傾きが上を向いていれば、下降トレンドでも買いでエントリーする
- D) 200MAは短期売買のエントリー・エグジットのシグナルとして使う
**正解:** B
**解説:** 200MAは方向を示すコンパスであり、価格が200MAの上にあれば上昇トレンド、下にあれば下降トレンドと判断します。ただし、これはエントリーの直接的な根拠ではなく、トレンド方向を確認する補助的な役割です。

## Q17
**問題:** フラクタル構造を活用した実践的なアプローチとして正しいものはどれですか？
- A) 下位足で波を数え、その結果を上位足に当てはめて方向を決める
- B) 上位足で方向を確認し、下位足でエントリーの起点を見つける
- C) 上位足と下位足で完全に同じ波動カウントが一致した場合のみエントリーする
- D) 下位足の波動をすべてカウントし終えてから上位足を確認する
**正解:** B
**解説:** フラクタル構造の実践的な活用方法は、まず上位足でトレンド方向を確認し、その後に下位足に切り替えてエントリーの起点を見つけるという二段階アプローチです。上位足の大方向に沿って下位足でタイミングを測ります。

## Q18
**問題:** 次のシナリオを読んで、最も適切な判断を選んでください。

「日足で価格が200MAの下にある。4時間足で上昇の5波カウントができ、現在5波付近に達している。オシレーターにダイバージェンスが出現している。」
- A) 200MAが下にあるため下降トレンドであり、5波の上昇は修正波の可能性が高い。買いエントリーは避けるべき
- B) 5波まで上昇している勢いに乗って、追加の買いエントリーをする
- C) ダイバージェンスは無視し、200MAのブレイクアウトを待ってからエントリーする
- D) 4時間足の5波は日足の3波の中にある可能性が高く、積極的に買いを入れる
**正解:** A
**解説:** 日足で200MAの下にあれば下降トレンドの可能性が高く、4時間足の5波カウントは下降トレンド中の修正波（上昇）の一部に過ぎない可能性があります。5波付近でダイバージェンスが出ていれば天井掴みリスクも高く、大方向に逆行するエントリーは避けるべきです。

## Q19
**問題:** 波動カウントに対する正しいスタンスはどれですか？
- A) すべての波を完璧にカウントできないとエントリーしてはならない
- B) 1波から5波まで完璧にカウントできた場合のみ3波を狙う
- C) 完璧なカウントは不要であり、「だいたい3波目あたり」で十分実践に役立つ
- D) 波動カウントは主観的すぎるため、一切参考にすべきではない
**正解:** C
**解説:** 実践では完璧な波動カウントは不要です。「だいたい3波目あたり」という大まかな判断でも、トレンド方向に乗るという目的には十分に機能します。完璧を求めすぎるとエントリーの機会を失います。

## Q20
**問題:** 次のシナリオを読んで、最も注意すべきリスクを選んでください。

「上昇トレンドが続いている銘柄で、直近高値をわずかに更新した後、押し目が来た。多くのトレーダーが「押し目買いのチャンス」と言っている。しかし、オシレーターは直近高値時の数値を下回っている。」
- A) 第3波の始まりであり、押し目買いの絶好のチャンスである
- B) ダイバージェンスが示唆する第5波の天井圏であり、B波の罠の可能性がある
- C) 第2波のふるい落としが終わった直後であり、3波開始の合図である
- D) A波の転換が始まっており、C波で大きく下落する可能性があるため空売りすべき
**正解:** B
**解説:** 直近高値を更新したもののオシレーターがそれを下回っているのはダイバージェンスの発生です。これは第5波の特徴であり、その後の押し目はB波（偽りの回復）の可能性があります。「押し目だ」と楽観視して飛び込むと、C波での本格的転換に巻き込まれる典型的な罠のパターンです。
