# Week 6: 資金管理の数学 — 2%ルールとロット計算

## Q1
**問題:** あるトレーダーが「今日は絶対に勝てる」と確信し、口座資金の20%を1回のトレードにリスクした。この行動がなぜ危険なのか、最も本質的な理由はどれですか？
- A) 利益確定目標を設定せずにエントリーしているから
- B) 数回の連敗で口座に致命的なダメージを受け、退場リスクが急上昇するから
- C) 自信がある時ほど相場は逆に動きやすいから
- D) 損切り注文を置いていないに違いないから
**正解:** B
**解説:** 20%をリスクすると5連敗で口座は33%以下に激減します。2%ルールで同じ5連敗なら約90%が残る。1回のリスクを2%以内に制限することが、退場を防ぐ構造的な防壁になります。

## Q2
**問題:** 口座資金100万円のとき2%ルールでの許容損失は2万円だった。連敗が続き口座が50万円に減った場合、次のトレードで許容できる損失額はいくらになりますか？
- A) 変わらず2万円（ルールは金額で固定されている）
- B) 1万円（口座残高の2%が自動的に新たな上限になる）
- C) 0.5万円（口座が半分になったのでリスクも半分にする）
- D) 2%ルールは口座が半分以下になったら停止する
**正解:** B
**解説:** 2%ルールは「残高の2%」という比率固定のため、口座が減れば許容損失も連動して縮小します。これにより連敗時に損失ペースが自然に鈍化し、口座への致命傷を構造的に防ぎます。

## Q3
**問題:** 正しいロット計算の手順として適切なものはどれですか？
- A) 総資金の2%でエントリー金額を決めてから、損切り位置を探す
- B) 自信の度合いに応じてロット数を調整し、損切り幅を後から決める
- C) 損切り幅（pips）を先に決め、許容損失額からロット数を逆算する
- D) 利益目標の値幅を先に決め、そこから逆算してロット数を計算する
**正解:** C
**解説:** 正しい手順は「許容損失額（総資金×2%）を決める → 損切り幅（pips）を決める → 許容損失額と損切り幅からロット数を逆算する」という順序です。ロット数は感情や自信ではなく、数学的に導きます。

## Q4
**問題:** 2%ルールを厳守して10連敗した場合、残存する資金はおよそ何%ですか？
- A) 約50%
- B) 約65%
- C) 約82%
- D) 約95%
**正解:** C
**解説:** 2%ルールでの10連敗は 0.98^10 ≒ 0.817、つまり資金の約82%が残存します。口座が減るたびにリスク額も減るため、損失のペースが自然に鈍化するのが2%ルールの強みです。

## Q5
**問題:** 2%ルールを守るトレーダーAと、毎回10%をリスクするトレーダーBが10連敗した。残存資金率の比較として正しいものはどれですか？
- A) AもBも約80%が残り、差はほぼない
- B) Aは約82%、Bは約35%しか残らない
- C) どちらも約60%前後で、リスク設定より運の影響が大きい
- D) Bの方がリスクを取っているため、最終的に口座は多く残る
**正解:** B
**解説:** Aは0.98^10≒82%、Bは0.90^10≒35%。同じ10連敗でも1回あたりのリスク設定の違いだけで残存率が2倍以上変わります。「小さなリスクを維持すること」が長期的な生存確率を決定づけます。

## Q6
**問題:** あるトレーダーが今日3回連続で負けた。「まだ取り返せる、もう1回だけ」と感じ、同じロットで続けようとしている。資金管理のルールに従った正しい行動はどれですか？
- A) ロットを1.5倍に増やして3回分を一発で取り返す
- B) ロットを半分に下げて今日の分を続ける
- C) その日のトレードを終了し、翌日冷静な状態で再開する
- D) エントリー前に5秒ルールを実行してから慎重に続ける
**正解:** C
**解説:** 3連敗は「その日のトレード終了」の自動ブレーキポイントです。連敗中は感情が乱れリベンジトレードに向かいやすく、さらに損失が重なるリスクが高い。強制停止することで感情的な損失拡大を防ぎます。

## Q7
**問題:** 今週に入り5連敗が続いているトレーダーが、翌日から同じロットで再開しようとしている。次にすべき行動として正しいものはどれですか？
- A) その週のトレードをすべて終了する
- B) 1週間の休止をとってから再開する
- C) ロットを半分に下げて継続する
- D) 損切り幅だけを狭めたうえで同じロットで再開する
**正解:** C
**解説:** 5連敗は「ロット半減」の自動ブレーキポイントです。3連敗でその日を切り上げても翌日も負けが続く場合、ロットを縮小して追加ダメージを抑えながら、手法の分析・検証に集中できます。

## Q8
**問題:** 月曜から金曜の間に口座資金の7%を失った。残り1日あるため「今週中に取り返したい」と思い、金曜のトレードを続けようとしている。資金管理のルールに従うと次にすべき行動は？
- A) 残り1日で取り返せるようロットを2倍に増やす
- B) 月間-10%に達するまでは続けてよい
- C) その週のトレードを終了する
- D) 5秒ルールを徹底してから1回だけエントリーする
**正解:** C
**解説:** 週で-6%が「その週のトレード終了」の自動ブレーキです。7%はすでにこれを超えています。「取り返したい」という焦りが生まれている状態こそ、ルールが最も機能すべきタイミングです。

## Q9
**問題:** あるトレーダーが10連敗を記録した。「これ以上ロットを下げると利益が出ない」と考え、現在のロットのまま続けようとしている。正しい行動は？
- A) ロットを4分の1に下げて継続する
- B) その月のトレードを終了する
- C) 1週間の休止をとる
- D) 手法を変えてすぐに再挑戦する
**正解:** C
**解説:** 10連敗は「1週間の休止」の自動ブレーキポイントです。ここまで負けが続く場合、手法や分析に根本的な問題がある可能性が高い。感情的な継続より、冷静に振り返る時間を確保することが最優先です。

## Q10
**問題:** 今月口座資金の11%を失ったトレーダーが「来月から気持ちを切り替えればいい」と思い、月末まで少額でトレードを続けようとしている。正しい行動は？
- A) 少額なら続けても問題ない
- B) 翌月の計画を立てながら様子見トレードを続ける
- C) その月のトレードを終了する
- D) 2%ルールを1%に変更して今月残りを継続する
**正解:** C
**解説:** 月間-10%に達した時点で「その月のトレード終了」が自動ブレーキです。月間で5回分の2%損失は手法・メンタル両面に問題がある可能性を示しており、翌月に備えて今月中に徹底的に振り返ります。

## Q11
**問題:** 「自信があるからロットを張る」のではなく、何に基づいてロット数を決めるべきですか？
- A) チャートの形状が完璧に見える度合い
- B) 損切りまでの距離が近いかどうか
- C) 直近の勝率の高さ
- D) 経済指標発表の重要度
**正解:** B
**解説:** ロット数は「自信」ではなく「損切りまでの距離」で決めます。損切り幅が狭い（距離が近い）ほど、同じ許容損失額でも大きなロットを張れる仕組みです。感情ではなく構造的にロットを算出することが資金管理の核心です。

## Q12
**問題:** 口座資金100万円で損切り幅が50pipsのトレードをする場合、2%ルールに従った正しいロット計算の考え方として適切なものはどれですか？（1pipsあたりの損失をロット単位とします）
- A) 100万円の2%である2万円を損切り幅50pipsで割り、ロット数を求める
- B) 100万円を50pipsで割り、その2%をロット数とする
- C) 100万円を一括でエントリーし、2万円下がったら損切る
- D) 50pips動くごとにロットを増やしていく
**正解:** A
**解説:** ロット計算の基本は「許容損失額（総資金×2%）÷ 損切り幅（pips）」です。100万円×2%=2万円を損切り幅50pipsで割ることで、逆算された適正ロット数が求まります。

## Q13
**問題:** 「夜、ぐっすり眠れるロット数が適正値」という言葉が意味する内容として最も適切なものはどれですか？
- A) ポジションを持ったまま寝るべきではないため、デイトレードのみを行えという意味
- B) 利益が出た日は安心して眠れるため、勝率を上げることに集中すべきという意味
- C) 含み損を気にして眠れないような大きなロットは、適正なロット数を超えているという意味
- D) 寝る前には必ず全ポジションを決済すべきという意味
**正解:** C
**解説:** 適正なロット数であれば、ポジションを持っていても精神的な負担は小さく安眠できます。逆に含み損が気になって眠れないなら、それはロットが大きすぎるサインであり、資金管理の観点から縮小が必要です。

## Q14
**問題:** 勝率40%のトレーダーが毎月安定して利益を上げている。「なぜ勝率が低いのに勝てるのか」という問いへの最も本質的な答えはどれですか？
- A) 運が良い月が続いているだけで、長期的には必ず負ける
- B) 1回の平均利益が1回の平均損失より十分に大きく、期待値がプラスになっているから
- C) エントリー回数を増やすことで勝率の影響が薄まるから
- D) 損切りを使わないポジション管理をしているから
**正解:** B
**解説:** 期待値は「勝率×平均利益 + 敗率×平均損失」で決まります。勝率40%でも平均利益が平均損失の2倍以上あれば長期的に利益が出ます。勝率より損益比（リスクリワード）の設計が収益の鍵です。

## Q15
**問題:** トレーダーの期待値がプラスであるために必要な条件はどれですか？
- A) 勝率が50%を超えること
- B) 平均利益が平均損失より大きいこと、または勝率が十分に高いこと
- C) 毎回同じロット数でエントリーすること
- D) 1日あたりのトレード回数を一定に保つこと
**正解:** B
**解説:** 期待値がプラスになるには「勝率×平均利益 + 敗率×平均損失 > 0」である必要があります。勝率が低くても平均利益が平均損失より十分に大きければプラスになりますし、逆に勝率が高ければ利益目標が小さくても成立します。

## Q16
**問題:** 「ロットを張らない勇気」が最も尊い勇気である理由として最も適切なものはどれですか？
- A) ロットを張らない期間が長いほど税金の負担が減るため
- B) 条件が揃っていない場面でエントリーを自制できることが、長期的な生存に直結するため
- C) 少ないロットで利益を出す方が技術が高いと評価されるため
- D) 大きなロットを張るよりも小さなロットを何度も張る方が統計的に有利なため
**正解:** B
**解説:** 相場には常にチャンスがあるわけではなく、条件が揃わない場面でエントリーしないこともトレードの重要な判断です。「張らない勇気」を持てることは、感情に抗える自律性の証明であり、長期的な生存に直結します。

## Q17
**問題:** あるトレーダーが1日に2%ルールを守りながら2回エントリーし、どちらも負けた。合計損失率は約4%になった。この状況への正しい対応はどれですか？
- A) 1回あたり2%ルールを守っているので、この日はそのまま続けてよい
- B) 合計損失が1日の上限（総資金の3%）を超えたため、その日のトレードを終了する
- C) 次のトレードだけロットを半分に下げて続ける
- D) 3%を超えた時点で自動的に1週間の休止が発動する
**正解:** B
**解説:** 1日の損失上限は総資金の3%が目安です。1回2%のルールでも2回負けると4%になるため、「1日あたり3%」の上限を設けることで、1日のうちに口座に致命傷を与えることを防ぎます。

## Q18
**問題:** 口座資金200万円のトレーダーが、損切り幅20pipsのトレードと損切り幅80pipsのトレードを比較します。2%ルールに従う場合、どちらがより大きなロット数でエントリーできますか？
- A) 損切り幅20pipsの方が大きなロット数でエントリーできる
- B) 損切り幅80pipsの方が大きなロット数でエントリーできる
- C) どちらも同じロット数になる
- D) 損切り幅に関係なく、総資金の2%分のロット数で固定される
**正解:** A
**解説:** 許容損失額が同じ（200万円×2%=4万円）場合、損切り幅が狭いほど1pipsあたりの許容損失が大きくなるため、より大きなロット数でエントリーできます。損切り幅20pipsなら4万円÷20pips、80pipsなら4万円÷80pipsで計算されます。

## Q19
**問題:** トレードの期待値がマイナスであるトレーダーが、トレード回数を増やした場合どうなりますか？
- A) 大数の法則により、次第に期待値がプラスに近づく
- B) 回数を増やすほど1回あたりの損失が減っていく
- C) 期待値がマイナスのまま回数を増やすほど、長期的な累積損失は大きくなる
- D) トレード回数と期待値には関係がない
**正解:** C
**解説:** 期待値がマイナスの場合、トレードを繰り返すほど数学的には確実に累積損失が拡大します。回数を増やすことが解決策ではなく、まず期待値をプラスにする（分析方法やリスクリワードの改善）ことが前提です。

## Q20
**問題:** あるトレーダーが今月を振り返ると、3連敗した日が2回あり、1週間で7%下落し、月間では12%下落していた。この月にどの自動ブレーキが正しく発動すべきでしたか？
- A) 3連敗時：その日終了 ／ 週-6%超：その週終了 ／ 月-10%超：その月終了
- B) 3連敗時：ロット半減 ／ 週-7%：その日終了 ／ 月-12%：1週間休止
- C) すべて継続OK、月末にまとめて振り返れば十分
- D) 3連敗時：1週間休止 ／ 週損失は問わない ／ 月-10%超：ロット半減
**正解:** A
**解説:** 自動ブレーキの5段階は「3連敗→その日終了」「5連敗→ロット半減」「週-6%→その週終了」「10連敗→1週間休止」「月-10%→その月終了」。感情が動く前にルールが止める。それが自動ブレーキの存在意義です。
