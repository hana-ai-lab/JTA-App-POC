# Week 8: 損切りの技術 — 逆指値は命綱

## Q1
**問題:** あるトレーダーが「損失が確定するのが嫌だ」と一切損切りをしなかった結果、1回の大きな下落で口座の70%を失った。このトレーダーの根本的な失敗は何か？
- A) レバレッジの設定が高すぎた
- B) 損切りのタイミングが遅かっただけで、方針自体は正しい
- C) 運が悪く、回避は難しかった
- D) 小さな損失（保険料）を惜しんだ結果、口座への致命的な損失を自ら招いた
**正解:** D
**解説:** 損切りは「保険料」。小さな損失を払い続けることで、口座を潰す大きな損失を防げる。損切りを拒絶すると、1回の「全焼」リスクを常に抱え続けることになる。

## Q2
**問題:** 買いポジション保有中、設定した損切りラインを超えて価格が下落した。「これは一時的な下落のはず」と考えて損切りしなかった。この直後、最も起きやすいことは何か？
- A) 相場が反転し、予想通りに利益になることが多い
- B) 時間が経てば冷静に損切りできるようになる
- C) 「まだ大丈夫」という認識が強まり、損失が拡大するほど損切りがさらに困難になる
- D) 損切りラインを設定し直すことで対処できる
**正解:** C
**解説:** 「一時的なはず」は否認の典型的な思考。これを許容すると「きっと戻る」→「もうどうでもいい」と心理が連鎖し、損切りできない状態が固定化される。

## Q3
**問題:** 経済指標発表直後に相場が瞬時に200pips動いた。成り行き逆指値を設定していたAと、指値逆指値を設定していたBの結果として最も正確なものはどれか？
- A) AはスリッページなしでBより有利な価格で約定した
- B) Aはスリッページが発生したが約定した。Bは指定価格を飛ばして約定しなかった可能性がある
- C) 両者とも同じ価格で約定した
- D) B（指値）の方が確実に約定し、A（成り行き）は未約定だった
**正解:** B
**解説:** 成り行き逆指値は価格ズレ（スリッページ）があっても確実に約定する。指値逆指値は急変時に価格を飛ばして未約定になるリスクがある。

## Q4
**問題:** 損切りラインを1.0800に設定していたが、価格が近づいたため「今回だけ」と1.0750に変更した。その後また近づいたので「もう少しだけ」と1.0700に変更した。このパターンの本質的なリスクは何か？
- A) 変更のたびに取引コストが増える
- B) ラインを変えると自動的にロット数が増えてしまう
- C) 「今回だけ」という例外が常態化し、最終的に損切りルール全体が機能しなくなる
- D) 損切りラインの変更には証券会社の承認が必要で手間がかかる
**正解:** C
**解説:** 「今回だけ」はルール崩壊の出発点。一度許容すると感情が常に新たな「例外理由」を作り出し、損切りラインは際限なく後退する。

## Q5
**問題:** 損切りが執行された直後、「今すぐエントリーしたい」と思える場面があった。このタイミングでの即座エントリーが危険な理由として最も正確なものはどれか？
- A) 損失の痛みで感情が高ぶっており、「取り返したい衝動」が分析を歪めている
- B) 損切り直後は口座残高が少ないため同じロットでエントリーできない
- C) 損切り直後は証券会社のシステムが不安定になる
- D) 損切りした通貨ペアは一定時間再エントリーできないルールがある
**正解:** A
**解説:** 損失直後は「取り返さなきゃ」という衝動が「根拠」に見えてしまう。感情が落ち着くまで判断を保留することで、衝動と分析を切り分けられる。

## Q6
**問題:** 「チャートを監視して手動で損切りするから逆指値は不要」と考えてエントリーしたトレーダーが、含み損が拡大するにつれて判断できなくなり大損した。この構造的な問題は何か？
- A) チャート監視の時間が足りなかった
- B) エントリーポイントの分析が間違っていた
- C) 手動と逆指値に差はなく、単純な判断ミスだった
- D) 含み損が増えるほど「損失確定の痛み」が強まり、手動判断では感情に負けやすい構造だった
**正解:** D
**解説:** 逆指値の本質的な価値は「感情が介入する前に自動執行する」こと。手動判断に頼ると、含み損拡大とともに心理的バリアが形成され、損切りが構造的に不能になる。

## Q7
**問題:** 成り行き逆指値で損切りが執行されたが、設定した価格より不利な価格（スリッページ）で約定した。このスリッページをどう捉えるべきか？
- A) スリッページは損失なので、成り行き逆指値は使うべきではない
- B) 確実な約定のためのコスト。指値逆指値が急変時に約定しないリスクと比べれば、許容できる代償
- C) スリッページは証券会社のミスなので補償を求めるべき
- D) スリッページを避けるため、急変時は手動で管理すべき
**正解:** B
**解説:** スリッページは市場の流動性に起因する価格差であり、損切りを確実に執行するためのコスト。「約定しなかった」という最悪のシナリオを回避できる対価と理解することで、成り行き逆指値を冷静に使えるようになる。

## Q8
**問題:** 買いポジションが含み損になったトレーダーが「平均取得単価を下げる」目的で追加買いをした。この行動の本質的な問題は何か？
- A) 平均単価を下げることは利益確定を早める有効な手法で問題ない
- B) 追加買いのタイミングが悪かっただけで、手法は正しい
- C) 実質的に損切りラインを後退させ、ポジション量を増やすことでリスクを拡大させている
- D) 元のポジションを先に損切りしてから再エントリーすべきだった
**正解:** C
**解説:** ナンピンは「平均単価改善」に見えるが、損切りラインを後退させ、リスク量を増やす行為。相場がさらに逆行すると、増えたポジション分だけ損失が拡大する。

## Q9
**問題:** 2人がUSD/JPYの買いポジションを持った。Aは「含み損3万円になったら損切り」という金額基準、Bは「直近安値の下5pipsに損切りライン」を設定した。どちらが適切で、理由は何か？
- A) Aの方が適切。金額管理が明確だから
- B) Bの方が適切。エントリーの根拠（安値サポート）が崩れた場所に設定しているから
- C) どちらでも良い。損切りが実行できればOK
- D) どちらも不適切。損切りラインは設定不要
**正解:** B
**解説:** 損切りラインは「エントリーの根拠が崩れる場所」に設定するのが原則。金額基準はリスク管理に有効だが、テクニカルな根拠と結びついていないと「なぜここで切るか」の意味を持てない。

## Q10
**問題:** 買いエントリー後、価格が50pips上昇した。損切りラインをエントリー価格付近から+30pipsに移動させた後、価格が反落して+30pipsで決済された。この判断の価値は何か？
- A) 最大値で決済できなかったため失敗
- B) 損切りラインを動かしてはいけないため、この判断は誤り
- C) もっと早く利確すべきだった
- D) 利益を確保しながらさらなる値幅を狙う構造を作り、損切りラインを不利方向には動かさなかった
**正解:** D
**解説:** トレーリングストップは、利益が出た分だけ損切りラインを利益側に移動させる手法。「利益を守りながら伸ばす」が可能になり、損切りラインを不利方向に動かさない原則も守られている。

## Q11
**問題:** 「面白そうなチャートを見つけた。まずエントリーしてからロットや損切りラインを考えよう」と行動したトレーダーが、想定外の大きなリスクを抱えることになった。最初の判断の何が問題だったか？
- A) エントリー前に損切りラインとロット数を確定させなかったため、リスクが設計されていない状態でポジションを持った
- B) チャートを見つけるのが早すぎた
- C) 面白そうなチャートはエントリーの根拠として十分
- D) ロットを後で調整しようとした意識は正しい
**正解:** A
**解説:** 正しい順番は「①エントリーポイント→②損切りライン→③ロット計算→④エントリー→⑤逆指値設定」。手順を崩すと感情的なロット変更を招き、リスク管理が崩壊する。

## Q12
**問題:** 損切り幅を30pipsで計画していたが「より良い場所に損切りラインがある」と発見し、損切り幅を60pipsに変更した。しかしロット数は変えなかった。この判断が引き起こす問題は何か？
- A) 損切り幅が広い方が勝率が上がる
- B) 損切りラインが遠くなるためリスクは減る
- C) 1トレードの最大損失額が2倍になり、資金管理のルールが崩れる
- D) ロット数と損切り幅は無関係
**正解:** C
**解説:** リスク額＝損切り幅×ロット数。損切り幅を2倍にしたならロットを半分に下げなければリスク額が2倍になる。損切りラインが変わったときは必ずロット数を再計算する。

## Q13
**問題:** 重要な中央銀行の発表を控えてポジションを保有したままにしたい。損切りラインに達したら「必ず」決済されるよう設定したい。どちらの逆指値が適切か？
- A) 指値逆指値。希望価格で約定できるから
- B) 成り行き逆指値。急変時でもスリッページを許容して確実に約定できるから
- C) どちらでも違いはない
- D) リスクイベント前は必ずポジションを閉じるべきで、逆指値設定は不要
**正解:** B
**解説:** 急変が予想される場面では「確実に損切りを執行できること」が最優先。成り行き逆指値はスリッページが発生しても約定が保証され、損切り未執行という最悪のシナリオを排除できる。

## Q14
**問題:** 含み損-10万円のポジションを2週間持ち続けた結果、-30万円になった。「もうどうでもいい」という気持ちになりチャートを開かなくなった。この状態の危険性は何か？
- A) 冷静な長期保有の判断で問題ない
- B) -30万円になった時点が損切りの正しいタイミング
- C) 2週間のホールドは一般的で特に問題ない
- D) 感情が麻痺した状態に入っており、この段階では損切りの実行が感情的に不可能に近くなる
**正解:** D
**解説:** 損失が大きすぎると感覚が麻痺し「どうせ戻らない」「もうどうでもいい」と無関心になる。この段階での損切り実行は極めて困難。これを防ぐのがエントリー時の逆指値設定。

## Q15
**問題:** 損切り後すぐに「さっきのエントリーは絶対に正しかった」という確信が湧き、即座に再エントリーした。しかし十分に時間を置いてから振り返ると「あの根拠は実は薄かった」と気づいた。この経験が示すことは何か？
- A) 損切り直後の「確信」は感情のバイアスであり、感情が落ち着いて初めて客観的な分析ができる
- B) 時間を置くと良いエントリーポイントを逃す
- C) 損切り後はその日のトレードをやめるべき
- D) 時間を置いても判断は変わらないため、すぐに再エントリーした方が良い
**正解:** A
**解説:** 損切り直後の「確信」は感情の産物であることが多い。感情が落ち着いた後に同じチャートを見ると、根拠が薄かったと気づける。「落ち着いてから判断する」習慣がリベンジトレードの連鎖を断ち切る。

## Q16
**問題:** エントリー後「様子を見てから損切りラインを設定しよう」と考えていたら、数分で含み損が大きくなり、設定するタイミングを失った。この根本原因は何か？
- A) 含み損が出てから設定した方が精度の高い損切りラインになる
- B) 数分の値動きはノイズのため気にしなくていい
- C) 時間が経つほど「損失を確定させたくない」感情が強まり、損切りライン設定を心理的に阻む
- D) 含み損が大きくなってから設定した方が正確なラインになる
**正解:** C
**解説:** エントリー直後が最も冷静に損切りラインを設定できるタイミング。エントリーと逆指値設定はセット。時間が経つほど感情が介入し、損切りライン設定を阻む力が大きくなる。

## Q17
**問題:** 下落中にAは損切りラインで決済した。Bは「平均単価を下げる」目的で追加買い（ナンピン）をした。その後さらに相場が下落した。Bの判断が持っていたリスクとして最も正確なものはどれか？
- A) ナンピンで平均単価が改善されたためBのリスクは減った
- B) ナンピンは問題なく、損切りしたAが機会損失を被った
- C) 両者のリスクは変わらない
- D) ポジション量が増えた状態でさらに下落したため、損失がナンピン前の数倍に膨らんだ
**正解:** D
**解説:** ナンピンは平均単価を改善するが、ポジション量を増やす。相場がさらに逆行すると、増えた量に比例して損失が拡大する。Aが損切りラインを守ったのは、最悪のシナリオへの備えだった。

## Q18
**問題:** 上昇トレンドの継続を根拠に買いエントリーした。「高値・安値が切り上がる」というシナリオが否定される場所はどこか？
- A) 含み損が2万円を超えた時
- B) 直近安値を下抜けた時（安値の切り上がりが崩れた時）
- C) エントリーから50pips下落した時
- D) 他の通貨ペアが下落し始めた時
**正解:** B
**解説:** 損切りラインは「エントリーのシナリオが崩れる場所」に置く。「高値・安値の切り上がり継続」が根拠なら、直近安値を割った時点でシナリオは否定される。根拠の崩壊点に置くことで損切りに意味を持たせられる。

## Q19
**問題:** 先月のトレード記録：損切り8回（合計-8万円）、利確4回（合計+18万円）。この月の損切りについて、最も適切な捉え方はどれか？
- A) 損切り8回分（-8万円）は口座を守るための「保険料」であり、純利益+10万円はその保険料が正常に機能した結果
- B) 損切り回数が多すぎる。勝率を上げることが先決
- C) 利益額だけを追うべきで、損切り回数は意味がない
- D) 損切り8回は判断ミスが多すぎるため改善が必要
**正解:** A
**解説:** 損小利大が成立していれば、勝率が低くても口座は成長する。損切りを「コスト」として設計することで感情的な回避が減り、口座の長期的な生存確率が上がる。

## Q20
**問題:** エントリー前に①エントリーポイント決定 ②損切りライン決定 ③ロット計算 ④エントリー ⑤即座に成り行き逆指値設定、を全て行った結果、損切りが執行された。このトレードの評価として最も適切なものはどれか？
- A) 5ステップを全て行ったのに損切りになったため、手法の見直しが必要
- B) 成り行き逆指値を使ったためスリッページが発生し、失敗だった
- C) 正しいプロセスで設計し、損切りが機能した「良い損切り」。このプロセスを繰り返すことが長期生存につながる
- D) 損切りになったのだから、エントリーの根拠が間違っていた
**正解:** C
**解説:** トレードの評価は「結果」ではなく「プロセス」で行う。正しい設計で損切りが機能したなら「仕組みが正常に動いた」証拠。損切りを失敗と捉えず保険料として受け入れる姿勢が長期生存の基盤になる。
