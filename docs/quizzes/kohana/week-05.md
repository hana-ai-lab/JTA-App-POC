# Week 5: レバレッジと生存確率 — 3つのスキル段階

## Q1
**問題:** レバレッジ管理における「Conservative（保守的）」スキル段階の実質レバレッジ上限はいくつですか？
- A) 実質レバレッジ1.0倍以下
- B) 実質レバレッジ1.25倍以下
- C) 実質レバレッジ1.5倍以下
- D) 実質レバレッジ2.0倍以下
**正解:** B
**解説:** Conservative（保守的）段階では実質レバレッジ1.25倍以下が上限です。初心者や口座が小さい段階では、この低いレバレッジで確実に資金を守ることが最優先になります。

## Q2
**問題:** レバレッジ管理の3段階フレームワークにおいて、Standard（標準）の実質レバレッジ上限はいくつですか？
- A) 実質レバレッジ1.0倍以下
- B) 実質レバレッジ1.25倍以下
- C) 実質レバレッジ1.5倍以下
- D) 実質レバレッジ2.0倍以下
**正解:** C
**解説:** Standard（標準）段階では実質レバレッジ1.5倍以下が上限です。基本的な資金管理が身についているトレーダーが、適度な効率を狙える中間的なレベルです。

## Q3
**問題:** レバレッジ管理の3段階のうち、Advanced（上級）の実質レバレッジ上限として正しいものはどれですか？
- A) 実質レバレッジ2.0倍以下
- B) 実質レバレッジ3.0倍以下
- C) 実質レバレッジ5.0倍以下
- D) 実質レバレッジ1.5倍以下
**正解:** A
**解説:** Advanced（上級）段階でも実質レバレッジ2.0倍以下が上限です。たとえ上級者であっても、レバレッジは抑え目に管理することが長期的な生存の鍵となります。

## Q4
**問題:** 証拠金維持率が200%を下回った場合、推奨される対応として正しいものはどれですか？
- A) そのままポジションを保有し続ける
- B) 即座にポジションを縮小する
- C) レバレッジを上げて早く回復させる
- D) 追加の証拠金を入金してから新規ポジションを取る
**正解:** B
**解説:** 証拠金維持率200%は最低ラインであり、これを下回ったら即座にポジションを縮小する必要があります。維持率が低い状態でポジションを持ち続けると、わずかな逆行で強制ロスカットに追い込まれるリスクが急増します。

## Q5
**問題:** トレード前の5項目チェックに含まれないものはどれですか？
- A) ポジションサイズの計算は完了しているか？
- B) リスクは2%以内か？
- C) 現在の口座残高が過去最高を更新しているか？
- D) 連敗中に「特別なこと」をしようとしていないか？
**正解:** C
**解説:** トレード前5項目チェックは「①ポジションサイズ計算完了？②リスク2%以内？③維持率200%以上？④レバレッジ推奨範囲内？⑤連敗中に特別なことしてない？」です。口座残高が過去最高かどうかはチェック項目に含まれません。

## Q6
**問題:** 口座資金が50%減少した場合、元の資金額に戻すために必要な利回りは何%ですか？
- A) 50%
- B) 75%
- C) 100%
- D) 150%
**正解:** C
**解説:** 損失と回復には非対称性があります。50%の損失を取り戻すには100%の利益が必要です。100万円が50万円になった場合、50万円を100万円に戻すには100%の利回りが必要で、これが「大きく負けないこと」の数学的根拠です。

## Q7
**問題:** 口座資金の10%を失った場合、元の金額に戻すために必要な利回りは約何%ですか？
- A) 約15%
- B) 約20%
- C) 約10%
- D) 約11.1%
**正解:** D
**解説:** 10%の損失を回復するには約11.1%の利回りが必要です。100万円が90万円になった場合、90万円から100万円に戻すには10万円／90万円＝約11.1%の利益が必要です。損失が大きくなるほど、回復に必要な利回りは非対称的に増大します。

## Q8
**問題:** 「破産確率」の観点から、トレードにおいて最も優先すべきことは何ですか？
- A) 勝率を高めること
- B) 退場しないこと（生存すること）
- C) 取引回数を増やすこと
- D) 1回のトレードで大きく勝つこと
**正解:** B
**解説:** トレードにおいて最優先は「退場しないこと」です。どれだけ勝率が高くても、1回の大きな損失で口座が吹き飛べば破産確率は100%になります。生存を確保した上で利益を追求するのが正しい順序です。

## Q9
**問題:** VIX（恐怖指数）が30を超えた場合のレバレッジ調整として正しいものはどれですか？
- A) レバレッジを変更せずそのままにする
- B) 全ポジションを即座に決済する
- C) レバレッジを+0.5倍引き上げる
- D) レバレッジを-0.5倍引き下げる
**正解:** D
**解説:** VIXが30を超えたパニック状態の相場では、ボラティリティが急拡大しているため、レバレッジを0.5倍引き下げる動的調整が推奨されます。例えばStandard段階（1.5倍上限）なら1.0倍まで下げてリスクを抑えます。

## Q10
**問題:** 口座資金100万円のトレーダーが2%ルールを厳守する場合、1回のトレードで許容される最大リスク額はいくらですか？
- A) 10,000円
- B) 20,000円
- C) 50,000円
- D) 100,000円
**正解:** B
**解説:** 100万円×2%＝20,000円です。2%ルールは、1回のトレードで失っていい金額を口座の2%に制限するルールで、連敗が続いても口座が急激に減少しない数学的な安全装置です。

## Q11
**問題:** 口座資金が90%減少した場合、元の金額に戻すために必要な利回りは何%ですか？
- A) 90%
- B) 200%
- C) 500%
- D) 900%
**正解:** D
**解説:** 90%の損失を回復するには900%の利回りが必要です。100万円が10万円になった場合、10万円を100万円に戻すには10倍（900%増）の利益が必要です。これが「退場しないこと」を最優先すべき圧倒的な理由です。

## Q12
**問題:** VIXが30を超えている状況で、Standard段階（通常上限1.5倍）のトレーダーが取るべき実質レバレッジの上限はいくつですか？
- A) 1.0倍以下
- B) 1.25倍以下
- C) 1.5倍以下
- D) 2.0倍以下
**正解:** A
**解説:** Standard段階の通常上限は1.5倍ですが、VIX30超えの場合は-0.5倍の調整が加わるため、1.0倍以下に抑える必要があります。ボラティリティの高い相場では、通常より保守的なポジションサイズが求められます。

## Q13
**問題:** 1%ルールを採用した場合、2%ルールに比べて何が数学的に有利ですか？
- A) 連敗時の口座減少をより緩やかにできる
- B) 勝率が上がる
- C) レバレッジを高く設定できる
- D) 利益が2倍になる
**正解:** A
**解説:** 1%ルールは1回のリスクを口座の1%に制限するため、2%ルールよりも連敗時の口座減少が緩やかになります。例えば10連敗した場合、2%ルールでは約82%が残りますが、1%ルールでは約90%が残り、生存確率がさらに高まります。

## Q14
**問題:** 証拠金維持率200%以上を保つべき理由として最も適切なものはどれですか？
- A) 急な相場変動でも強制ロスカットのリスクを回避でき、冷静な判断を保てるから
- B) 税金の計算が簡単になるから
- C) レバレッジをより高く設定できるから
- D) ブローカーの手数料が安くなるから
**正解:** A
**解説:** 証拠金維持率200%以上を保つことで、急激な相場変動が起きても強制ロスカットのリスクを回避できます。また、心理的な余裕が保てるため、冷静な判断が可能になります。維持率がギリギリだと、少しの逆行でパニックに陥りやすくなります。

## Q15
**問題:** 連敗時の自動ブレーキにおいて「5連敗」した場合の対応として正しいものはどれですか？
- A) 1週間の休止を取る
- B) その日のトレードを終了する
- C) その月のトレードを終了する
- D) ロットを半分に下げる
**正解:** D
**解説:** 連敗時の自動ブレーキでは、5連敗した場合はロットを半分に下げます。3連敗でその日終了、5連敗でロット半減、10連敗で1週間休止という段階的な安全装置が設けられています。

## Q16
**問題:** 「大きく勝つ」ことより「退場しない」ことを優先すべき根拠として最も適切なものはどれですか？
- A) 大きな利益には高い税率がかかるから
- B) ブローカーが大きな利益の引き出しを制限するから
- C) 損失と回復には非対称性があり、大きな損失を取り戻すには不相応な利回りが必要だから
- D) 大きく勝つことができるトレーダーは稀だから
**正解:** C
**解説:** 損失と回復の非対称性が「退場しない」ことを優先すべき数学的根拠です。50%の損失には100%、90%の損失には900%の利回りが必要になります。つまり、大きな損失を回避すること自体が、将来の利益の可能性を残すことに直結します。

## Q17
**問題:** 口座資金50万円のトレーダーがConservative段階で、損切り幅50銭（0.5円）のUSD/JPYを取引する場合、2%ルールに基づく最大取引量（1万通貨単位）として最も近いものはどれですか？（1銭＝1万通貨あたり100円）
- A) 5万通貨（0.5ロット）
- B) 10万通貨（1.0ロット）
- C) 1万通貨（0.1ロット）
- D) 2万通貨（0.2ロット）
**正解:** D
**解説:** 許容リスク額＝50万円×2%＝10,000円。損切り幅50銭の場合、1万通貨あたりのリスク＝50銭×100円＝5,000円。したがって最大取引量＝10,000円÷5,000円＝2万通貨（0.2ロット）です。

## Q18
**問題:** トレード前の5項目チェックにおいて「証拠金維持率は200%以上か？」を確認する理由として最も重要なものはどれですか？
- A) 新しいポジションを取るために十分な余裕があるかを確認するため
- B) 過去のトレードの損益を正確に把握するため
- C) 現在のトレンド方向を確認するため
- D) 損切りラインの妥当性を検証するため
**正解:** A
**解説:** 新しいポジションを取る前に維持率200%以上を確認することで、取引後に急な逆行があっても強制ロスカットのリスクを回避できる余裕があるかを確認します。維持率が低い状態での追加ポジションは、生存確率を著しく低下させます。

## Q19
**問題:** VIXが通常レベル（12〜15）から30を超えた状態に変化した場合、Advanced段階（通常上限2.0倍）のトレーダーが取るべき対応として正しいものはどれですか？
- A) レバレッジを変更せず、普段通りにトレードを続ける
- B) レバレッジ上限を1.5倍に引き下げ、リスクを抑制する
- C) 全てのポジションを保有したまま、指値で利確を待つ
- D) レバレッジ上限を2.5倍に引き上げ、大きく動く相場で利益を狙う
**正解:** B
**解説:** VIX30超え時の動的調整により、Advanced段階（2.0倍上限）は-0.5倍されて1.5倍以下に引き下げます。パニック相場では値動きが荒くなるため、通常より保守的なレバレッジで臨むことが生存確率を高めます。

## Q20
**問題:** 連敗時の自動ブレーキで「週-6%」に達した場合、その後どうすべきですか？
- A) その月のトレードを終了する
- B) 1週間の休止を取る
- C) その週のトレードを終了する
- D) ロットを半分にしてトレードを続ける
**正解:** C
**解説:** 週で口座の-6%に達した場合は、その週のトレードを終了します。段階的な自動ブレーキは「3連敗→当日終了」「5連敗→ロット半減」「週-6%→週終了」「10連敗→1週休止」「月-10%→月終了」と設定されており、損失の拡大を構造的に防ぎます。
