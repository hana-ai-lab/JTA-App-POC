# Week 2: ゾーン分析入門 — LB/LS/ZB/ZS

## Q1
**問題:** STRADAのゾーン分析において、「Level（点）」と「Zone（面）」の違いとして正しい説明はどれですか？
- A) Levelは抵抗線、Zoneは支持線
- B) Levelは過去の価格、Zoneは未来の予想価格
- C) Levelはピンポイントの価格、Zoneは幅を持った価格帯
- D) Levelは短期足でのみ確認できる、Zoneは長期足でのみ確認できる
**正解:** C
**解説:** Levelは交差点のように一点の価格を指し、Zoneは商店街のように幅を持った価格帯を表します。大口の注文は一点に集中するのではなく帯状に分布するため、「面で待つ」ことが重要になります。

## Q2
**問題:** ZB（Zone Buy）の定義として正しいものはどれですか？
- A) 個人投資家の買い注文が集中した価格帯
- B) LSの集合体で形成された売りゾーン
- C) フィボナッチ61.8%に位置するサポートライン
- D) LBの集合体で形成された買いゾーン
**正解:** D
**解説:** ZB（Zone Buy）は、複数のLB（Level Buy＝大口買い注文の痕跡）が集合して形成される買いゾーンです。点（LB）が集まって面（ZB）になるという「点から面への変換」がゾーン分析の基本構造です。

## Q3
**問題:** LS（Level Sell）が表すものとして正しいものはどれですか？
- A) 個人投資家の損切り注文が並ぶ価格
- B) 大口買い注文の痕跡（点）
- C) 大口売り注文の痕跡（点）
- D) 売りゾーンの全体像（面）
**正解:** C
**解説:** LS（Level Sell）は大口の売り注文が残した痕跡をポイント（点）で示したものです。複数のLSが集合することでZS（Zone Sell＝売りゾーン）という面が形成されます。

## Q4
**問題:** ゾーン分析において「面で待つ」ことが推奨される理由として最も適切なものはどれですか？
- A) 面で待つ方が損切り幅を広く設定でき、負けにくくなるため
- B) 線（点）で待つと価格がちょうどその位置に届かないことが多く、実際の注文は帯状に分布しているため
- C) テクニカル指標の精度が面の方が高いため
- D) 上位足の分析では線での判定が技術的に不可能なため
**正解:** B
**解説:** 「線」で待つと価格がその一点に届かずに反転することが多々あります。大口の注文は特定の価格に集中するのではなく帯状に分布しているため、幅を持ったゾーン（面）で待つ方が現実的なアプローチです。

## Q5
**問題:** ゴールデンゾーンとは何を指しますか？
- A) 過去3回以上サポートとして機能した価格帯
- B) 出来高が過去最多を記録した価格帯
- C) フィボナッチ50%〜61.8%のリトレースメントとゾーンが重なるポイント
- D) 200MAと価格が交差する価格帯
**正解:** C
**解説:** ゴールデンゾーンは、フィボナッチの50%〜61.8%リトレースメント帯とゾーンが重なるスイートスポットです。複数の分析ツールの結果が一致するため、最も反応しやすい重要なエリアとなります。

## Q6
**問題:** 次の記述のうち、LBとZBの関係として正しいものはどれですか？
- A) LBが集合してZBを形成する。つまり「点」が集まって「面」になる
- B) ZBはLBを上位足で確認し直したもの
- C) LBとZBは別々の分析手法で、互いに関連はない
- D) LBはZBの中で最も重要な一点だけを抽出したもの
**正解:** A
**解説:** LB（Level Buy）は大口の買い注文の痕跡を点として捉え、複数のLBが集合したものがZB（Zone Buy）という買いゾーン（面）になります。同様に、LSが集合してZS（Zone Sell）が形成されます。

## Q7
**問題:** ゾーンの強さを判定する5要素に含まれないものはどれですか？
- A) 出来高の増減率
- B) 出発の勢い
- C) フレッシュさ
- D) 時間軸
**正解:** A
**解説:** ゾーンの強さを判定する5要素は「出発の勢い」「フレッシュさ」「時間軸」「合流」「トレンドとの一致」です。出来高の増減率はワイコフ分析の要素であり、ゾーンの強さ判定には含まれません。

## Q8
**問題:** ゾーンの強さ要素のひとつ「フレッシュさ」が最も高い状態とはどのような状況ですか？
- A) 過去3回価格が到達し、その都度反発を確認したゾーン
- B) 直近の経済指標発表後に形成されたゾーン
- C) まだ一度も価格が戻ってきていない、未到達のゾーン
- D) 出来高が直近で最も多く記録されたゾーン
**正解:** C
**解説:** フレッシュさとは「まだ価格がそのゾーンに到達していない」ほど高いと評価されます。未到達のゾーンが最強であり、再到達の回数が増えるほどフレッシュさが失われ、ゾーンの効力は低下します。

## Q9
**問題:** ゾーンの強さにおいて「時間軸」が意味する内容として正しいものはどれですか？
- A) 取引時間帯（東京・ロンドン・NY）のどれで形成されたか
- B) 1日のうち最も出来高が多い時間に形成されたゾーンほど強い
- C) 上位足で確認されたゾーンほど強い
- D) ゾーンが形成されてからの経過時間が短いほど強い
**正解:** C
**解説:** ゾーンの強さは、それがどの時間軸の足で確認できるかによって大きく変わります。日足や週足などの上位足で確認できるゾーンは、4H足や1H足などの下位足のゾーンより強いと評価されます。

## Q10
**問題:** S級ゾーンの条件として正しい組み合わせはどれですか？
- A) 日足以上 / フレッシュ / フィボ50〜61.8%合流 / トレンド一致
- B) 週足以上 / 再到達2回以内 / フィボ23.6%合流 / 出来高増加
- C) 4H足 / 再到達1回以内 / フィボ38.2%合流 / レンジ方向と一致
- D) 日足以上 / 再到達3回以内 / フィボ78.6%合流 / トレンド一致
**正解:** A
**解説:** S級ゾーンの条件は「日足以上で確認」「フレッシュ（未到達）」「フィボナッチ50%〜61.8%と合流」「トレンド方向と一致」の全てを満たすことです。これら最高条件が揃ったゾーンが最も信頼性が高いとされます。

## Q11
**問題:** B級ゾーンの特徴として正しいものはどれですか？
- A) 日足で確認でき、フレッシュで、フィボナッチと合流する
- B) 4H足で確認でき、フレッシュで、何らかの合流がある
- C) 上位足で確認でき、トレンドに逆行しているが合流がある
- D) 再到達1〜2回あり、合流なし、下位足のみで確認できる
**正解:** D
**解説:** B級ゾーンは「価格の再到達が1〜2回ある」「他ツールとの合流がない」「下位足のみで確認できる」という条件に当てはまります。S級・A級に比べて信頼性が低く、トレードの根拠としては慎重な判断が必要です。

## Q12
**問題:** 価格が買いゾーンに到達した際、推奨される正しい対応はどれですか？
- A) ゾーン到達と同時に即座に成行買い注文を入れる
- B) ゾーン到達後は損切り幅を広く設定してエントリーする
- C) ゾーン到達を確認したら、すぐに指値注文を入れて待機する
- D) ゾーン到達はあくまで準備エリアであり、ピンバー・包み足等のプライスアクション確認後に判断する
**正解:** D
**解説:** ゾーンは「ここで反応する可能性がある」という準備エリアに過ぎません。到達即エントリーはギャンブルと同じであり、ピンバーや包み足などのプライスアクションによる反転確認を待ってから判断するのが正しい手順です。

## Q13
**問題:** 次のうち、「効きにくいゾーン」の特徴として正しいものはどれですか？
- A) 価格の再到達が3回以上ある
- B) 出発の勢いが強い
- C) フィボナッチ等との合流がある
- D) 上位足で確認できる
**正解:** A
**解説:** 再到達が3回以上あるゾーンは効きにくくなります。その他にも「下位足のみで確認できる」「他ツールとの合流がない」「トレンドに逆行している」などは効きにくいゾーンの特徴です。ゾーンは消化されるほど効力が落ちます。

## Q14
**問題:** 次のシナリオを読んで、このゾーンのグレードとして最も適切なものを選んでください。

「日足で確認された買いゾーン。価格はまだ一度もこのゾーンに戻っていない。フィボナッチのリトレースメント58%付近と重なっており、上位足の上昇トレンド方向と一致している。」
- A) A級
- B) S級
- C) B級
- D) グレード外（条件不足）
**正解:** B
**解説:** 日足以上で確認（S級条件）、フレッシュ（未到達）、フィボ50〜61.8%と合流、トレンド方向と一致。S級の4条件すべてを満たしています。

## Q15
**問題:** 次のシナリオを読んで、このゾーンのグレードとして最も適切なものを選んでください。

「4時間足で確認された売りゾーン。価格はまだ一度も戻っていない。フィボナッチ38.2%付近とは離れているが、直近の高値と重なる。」
- A) S級
- B) B級
- C) グレード判定不能
- D) A級
**正解:** D
**解説:** A級の条件は「4H足で確認」「フレッシュ」「合流あり」です。このゾーンは4時間足・フレッシュ・直近高値との合流があるためA級に該当します。フィボナッチとの合流がなくても、何らかの合流があればA級条件を満たします。

## Q16
**問題:** ゾーン分析において「合流」とは何を意味しますか？
- A) 複数のLBが同じ価格帯に集中していること
- B) フィボナッチ等の他の分析ツールの結果とゾーンが重なること
- C) 複数の通貨ペアで同じゾーンが確認できること
- D) 東京・ロンドン・NYの全セッションでゾーンが形成されること
**正解:** B
**解説:** 合流とは、フィボナッチリトレースメントやその他の分析ツールで示される水準とゾーンが重なることを指します。複数の分析結果が一致するほど、そのゾーンの信頼性が高まります。

## Q17
**問題:** 「出発の勢い」が強いゾーンほど信頼性が高い理由として最も適切な説明はどれですか？
- A) 急騰や急落の背後には大口の大量注文が存在し、その痕跡が濃いほどゾーンとしての根拠が強いため
- B) 急激な動きの後はテクニカル指標が全て買いシグナルを出すため
- C) 勢いが強いほど損切り幅を小さくできるため
- D) 急激な動きの後は必ず反転するという統計的傾向があるため
**正解:** A
**解説:** 急激な価格変動（急騰や急落）は、その背後に大口の大量注文があったことを示唆します。この「出発の勢い」が強いほど大口の関与が濃厚で、残された注文痕跡も明確になるため、ゾーンとしての信頼性が高まります。

## Q18
**問題:** 次の記述のうち、トレンドに逆行するゾーンについて正しいものはどれですか？
- A) トレンドに逆行するゾーンは逆張りのチャンスとして積極的に狙うべき
- B) トレンドに逆行するゾーンは、他の条件を満たしていても効きにくい
- C) トレンドに逆行するゾーンは、時間軸が上位であれば信頼性が変わらない
- D) トレンドに逆行するゾーンは、出発の勢いが強ければS級になりうる
**正解:** B
**解説:** ゾーンの強さ5要素のひとつ「トレンドとの一致」は非常に重要です。他の条件を満たしていても、トレンドに逆行するゾーンは効きにくくなります。トレンド方向に一致するゾーンの方が反応する確率が高くなります。

## Q19
**問題:** 次のシナリオについて、最も適切な判断を選んでください。

「15分足で確認された買いゾーンに価格が到達した。このゾーンは過去に2回価格が到達して反発している。フィボナッチ等との合流はなく、直近のトレンドは下降トレンドである。」
- A) 過去2回の反発実績があるため、信頼性は高く即エントリーすべき
- B) 下位足のみ・合流なし・トレンド逆行・再到達済みと条件が弱く、エントリーを見送るべき
- C) 15分足で2回の反発実績はA級以上の根拠になるため、慎重にエントリーを検討すべき
- D) フレッシュではないが、反発実績があるため小ロットでエントリーすべき
**正解:** B
**解説:** このゾーンは下位足のみ（15分足）・合流なし・トレンドに逆行・再到達2回済みと、効きにくいゾーンの特徴ばかり揃えています。個別の条件が弱いことに加え、複数の弱点が重なっているため見送りが適切です。

## Q20
**問題:** A級ゾーンの条件として正しい組み合わせはどれですか？
- A) 4H足 / 再到達1回以内 / フィボ38.2%合流 / レンジ相場
- B) 下位足 / 再到達なし / 合流なし / トレンド一致
- C) 日足以上 / フレッシュ / フィボ50〜61.8%合流 / トレンド一致
- D) 4H足 / フレッシュ / 何らかの合流あり
**正解:** D
**解説:** A級ゾーンの条件は「4H足で確認」「フレッシュ（未到達）」「何らかの合流がある」の3点です。S級ほど厳しくないものの、実践で十分に機能する信頼性の高いゾーンです。
