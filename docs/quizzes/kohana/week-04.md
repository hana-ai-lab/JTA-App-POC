# Week 4: なぜ9割が負けるのか — STRADAで学ぶ意味

## Q1
**問題:** あるトレーダーが買いポジションで含み益+3万円になった途端に利確した。その後、別のポジションで含み損-3万円になっても「戻るはず」と保有し続けた。なぜこのような行動パターンが生まれるのか？
- A) 損失の痛みを利益の喜びの約2倍強く感じるため、利益は早く確定し損失は先延ばしにするから
- B) 利益の喜びが損失の痛みより強く、もっと利益を得たいという欲求が行動を支配するから
- C) 利益と損失の感じ方は同じで、単に取引経験の不足による判断ミスだから
- D) 含み損になると脳が危機モードになり、何もできなくなるから
**正解:** A
**解説:** 損失の痛みは利益の喜びの約2倍強く感じるため、利益が出ると「早く確定したい（利小）」、損失が出ると「なかったことにしたい（損大）」という行動パターンが生まれます。これが利小損大の根本原因です。

## Q2
**問題:** 確証バイアスによる判断の歪みを防ぐ対策として、最も有効なものはどれか？
- A) 自分のエントリー方向を支持するアナリストのレポートを複数集め、根拠を固める
- B) エントリー前に「自分が間違っている場合のシナリオ」を書き出し、反対意見も意図的に探す
- C) ポジションを持ったら情報収集を止め、チャートの動きだけを見て判断する
- D) 確証バイアスは経験を積めば自然に消えるため、トレード数をこなして慣れる
**正解:** B
**解説:** 確証バイアスは、都合のいい情報だけを集める心理です。対策は意図的に反対意見・反証を探すこと。「自分が間違っていたらどうなるか」を書き出すことで、バイアスを外した客観的な判断ができます。

## Q3
**問題:** 「損切りラインの-2万円を超えたが、2週間も我慢してきたから今さら損切りできない」と考えてポジションを保有し続けた。このトレーダーの判断の何が問題か？
- A) 2週間という保有期間が短すぎるため、もっと長期で様子を見るべきだったから
- B) 損切りラインを-2万円と設定したこと自体が間違いで、もっと広く設定すべきだったから
- C) 現在の損失額ではなく「過去に費やした時間・忍耐」を判断の根拠にしてしまっているから
- D) 含み損の状況で感情的になることは当然で、感情に従って待つことも合理的な選択だから
**正解:** C
**解説:** 過去に費やした時間や忍耐は、今後の損益には一切関係しません。「ここまで我慢したから」という理由で損切りを先延ばしするのは取り戻せないコストにとらわれた判断であり、損失をさらに拡大させます。

## Q4
**問題:** 含み損のポジションについて「このペアは自分で調べて選んだものだから」「手放すのが惜しい」と感じて損切りできない。このトレーダーの根本的な問題は何か？
- A) ポジションを持つ前の分析が不十分だったため、再度ファンダメンタルを確認すべきだから
- B) ポジション保有中にネガティブな感情が生まれることは正常であり、感情への対処法を学ぶべきだから
- C) 損切りラインを事前に明確に設定していなかったことが根本的な原因だから
- D) ポジション自体に「自分が選んだ」という感情的な価値を付加してしまい、客観的な損切り判断ができなくなっているから
**正解:** D
**解説:** 保有効果は、自分が持っているものに実際より高い価値を感じる心理です。「自分が選んだポジション」に思い入れを持つほど手放せなくなり、損切りのタイミングを逃して損失が拡大します。

## Q5
**問題:** 含み損-5万円で損切りラインをすでに超えているのに、「今は動かない方がいい」「もう少し待てば…」と何もできない状態が続いている。このトレーダーが今すべき行動として正しいものはどれか？
- A) 感情に関係なく、事前に設定した損切りラインを根拠に今すぐ損切りを実行する
- B) 含み損が-8万円まで広がったら強制決済されるよう、逆指値を入れ直す
- C) チャートを閉じて一晩置き、翌朝冷静になってから判断する
- D) 「待てば回復する」という直感を信じ、ファンダメンタル分析で根拠を探す
**正解:** A
**解説:** 「今は動かない方が安全」と感じて損切りを先延ばしするのは現状維持バイアスの典型です。すでに損切りラインを超えている以上、感情ではなく事前のルールに従って即座に実行することが損失の拡大を防ぐ唯一の方法です。

## Q6
**問題:** 5つの認知バイアスが重なって生まれる負けパターンの流れとして正しいものはどれですか？
- A) 利小損大 → 損切り遅延 → 塩漬け → 破滅
- B) 確証バイアス → リベンジ心理 → 利小損大 → 破滅
- C) 塩漬け → 利小損大 → リベンジ心理 → 確証バイアス
- D) リベンジ心理 → 保有効果 → サンクコスト → 利小損大
**正解:** A
**解説:** プロスペクト理論による利小損大から始まり、損失回避による損切り遅延、現状維持バイアスによる塩漬けへと連鎖し、最終的に破滅に至るのが典型的な負けパターンです。

## Q7
**問題:** 「ここまで耐えたのだから」と損切りを先延ばしにした結果、-10万円の損切り後にロットを3倍にしてリベンジトレードに走った。この連鎖を最初の段階で断ち切るために必要だったことは何か？
- A) -10万円になった時点でチャートを閉じ、頭を冷やしてから再エントリーを検討すること
- B) ロットを3倍にする前に、過去の連敗パターンをトレード日誌で振り返ること
- C) 「ここまで耐えた」という感情が湧いた時点でそれをシグナルと捉え、事前の損切りルールに機械的に従うこと
- D) コミュニティで他のトレーダーに現状を相談し、客観的な視点をもらうこと
**正解:** C
**解説:** 「ここまで耐えた」という感情はサンクコストの典型的なシグナルです。この感情が湧いた瞬間に気づき事前のルールに機械的に従うことで、保有効果→リベンジ心理へと続く破滅の連鎖を最初の段階で止められます。

## Q8
**問題:** 損切りを「保険料」と捉える理由として最も適切なものはどれですか？
- A) 損切りをすることで税金上の優遇措置が受けられるから
- B) 小さな損失を払うことで大きな損失（破産）を防げるから
- C) 損切りの回数を重ねることで自動的に勝率が上がるから
- D) 損切りした金額は後で必ず回収できる仕組みになっているから
**正解:** B
**解説:** 損切りは失敗ではなく、大損を防ぐための保険です。火災保険と同じで、小さな費用（損失）を払うことで、破産級の損失を未然に防ぐことができます。

## Q9
**問題:** メタ認知の最初のステップ「気づく」で最も重要なことは何か？
- A) 感情に気づいたらすぐチャートを閉じて、感情が収まるまで休むこと
- B) 過去の失敗トレードのパターンをノートで振り返り、同じ状況かどうか確認すること
- C) 深呼吸や瞑想で心を落ち着かせ、感情を中立の状態に戻すこと
- D) 「今、焦りを感じている」「今、恐怖を感じている」と、感情に具体的な名前をつけること
**正解:** D
**解説:** 感情に名前をつける（ラベリング）だけで、感情と自分の間に距離が生まれます。「焦り」「恐怖」と認識した瞬間、それは「脳の反応」になり、衝動的なエントリーを防ぐことができます。

## Q10
**問題:** 「今この波に乗らなければ置いていかれる！」という強い衝動でエントリーしようとした。この暴走感情の正体と、取るべき行動の正しい組み合わせはどれか？
- A) 焦り（FOMO）が暴走している。5秒止まって「計画外のエントリーではないか？」を確認し、計画になければ見送る
- B) 欲望（もっと欲しい）が暴走している。利確目標をあらかじめ設定し、その目標まで機械的に待つ
- C) 恐怖（損を認められない）が暴走している。逆指値を入れて感情の介入余地をなくす
- D) 後悔（過去に囚われる）が暴走している。過去のトレード日誌を振り返ってから判断する
**正解:** A
**解説:** 「乗り遅れる恐怖（FOMO）」は焦りが暴走した状態です。5秒止まって「計画通りのエントリーか？」を確認することで感情による飛び乗りを防げます。5秒のチェックに通らない衝動は、そもそも計画外のエントリーです。

## Q11
**問題:** 5秒ルール（5問チェック版）の5つの質問として含まれないものはどれですか？
- A) 計画通りのエントリーか？
- B) 損切り位置は決まっているか？
- C) 過去10回の勝率は50%以上か？
- D) 上位足と整合しているか？（逆張りではないか）
**正解:** C
**解説:** 5秒ルールの5問は「計画通りか」「損切りは決まっているか」「RR1:2以上か」「上位足と整合しているか」「夜眠れるロット数か」です。過去の勝率の確認は含まれません。

## Q12
**問題:** 保有効果を打破するための「最重要の問いかけ」として正しいものはどれですか？
- A) 「今日の損益はプラスだろうか？」
- B) 「もし今ポジションを持っていなかったら、このレートでエントリーするか？」
- C) 「過去に同じパターンで勝てたか？」
- D) 「この通貨ペアは今上昇トレンドにあるか？」
**正解:** B
**解説:** 「もし今ポジションを持っていなかったら、このレートでエントリーするか？」——答えがNoなら、保有効果にとらわれずそのポジションを手放すべきです。

## Q13
**問題:** 「5秒で逃すチャンスは、そもそもあなたのチャンスではなかった」という言葉が意味することとして最も適切なものはどれですか？
- A) チャンスは頻繁に来るので、1つ見逃しても気にする必要はない
- B) 5秒の冷静チェックに通らないエントリーは、計画外の感情的なものである
- C) エントリーの判断は5秒以内に直感で行うべきである
- D) どんなチャンスも5秒で判断しなければ次のチャンスに切り替えるべきである
**正解:** B
**解説:** 5秒ルールのチェックで弾かれるエントリーは、元々計画になかった感情的な飛び乗りです。5秒止まることで防げる損失の方が、逃すチャンスよりはるかに大きいという意味です。

## Q14
**問題:** 次のシナリオを読んで、作動している認知バイアスを答えてください。

「USD/JPYの買いポジションを持っている。含み損が-3万円まで膨らんだが、SNSで「ドル円は上昇トレンド」という投稿を3つ見つけて安心し、損切りを見送った。逆に「下落リスク」を指摘する記事は読まなかった。」
- A) プロスペクト理論
- B) 確証バイアス
- C) 保有効果
- D) サンクコスト
**正解:** B
**解説:** 自分のポジションに都合のいい情報（上昇トレンドの投稿）だけを集め、不都合な情報（下落リスクの記事）を無視しているのが確証バイアスの典型的なパターンです。

## Q15
**問題:** 次のシナリオを読んで、最も優先すべき行動を選んでください。

「連敗が続いてイライラしている。「今日中に取り返そう」とロットを2倍に上げてエントリーしようとしている。上位足は下降トレンド、エントリー方向は買い。」
- A) 焦り（FOMO）に対処するため、上位足の下降トレンドに従って売りエントリーに切り替える
- B) ロットを2倍にするのではなく、いつものロットで買いエントリーする
- C) リベンジトレードの典型パターンなので、エントリーを見送りチャートを閉じる
- D) 「今日中に取り返す」という目的を達成するため、損切り幅を広く設定してエントリーする
**正解:** C
**解説:** 連敗後の「取り返したい」という衝動はリベンジ心理です。上位足に逆行し、ロットを上げ、感情で動いている状態はリベンジトレードの典型であり、エントリーを見送るべきです。

## Q16
**問題:** 次のケースで、5秒ルールの5問チェックを適用した場合の判断として正しいものはどれですか？

「日足上昇トレンド中。4時間足のZBに到達し、ピンバーが出た。損切りは決まっている。RR比は1:1.8。ロットは夜眠れるサイズ。計画通りのエントリー。」
- A) 全ての条件を満たしているのでエントリーする
- B) RR比が1:2未満なのでNoが1つあり、見送りまたは条件が整うまで待つ
- C) ロットが適正ならRR比は関係ないのでエントリーする
- D) ピンバーが出ているならRR比の確認は不要なのでエントリーする
**正解:** B
**解説:** 5秒ルールは5つすべてYesでなければエントリー不可です。RR比1:1.8は「RR1:2以上か？」という質問に対してNoとなり、1つでもNoがあれば見送りが正解です。

## Q17
**問題:** 次のシナリオを読んで、作動しているバイアスの連鎖として最も適切なものを選んでください。

「含み損-2万円のポジション。「ここまで耐えたのだから」と損切りを見送った。含み損が-5万円に拡大。「自分のポジションを手放したくない」とさらに保有し続けた。ついに-10万円で損切り。「すぐに取り返さないと」とロットを3倍にして再エントリーした。」
- A) プロスペクト理論 → 確証バイアス → 現状維持バイアス
- B) 現状維持バイアス → 確証バイアス → プロスペクト理論
- C) サンクコスト → 保有効果 → リベンジ心理
- D) 保有効果 → サンクコスト → 確証バイアス
**正解:** C
**解説:** 「ここまで耐えたから」がサンクコスト、「手放したくない」が保有効果、「すぐに取り返さないと」がリベンジ心理です。この3つの連鎖が損切りできない人の典型的な破滅パターンです。

## Q18
**問題:** 恐怖（Fear）による暴走を防ぐ対策として最も有効なものはどれですか？
- A) 勝率を上げるために分析をさらに細かくする
- B) 小さいロットで何度も練習して恐怖に慣れる
- C) 損失を認めない自信をつけるために勝率の高い手法を探す
- D) エントリー前に逆指値注文を入れておき、感情の介入余地をなくす
**正解:** D
**解説:** 恐怖は「損を認められない感情」です。エントリー前に逆指値（損切り注文）を設定しておけば、損切りラインに達した時点で自動的に決済され、感情の介入余地がなくなります。

## Q19
**問題:** 次のシナリオを読んで、メタ認知の3ステップを適用した場合の正しい順番を選んでください。

「チャートを見ていて「乗り遅れる！」と焦ってエントリーしようとした瞬間に立ち止まった。」
- A) 「今、焦り（FOMO）を感じている」と認識する → 「これは一時的な脳の反応だ」と切り離す → 5秒ルールのチェックリストに従う
- B) 5秒ルールのチェックリストに従う → 「今、焦りを感じている」と認識する → 「これは一時的な脳の反応だ」と切り離す
- C) 「これは一時的な脳の反応だ」と切り離す → 5秒ルールのチェックリストに従う → 「今、焦りを感じている」と認識する
- D) エントリーしてから感情を分析する → 次回同じ失敗をしないようルールを見直す → 新しいルールを記録する
**正解:** A
**解説:** メタ認知は「気づく（感情にラベルを貼る）→ 距離を置く（感情と自分を切り離す）→ ルールに戻る（事前のルールに従う）」の順です。感情に名前をつけるだけで行動の質が変わります。

## Q20
**問題:** 次のシナリオを読んで、最も適切な判断を選んでください。

「EUR/USDの買いポジションを持っている。含み益+1.5万円。「まだ上がるかも」と思ったが、同時に「利益が消えたら嫌だ」という不安も感じた。利確目標の+3万円にはまだ届いていない。」
- A) 「まだ上がるかも」という欲望に従い、そのまま保有し続ける
- B) 利益が消える恐怖から、利確目標より早く利益を確定させる
- C) プロスペクト理論による利小損大の兆候と気づき、事前に決めた利確目標まで機械的に待つ
- D) 含み益が出ているうちにロットを追加して利益を最大化する
**正解:** C
**解説:** 「利益が消えたら嫌だ」と早く利確したくなるのはプロスペクト理論による反応です。メタ認知で気づき、事前に決めたルール（利確目標+3万円）に戻って機械的に実行するのが正しい対処です。
